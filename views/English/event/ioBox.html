I/O box

An external I/O device can be connected to the camera for use.

In [I/O box] page, the information of I/O box to be connected to the camera can be entered, and the connection status of the I/O box and camera can be checked. Once the setup is complete, click the [Apply] button at the bottom of the page.

I/O box

The IP type, protocol type, IP address, port number, etc. of an external I/O box to be connected to the camera can be set.

To connect an external I/O box to the camera and use it, select [Enable I/O box].

IP type

Selects between IPv4 and IPv6 for the IP address type to use when I/O box communicates with the camera.

HTTP/HTTPS

Selects between HTTP and HTTPS for the communication method to use when I/O box communicates with the camera.

IP address

Enter the IP address of the I/O box.

HTTP port/HTTPS port

Enter the port number of the I/O box.

Depending on the value selected for [HTTP/HTTPS], the option name is displayed as either [HTTP port] or [HTTPS port].

ID

Enter the ID to access the I/O device.

Password

Enter the password to access the I/O device.

Status

Displays the connection status between the I/O device and camera. If the camera and I/O device is properly connected, [Connected] is displayed; if they are in the process of being connected, [Connecting] is displayed.
