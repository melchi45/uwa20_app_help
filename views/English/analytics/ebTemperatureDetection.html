Estimated body temperature detection

Body temperature can be detected based on the face temperature. Criteria or detailed options to apply when detecting body temperature can be set up.

Once the setup is complete, click the [Apply] button at the bottom of the page. The setting on the estimated body temperature detection event applies to [Estimated body temperature detection] in [Event]>[Event setup] pages equally.

Estimated body temperature detection

Enable estimated body temperature detection

To enable estimated body temperature detection event, select [Enable estimated body temperature detection].

Temperature detection

Detailed settings for the camera to extract the highest temperature value based on the detected face area and trigger the estimated body temperature detection event can be set.

Detection area overlay

Selecting [Enable] displays the detection area. It displays the area for face detection to operate, and any faces outside the area are not detected.

Detection temperature

Enter the temperature that will trigger an estimated body temperature detection event.

Minimum duration (s)

Sets the minimum duration of the trigger condition to trigger an estimated body temperature detection event. When the highest temperature on the face area set in [Reference detection area] exceeds the temperature set in [Detection temperature] for longer than the duration set in [Minimum duration], an estimated body temperature detection event occurs.

Temperature detection beep

To make a beep sound when body temperature is detected, select [Enable].

Reference detection area

Detects body temperature based on the highest temperature on the face. In [Reference detection area], you can set the face area to extract the highest temperature value.

If you click [Reset], the settings are discarded and the face area defaults back to the size and location of the initial detection area.

  • Horizontal ratio: The size of face area can be adjusted by adjusting the horizontal reduction ratio of the face area.
  • Vertical ratio: The size of face area can be adjusted by adjusting the vertical reduction ratio of the face area.
  • Location: The location of face area can be adjusted with up/down/left/right buttons.

Compensation

Detailed settings for temperature compensation to be applied when camera detects face temperature can be set.

Enable custom offset

You can set the temperature value that compensates an error between the actual temperature of the measured object and the temperature detected by the camera. Once the temperature value is set, the camera detection temperature value, which is added by the set temperature value, is displayed. To use the custom offset, check the [Enable custom offset] box.

Enable ambient temperature

You can set the current temperature for the environment where the camera is installed. To enable the ambient temperature to compensate the camera's detection temperature, check the [Enable ambient temperature] box.

Enable ambient humidity

You can set the current humidity for the environment where the camera is installed. To enable the ambient humidity to compensate the camera's detection temperature, check the [Enable ambient humidity] box.

Detection distance

Sets the distance between the subject and camera in order to apply to the compensation of the camera detection temperature using the detection distance.

Enable blackbody compensation

Select [Enable blackbody compensation] to compensate for temperature using blackbody. When blackbody compensation is enabled, the temperature can be compensated by setting the following options:

  • Blackbody temperature: Sets the blackbody operating temperature value.
  • Blackbody temperature modification: Sets the error between the entered blackbody operating temperature value and actual value.
  • Blackbody emissivity: Sets the operating emissivity value for the blackbody.
  • Blackbody distance: Sets the distance between the blackbody and camera.
  • Blackbody height: Sets the installation height of the blackbody.  You can set the installation height of the blackbody.
  • Camera height: Sets the installation height of the camera.
  • Camera tilt angle: Sets the angular difference of installation between the camera and blackbody.

Blackbody error beep

If you select [Enable], the camera makes a beep sound when the blackbody temperature value detected by the camera and the blackbody temperature value you set is different.

Calibration

Synchronizes video coordinates to share the location of the face area as coordinates of thermal and visible videos.

By dragging the mouse on the camera video screen to a desired location or using the [Location] button, you can fine-tune its up/down/left/right location.

Location

Click this button to adjust the up/down/left/right location.

Common

You can set the common settings to apply for camera to detect face area.

Sensitivity

Sets the sensitivity of detecting faces. Decrease the sensitivity in an environment where background and objects are clearly distinguished, and increase the sensitivity in a dark environment where background and objects are hard to be distinguished.

Size

Sets the minimum and maximum size of objects that will be subject to body temperature detection. If you click [Reset], the settings are discarded and the minimum and maximum size reset to factory defaults.

Temperature

  • [Maximum temperature limit]: Does not detect any temperature above the set temperature. For example, if [40] is entered, temperature beyond 40 degrees is not detected.
  • [Minimum temperature limit]: Does not detect any temperature less than the set temperature. For example, if [35] is entered, temperature below 35 degrees is not detected.
