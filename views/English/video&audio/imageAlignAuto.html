Image Alignment

You can set the field of view and detection distance for your mounted camera.

Note

  • Field of view and detection distance cannot be set simultaneously.

Angular field of view

When you arrange the videos of mounted cameras to create a panoramic view of the videos, you can adjust the field of view by choosing the way of arranging each individual camera mounted.

  • 180° mode: Arrange camera screens vertically. Since this mode arranges the screens according to the length of each screen, you will see a video whose length is longer than that in the 220° mode mode.  
  • 220° mode: Arrange camera screens horizontally. Since this mode arranges the screens according to the width of each screen, you will see a video whose width is longer than that in the 180° mode.

Note

  • If you change the Angular field of view option, the camera reboots and the video setup will be reset according to the selected angular field of view. Please set privacy area, Analytics, and Statistics again. These can be set in the [Video setup], [Analytics], and [Statistics] menus, respectively.

Target distance

After selecting the field of view of your camera, you can enter a distance value for the target object that is being monitored. You can set the value by adjusting the slide bar or enter a value in meters.

An LUT (Look Up Table) will be generated according to the entered distance value and applied to the panoramic view.
