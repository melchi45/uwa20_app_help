Location setup

You can set to display direction information on the camera video.

Set up to eight directions: north, northern east, east, southern east, south, southern west, west, and northern west. Then, every time PTZ is adjusted and the location of camera pan changes, the direction information on the current video screen is updated accordingly.

Note

  • Changing the OSD size or language setting resets all of the direction setting values (azimuth, description, and location information). You can set the OSD size in [Video & Audio]>[Camera setup]>[OSD], and the language in [System]>[Product information].

Direction indicator

Enable direction indicator

To enable the direction indicator feature, select [Enable direction indicator]. Every direction information set for any of the eight directions are displayed.

[Enable direction indicator] is the option that allows the use of the whole direction display function, whereas [Indicator] is the option to set up the display of an individual direction.

Setting up the direction indicator

  1. Adjust the camera PTZ as shown below to designate a reference video screen when setting up the camera direction.
    If you want to activate the PTZ jog panel, [Enable direction indicator] should not be selected.
    • When you drag the icon with the mouse, the camera moves to the pan and tilt.
    • Use the zoom bar to adjust the image magnification, or select an image magnification by clicking the icon to zoom in or the icon to zoom out.
    • Move the focus nearer by clicking the icon or farther by clicking the icon.
    • To automatically adjust the camera focus once, click the icon.
  2. Select [Enable direction indicator].
  3. Select the arrow indicating direction on the camera screen.
  4. To make the direction selected in Step 3 appear on the screen, go to [Indicator] and select [Enable].
  5. The details can be set as shown below:
    • To show the azimuth information, go to [Azimuth] and select [Enable]. Azimuth information such as N, NE, E, SE, S, SW, W or NW is displayed next to the directional arrow.
    • Enter the description of the direction in [Text].
    • Set the location of direction display (directional arrow, azimuth, and text) for [X] and [Y], respectively.  
  6. To see the set direction display on the video screen, click [Preview]. To return to the setup screen, click [Back].
  7. To apply the settings, click the [Apply] button at the bottom of the page.

Whenever the direction of the camera changes due to pan adjustment after direction is set, the direction information on the current video screen is updated accordingly.

Indicator

Determines the individual display of each direction.

Select an arrow on the screen and click [Enable] to show the arrow image along with its direction information on the camera screen.

Azimuth

Select [Enable] to display azimuth information of the direction.

North is marked as N while northern east, east, southern east, south, southern west, west, and northern west as NE, E, SE, S, SW, W, and NW, respectively.

In [Video & Audio]>[Camera setup], click the (Set North) button to set your desired direction as north.

Text

You can enter description of the direction.

Note

  • You can enter up to 12 characters, which can contain any of the following: a to z, A to Z, 0 to 9, special characters such as . -  # * + - / ( ) _, and spaces.
  • To change the size, color, and transparency of the text, go to [Video & Audio]>[Camera setup]>[OSD], and change the values of [Size], [Color], and [Transparency].
  • Korean or Chinese text typed in under the language preference set to Korean or Chinese will be lost when the language preference is changed.

X/Y

Set the location where the arrow indicating direction displays on the camera video.

The X/Y coordinate values are based on the current camera screen.

Preview

To see the set camera direction information displayed on the camera image, click [Preview]. To return to the setting, click [Back].
