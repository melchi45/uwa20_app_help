Configuration du serveur de données

Lorsqu’un serveur d’aperçu de détail est enregistré, les données de la caméra (comptage de personnes, gestion des files d’attente et données de carte thermique) sont transférées vers le serveur enregistré. Sur la page [Configuration du serveur de données] les informations du serveur d’aperçu de détail peuvent être entrées et enregistrées. Cliquez sur le bouton [Appliquer] ci-dessous après avoir entré les informations nécessaires.

Configuration du serveur de données

Activer le serveur de données

Pour transférer les données de la caméra vers le serveur d’aperçu de détail, sélectionnez [Activer le serveur de données].

Connexion

Les informations du serveur peuvent être entrées pour enregistrer le serveur d’aperçu de détail.

Adresse IP

Entrez l’adresse IP du serveur d’aperçu de détail.

Remarque

  • À côté du champ [IP], un message indiquant l’état de la connexion au serveur est affiché.
    • En cours de connexion : Affiché lors d’une tentative de connexion au serveur d’aperçu de détail. Lorsque le bouton [Appliquer] en bas de la page ou lorsque [Activer le serveur de données] est sélectionné, une tentative de connexion au serveur est tentée.
    • Réussite : Affiché lorsque la connexion au serveur d’aperçu de détail est établie.
    • Échec : Affiché lorsque la connexion au serveur d’aperçu de détail a échoué. Si ce message persiste, assurez-vous que les informations du serveur d’aperçu de détail sur la page [Configuration du serveur de données] sont correctes.

Port

Entrez la valeur de port du serveur d’aperçu de détail.

La valeur du port peut être 80 ou comprise entre 1024 et 65535.

ID

Entrez l’ID du compte pour vous connecter au serveur d’aperçu de détail.

L’ID peut contenir jusqu’à 8 lettres, mais n’autorise pas les caractères spéciaux : #%&+=\:<>"'

Mot de passe

Entrez le mot de passe du compte pour vous connecter au serveur d’aperçu de détail.

Le mot de passe peut contenir entre 8 et 20 lettres, mais n’autorise pas les caractères spéciaux : #%&+=\:<>"'

Nom du magasin

Entrez le nom de votre magasin.

Le nom du magasin est utilisé par le serveur d’aperçu de détail pour distinguer votre caméra.
