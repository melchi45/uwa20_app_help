Transfert

Le relais est une fonction permettant à la caméra récepteur de passer à la position PTZ prédéfinie lorsqu'un événement se produit. Sur cette page, vous pouvez régler la caméra et la position prédéfinie afin qu’elles passent à la position prédéfinie. Lorsque vous avez complété le réglage, cliquez sur le bouton [Appliquer] en bas de la page.

Caméra récepteur

Vous pouvez ajouter ou supprimer des caméras PTZ pour la réception. Jusqu’à 32 caméras récepteurs peuvent être enregistrées.

Les informations des caméras enregistrées en tant que caméra récepteur sont affichées.

Comment ajouter une caméra récepteur

  1. Cliquez sur le bouton [Ajouter].
  2. Dans la fenêtre [Ajouter caméra], entrez les informations sur la caméra PTZ afin de recevoir le relais.
    • No. : Définissez le numéro de la caméra récepteur. Dans le sous-menu [Analyse], le numéro de la caméra récepteur est utilisé lors du réglage de la caméra récepteur de relais.
    • Type d’IP : Réglez le type d’adresse IP entre IPv4 et IPv6.
    • HTTP/HTTPS: Sélectionnez le type de communication à utiliser entre HTTP et HTTPS.
    • Adresse IP : Entrez l’adresse IP de la caméra récepteur.
    • Port : Entrez le numéro de port de la caméra récepteur.
    • Utilisateur : Entrez l’identifiant d’accès afin d’accéder à la caméra récepteur.
    • Mot de passe : Entrez le mot de passe afin d’accéder à la caméra récepteur.
    • Préréglage n° : Entrez le numéro prédéfini de la position à laquelle la caméra récepteur passera. Le numéro prédéfini doit être réglé à l'avance dans la caméra PTZ de réception. Dans le menu de relais, entrez le numéro PTZ prédéfini.
  3. Cliquez sur [OK].

Comment supprimer une caméra récepteur

  1. Cliquez sur une case à cocher dans la liste des caméras récepteur et sélectionnez la caméra que vous souhaitez supprimer.
  2. Cliquez sur le bouton [Effacer] et lorsque la fenêtre de confirmation apparaît, cliquez sur [OK].

Transfert

Vous pouvez définir un numéro de transfert pour que la mise au point de la caméra PTZ puisse se déplacer vers la zone préréglée en cas de survenance d’un événement dans la caméra fixe.

Préréglage est un ensemble de valeurs prédéterminées de panoramique, inclinaison et de grossissement du zoom. Allez à la partie [PTZ] > [Configuration PTZ] > [Préréglage] > [Configuration préréglée] pour définir le préréglage PTZ.

Lorsque vous avez complété le réglage, cliquez sur le bouton [Appliquer] en bas de la page.

Remarque

  • Le numéro de transfert défini sur cette page s’affiche dans la liste « Transfert n° ». Vous pouvez afficher la liste du numéro de transfert dans chaque page de [Détection de mouvement], [IVA], et [Modification] dans le sous-menu [Analyse].

Définir le n° de transfert

Le numéro de transfert peut être ajouté, et les données de préréglage peuvent être définies pour chaque numéro.

Ajouter le n° de transfert

  1. Cliquez sur le bouton [Ajouter].
  2. Dans la fenêtre [Ajouter], réglez comme suit :
    • No. : Sélectionnez un numéro de transfert. Les numéros spécifiés par N° de transfert s’affichent sous forme de liste lorsque vous accédez à [Transfert] > [Transfert n°] dans chaque page de [Détection de mouvement], [IVA], et [Détection de sabotage] dans le sous-menu [Analyse].
    • Canal : Le numéro de canal de la caméra PTZ s’affiche automatiquement et ne peut pas être modifié.
    • Préréglage n° : Saisissez un numéro de préréglage. Les numéros préréglés doivent être définis à l’avance dans la partie [PTZ] > [Configuration PTZ] > [Préréglage] > [Configuration préréglée].
  3. Cliquez sur [OK].

Supprimer le n° de transfert

  1. Cliquez sur une case à cocher dans la liste des numéros de transfert et sélectionnez les numéros à supprimer.
  2. Cliquez sur le bouton [Effacer] et lorsque la fenêtre de confirmation apparaît, cliquez sur [OK].
  3. Cliquez sur [OK].
