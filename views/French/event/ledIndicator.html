Voyant LED

Règle l'action de l'indicateur LED sur la caméra.

La fonction de détection de stationnement de la caméra de guidage de stationnement et la fonction de l'indicateur LED peuvent être utilisées ensemble. La fonction de détection de stationnement peut être définie sous [Analyse]>[Détection de stationnement].

Après avoir terminé la configuration, cliquez sur le bouton [Appliquer] en bas de la page.

Contrôle LED

Sélectionnez un type de LED pour définir le mouvement pour chaque type de LED.

Type LED

Sélectionnez [Séparé] lorsque l’indicateur LED de la caméra de détection de stationnement est divisé à gauche/droite. Sélectionnez [Intégré] lorsque l’indicateur LED n’est pas divisé.

LED 1/LED 2

Lorsque vous sélectionnez [Séparé] sous [Type LED], vous avez le choix entre la [LED 1] et la [LED 2].

Les champs supérieurs de [LED 1] et [LED 2] sont utilisés pour déterminer la couleur à afficher lorsqu’un événement se produit, tandis que leurs champs inférieurs sont utilisés pour déterminer la couleur à afficher lorsqu’aucun événement ne s’est produit. Par exemple, si vous définissez la couleur comme étant rouge et orange dans les champs supérieurs de [LED 1] et [LED 2] respectivement, alors la LED 1 s’affichera en rouge et la LED 2 en orange lorsqu’un événement de détection de stationnement se produira.

Vous pouvez également saisir du texte pour indiquer quelle couleur est utilisée dans quelle situation. Par défaut, le champ supérieur affiche « Occupied » tandis que le champ inférieur affiche « Vacant », et vous pouvez le modifier à votre guise.

Sur l’évènement/Pas sur l’événement

Si vous sélectionnez [Intégré] sous [Type LED], les options [Sur l’évènement] et [Pas sur l’événement] s’affichent.

Vous pouvez définir les couleurs des LED à afficher lorsqu'un événement se produit et lorsqu'aucun événement ne se produit sous [Sur l’évènement] et [Pas sur l’événement]. Par exemple, si vous avez sélectionné le rouge, l’indicateur LED deviendra rouge lorsqu’un événement de détection de stationnement se produira.

Vous pouvez également saisir du texte pour indiquer quelle couleur est utilisée dans quelle situation. Par défaut, le champ [Sur l’évènement] affiche « Occupied » tandis que le champ [Pas sur l’événement] affiche « Vacant », et vous pouvez le modifier à votre guise.

indicateurs d’envoi de données

Si vous sélectionnez [Intégré] sous [Type LED], l’option [indicateurs d’envoi de données] s’affiche.

Vous pouvez régler les deux caméras avec indicateurs LED de façon à ce qu’elles envoyent/reçoivent des données l'une de l'autre. Cela signifie que l'indicateur LED de la caméra 2 peut utiliser les données analysées par la caméra 1.

Utilisation de la caméra de réception

  1. Cliquez sous [Caméra de réception] sur [Activer].
  2. Saisissez les informations relatives à la caméra de réception.
    • Type d’IP : Sélectionnez un type d'adresse IP : IPv4 ou IPv6.
    • HTTP/HTTPS : Sélectionnez un mode de communication : HTTP ou HTTPS.
    • Adresse IP : Saisissez l'adresse IP de la caméra de réception.
    • Port : Saisissez le port de la caméra de réception.
    • ID : Saisissez l'ID d'accès nécessaire pour accéder à la caméra de réception.
    • Mot de passe : Saisissez le mot de passe nécessaire pour accéder à la caméra de réception.
  3. Cliquez sur [Appliquer].
