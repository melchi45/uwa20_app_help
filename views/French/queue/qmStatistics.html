Rechercher

La gestion des files d'attente est une fonction utilisée pour surveiller le nombre moyen de personnes séjournant dans une zone spécifiée et la durée totale du séjour grâce à une analyse des images de la caméra. Le nombre de personnes en file d'attente et les données statistiques sur le temps de mise en file d'attente peuvent être recherchés et visualisés à partir de la page « Recherche » de la gestion des files d'attente.

Remarque

  • Pour utiliser la fonction de recherche dans la gestion des files d'attente, vous devez sélectionner [Activer la gestion des files d'attente] dans [Statistiques]>[Gestion des files d'attente]>[Configuration] et définir la zone de gestion des files d'attente.

Rechercher

Le jeu de règles de file d'attente défini dans [Statistiques]>[Gestion des files d'attente]>[Configuration] s'affiche à l'écran.  

Occupation

Les données d'analyse en temps réel pour chaque règle de file d'attente sont affichées.

Aujourd'hui

Nombre moyen de personnes

Le nombre de personnes comptées dans chaque règle de file d'attente entre 00:00 et 24:00 est indiqué dans un graphique affiché par heure.

Lorsque vous sélectionnez le nom d'une règle de file d'attente située en haut à droite du graphique, seul le graphique correspondant s'affiche en cyan. Si vous placez la souris au-dessus du graphique, une bulle indiquant le nombre de personnes à l'heure sélectionnée apparaît.

Durée totale

La durée totale de mise en file d'attente est analysée par les règles de file d'attente et les niveaux de détection (haut et moyen), et s’affiche dans le graphique.

Le niveau de détection est spécifié dans [Niveau de détection] sur la page [Configuration], et le niveau de mise en file d'attente est classé en « Élevée » et « Moyenne », en fonction du nombre de personnes dans la file d'attente.

Par exemple, « File d’attente 1- Élevée » indique la durée totale pendant laquelle il y a un grand nombre de files d'attente dans « File d'attente 1 » comme niveau de détection « Élevée », et « File d'attente 1- Moyenne » indique la durée totale pendant laquelle il y a un grand nombre de files d'attente dans la file d'attente 1 comme niveau de détection « Moyenne ».

Hebdomadaire

La tendance pour le nombre moyen de personnes par file d'attente et la date pour les 7 derniers jours, à partir d’aujourd'hui, est représentée dans un graphique.

Si vous placez la souris au-dessus du graphique, une bulle affichant les informations concernant le nombre de personnes à la date sélectionnée s'affichera en une valeur numérique exacte.  

Rechercher

Date

Lorsque vous sélectionnez [Récent], vous pouvez effectuer une recherche par jour, semaine ou mois à partir de l'heure actuelle.

Lorsque vous sélectionnez [Période], vous pouvez spécifier une période par jour, mois ou année et effectuer une recherche en conséquence. Sélectionnez l'icône du calendrier pour changer la date.

Règle

Sélectionnez les conditions de recherche. La règle de file d'attente et les conditions d'analyse (nombre moyen de personnes, élevée ou moyenne pour le niveau de détection) peuvent être sélectionnées en combinaison. Par exemple, lorsque vous sélectionnez la case à cocher où « File d'attente 1 » et « Moyenne » se rencontrent, le nombre moyen de personnes dans la file d'attente « File d'attente 1 » est recherché. Lorsque vous sélectionnez la case à cocher où « File d'attente 1 » et « Élevée » se rencontrent, la durée totale du nombre de personnes correspondant à « Élevée » pour le niveau de mise en file d'attente de détection dans « File d'attente 1 » est recherchée. Vous pouvez également sélectionner « Moyenne » et « Élevée » correspondant à « File d'attente 1 » pour la recherche.

La règle peut être définie dans [Statistiques]>[Gestion des files d'attente]>[Configuration].

Rechercher les données de la file
  1. Sélectionnez la date que vous souhaitez rechercher dans [Date] dans [Recherche].
  2. Sélectionnez la file d'attente que vous souhaitez rechercher et l'option d'analyse dans [Règle]. Par exemple, si vous souhaitez rechercher le nombre moyen de files d'attente dans « File d'attente 1 » et le temps total pour le nombre de personnes correspondant à « Élevée » pour le niveau de file d'attente de détection dans « File d'attente 1 », cochez la case « File d'attente 1 » et « Moyenne » 'se rencontrent et la case à cocher où « File d'attente 1 » et « Élevée » se rencontrent.
  3. Cliquez sur le bouton [Rechercher]. Les résultats de la recherche seront affichés dans [Résultats].

Résultats

Si vous recherchez des données de file d'attente, les résultats seront affichés dans un graphique et un tableau.

Si vous recherchez « un jour récent », les données par heure de 00:00 à 24:00 le jour actuel seront affichées. Si vous recherchez « une semaine récente », les données par date des 7 jours précédant le jour actuel seront affichées. Si vous recherchez « un mois récent », les données par date pour les 30 jours précédant immédiatement le jour actuel seront affichées.

Si vous sélectionnez « Période » et que vous spécifiez un jour, un mois et une année spécifiques, les données par heure de 00:00 à 24:00 pour la date désignée et les données par date pour le mois et l'année indiqués s'afficheront.

Les données de la file d'attente sont affichées séparément dans « Nom de la file d'attente-Moyenne », « Nom de la file d'attente-Élevée » et « Nom de la file d'attente-Moyenne » dans la combinaison du nom de la file et de la méthode d'analyse (nombre moyen de personnes niveau de file d'attente). Si vous sélectionnez une option dans un graphique, seul le graphique correspondant sera affiché en Cyan.  De plus, si vous placez la souris au-dessus d'un graphique, une bulle affichant les données à l'heure sélectionnée en une valeur numérique exacte apparaîtra.  

Remarque

  • Les données recherchées sont actualisées toutes les 15 minutes. Les résultats de la recherche diffèrent souvent des résultats en temps réel affichés sur page[Paramètres].

Télécharger

Lorsque vous cliquez sur le bouton [Télécharger] après la recherche, vous pouvez télécharger les résultats de la recherche. Les résultats de la recherche peuvent être téléchargés aux formats de fichier .txt, et le fichier téléchargé sera sauvegardé dans le répertoire par défaut du navigateur. Vous pouvez saisir jusqu'à 64 caractères, y compris des caractères alphabétiques et des chiffres, pour le nom de fichier.
