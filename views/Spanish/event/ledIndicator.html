Indicador LED

Establece la acción del indicador LED en la cámara.

La función de detección de aparcamiento de la cámara de guía de estacionamiento y la función de indicador LED pueden funcionar simultáneamente. La función de detección de aparcamiento se puede configurar en el menú [Analítica]> [Detección de aparcamiento].

Una vez completada la configuración, haga clic en el botón [Aplicar] en la parte inferior de la página.

Control LED

Seleccione un tipo de LED para establecer el movimiento para cada tipo de LED.

Tipo de LED

Seleccione [Separado] cuando el indicador LED de la cámara de detección de aparcamiento esté dividido a izquierda/derecha. Seleccione [Integrado] cuando el indicador LED no esté dividido.

LED 1/LED 2

Cuando se selecciona [Separado] en [Tipo de LED], se pueden configurar [LED 1] y [LED 2].

Los campos superiores de [LED 1] y [LED 2] se usan para determinar el color a mostrar cuando ocurre un evento, mientras que sus campos inferiores se usan para determinar el color a mostrar cuando no ha ocurrido ningún evento. Por ejemplo, si establece que el color sea rojo y naranja en los campos superiores del [LED 1] y [LED 2], respectivamente, el LED 1 mostrará rojo y el LED 2 naranja cuando se produzca un evento de detección de aparcamiento.

Además, puede escribir el texto para indicar qué color se utiliza para cada situación. Por defecto, el campo superior muestra "Ocupado» y el inferior muestra "Vacante", y usted puede modificarlo como desee.

En evento/No en evento

Si se selecciona [Integrado] en [Tipo de LED], se mostrarán las opciones [En evento] y [No en evento].

Configure los colores del LED para que se muestren cuando ocurra un evento y cuando no ocurra ningún evento utilizando las opciones [En evento] y [No en evento], respectivamente. Por ejemplo, si ha seleccionado el color rojo, el indicador LED se iluminará en rojo cuando se produzca un evento de detección de aparcamiento.

Además, puede escribir el texto para indicar qué color se utiliza para cada situación. Por defecto, el campo [En evento] muestra "Ocupado» mientras que el campo [No en evento] muestra "Vacante", y usted puede modificarlo como desee.

Envío de datos del indicador

Cuando se selecciona [Integrado] en [Tipo de LED], se mostrará la opción [Envío de datos del indicador].

Configure las dos cámaras con indicadores LED para enviar/recibir datos entre sí. En otras palabras, los datos analizados por la cámara 1 se pueden usar para controlar el indicador LED de la cámara 2.

Uso de la cámara receptora

  1. Haga clic en [Activar] en la [Cámara receptora].
  2. Introduzca el Información para la cámara receptora.
    • Tipo IP: Seleccione el tipo de dirección IP entre IPv4 e IPv6.
    • HTTP/HTTPS: Seleccione el modo de comunicación entre HTTP y HTTPS.
    • Dirección IP: Introduzca la dirección IP de la cámara receptora.
    • Puerto: Introduzca el puerto para la cámara receptora.
    • ID: Introduzca el ID de acceso para acceder a la cámara receptora.
    • Contraseña: Introduzca la contraseña para acceder a la cámara receptora.
  3. Haga clic en [Aplicar].
