IVA

영상 분석에 대한 규칙을 설정하고 이벤트 규칙 조건을 만족하는 움직임을 감지하면 IVA 이벤트를 발생시킬 수 있습니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오. IVA 이벤트에 대한 설정은 [이벤트]>[이벤트 설정] 페이지의 [IVA]에 동일하게 적용됩니다.

알아두기

  • [객체] 옵션에서 특정 객체를 선택한 경우, 카메라 영상에서 선택한 객체만을 감지하며, 해당 객체가 IVA 이벤트 규칙을 만족하는 경우 IVA 이벤트가 발생합니다. 반면에 [객체]에서 어떤 옵션도 선택하지 않은 경우, 움직이는 물체를 감지하며 움직이는 물체가 IVA 이벤트 규칙을 만족하는 경우 IVA 이벤트가 발생합니다.

채널 선택

카메라 채널을 선 택한 후 채널별로 IVA 세부 항목을 설정할 수 있습니다.

라이브 페이지나 재생 페이지에서 가장 최근에 선택한 채널 번호가 자동으로 선택되어 있습니다.

채널 1은 라이브 영상을 보여 주는 채널이며, 채널 2는 디지털 PTZ 영상을 보여 주는 채널입니다. [IVA]는 채널 1에서만 설정할 수 있습니다.

고정 카메라의 채널은 번호로 표기되고, PTZ 채널은 'PTZ'로 표기됩니다.

채널 2~5번은 장착된 개별 카메라 영상이고, 채널 1은 2~5번 채널의 카메라 영상을 하나로 연결하여 파노라마 형식으로 보여주는 영상입니다. 채널 6번과 7번은 채널 1 영상에 대해 디지털 PTZ가 적용된 영상입니다. [IVA]는 채널 1에서만 설정할 수 있습니다.

채널 1은 라이브 영상을 보여 주는 채널이며, 채널 2와 3은 디지털 PTZ 영상을 보여 주는 채널입니다. [IVA]는 채널 1에서만 설정할 수 있습니다.

IVA (지능형 영상 분석)

IVA 사용

IVA 이벤트를 사용하려면 [IVA 사용]을 선택합니다.

타입

모든 프리셋에 동일한 IVA(지능형 영상 분석)을 설정하려면 [전역]을 선택합니다. PTZ 프리셋 별로 각각 IVA(지능형 영상 분석)을 설정하려면[프리셋]을 선택합니다. [프리셋 설정] 페이지에서 프리셋을 설정한 경우에만 [프리셋] 옵션이 활성화됩니다.

[채널 선택]에서 [PTZ] 채널을 선택한 경우에만 [타입] 옵션이 나타납니다.

타입

모든 프리셋에 동일한 움직임 감지를 설정하려면 [전역]을 선택합니다. 팬/틸트 프리셋 별로 각각 IVA(지능형 영상 분석)을 설정하려면[프리셋]을 선택합니다.

PT 설정 페이지에서 프리셋을 설정한 경우에만 [프리셋] 옵션이 활성화됩니다. [프리셋]을 선택하여 프리셋 별로 움직임 감지 설정을 지정하는 경우, 세부 설정 항목 중 일부 기능은 설정할 수 없습니다.

알아두기

  • 카메라 영상이 겹치는 중첩 영역에서 움직임이 발생하는 경우 IVA 이벤트 성능이 저하되거나 오작동할 수 있습니다.

가상선

사용자가 설정한 가상선과 방향을 기준으로 객체가 지나가는 것을 감지합니다.

가상선 설정하기

카메라 영상 화면의 원하는 위치에서 라인의 시작점과 끝점을 각각 클릭하면 가상선이 설정되며 [라인]의 숫자 버튼 색상이 변경됩니다.

가상선을 설정한 뒤 [이름]필드에 설정한 가상선의 이름을 입력할 수 있습니다.

가상선의 가운데 화살표를 클릭하면 방향을 A로부터 B 방향, B로부터 A 방향, 양방향으로 변경하거나 [카테고리]의 [방향]에서 원하는 방향을 선택할 수도 있습니다. 가상선은 최대 8개까지 설정할 수 있습니다.

가상선 변경하기

설정한 가상선의 꼭지점을 움직여 가상선의 크기를 변경할 수 있습니다. 가상선을 마우스로 클릭하고 드래그하여 가상선의 위치를 변경할 수 있습니다.

가상선 삭제하기

가상선을 삭제하려면 화면에서 해당 영역에 마우스 오른쪽 버튼을 클릭한 후 삭제 확인 창에서 [확인] 버튼을 클릭합니다.

PTZ 조정하기

[채널 선택]에서 [PTZ] 채널을 선택하면 PTZ 조그 패널이 나타납니다.

  • 아이콘을 마우스로 드래그하면 카메라가 팬과 틸트로 이동합니다.
  • 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 아이콘을 클릭하여 화면을 확대하거나 아이콘을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
  • 아이콘을 클릭하여 포커스를 가까이(near)이동하거나 아이콘을 클릭하여 포커스를 멀리(far)이동합니다.
  • 카메라 포커스를 자동으로 한 번 맞추려면 버튼을 클릭합니다.

라인

가상선이 추가되면 순서대로 [라인]의 숫자 버튼 색상이 변경됩니다. 숫자 버튼을 클릭하면 해당 가상선이 영상 화면에 나타납니다.

이름

[라인]에서 선택한 가상선의 이름을 확인하거나 변경할 수 있습니다.

객체

IVA 이벤트를 발생하기 위한 객체 타입을 선택합니다.

예를 들어 객체 타입에서 '사람'을 선택한 경우, 사람이 가상선을 지나가면 IVA 이벤트가 발생합니다. 이벤트 발생 시 핸드오버가 동작하도록 하거나 이벤트 동작이 실행하도록 설정할 수 있습니다.

알아두기

  • 객체를 선택한 경우 카메라 영상에서 선택한 객체만을 감지합니다. 객체를 선택하지 않는 경우, 카메라 영상에서 움직임을 감지합니다. 예를 들어 [객체]에서 어떤 옵션도 선택하지 않은 경우, IVA 이벤트 규칙 조건을 만족하는 움직이는 물체를 감지하면 IVA 이벤트가 발생합니다.

카테고리

  • 지나감: 설정한 규칙에 따라 가상선을 지나가는 객체를 감지했을 때 이벤트를 발생시키려면 체크박스를 선택합니다.
  • 방향: 객체가 가상선을 지나가는 방향을 설정합니다.

핸드오버

핸드오버는 고정 카메라에서 가상선을 기준으로 움직이는 물체를 감지하면 PTZ 카메라가 미리 설정해 둔 위치로 이동하는 기능입니다.

  • 사용 안 함: 핸드오버 기능을 사용하지 않습니다.
  • 스마트줌: 가상선을 지나가는 물체를 감지하면, PTZ 카메라가 가상선의 중앙 위치를 중심으로 영상을 확대하여 보여줍니다. 고정 카메라와 PTZ 카메라가 연동할 수 있도록 영상 좌표값은 미리 설정해 두어야 하며, [PTZ] > [캘리브레이션] 메뉴에서 설정할 수 있습니다.
  • 핸드오버 번호: 가상선을 지나가는 물체를 감지하면, PTZ 카메라가 프리셋 위치로 이동합니다. 프리셋 정보를 가진 핸드오버 번호를 리스트 목록에서 선택합니다. 리스트 목록은 [이벤트] > [핸드오버]에서 추가하거나 변경할 수 있습니다.

알아두기

  • 가상선 별로 핸드오버 기능을 설정할 수 있으며, 핸드오버 기능은 가상선을 먼저 설정한 후 사용할 수 있습니다.
  • [채널 선택]에서 고정 카메라 채널을 선택한 경우에만 '핸드오버' 메뉴가 나타납니다. PTZ 채널을 선택하면 해당 메뉴가 보이지 않습니다.

가상 영역

사용자가 가상 영역을 설정하여 해당 영역에 침입, 들어감, 나감, 나타남(사라짐), 배회의 영상 규칙을 적용하여 물체의 움직임을 감지합니다.

사용자가 가상 영역을 설정하여 해당 영역에 들어감, 나감의 영상 규칙을 적용하여 물체의 움직임을 감지합니다.

가상 영역 설정하기

미리보기 화면에서 마우스로 꼭지점 4개를 선택하거나 마우스로 드래그하여 사각형을 그리면 가상 영역이 설정됩니다.

가상 영역은 최대 8개까지 설정할 수 있으며, 영역별로 영상 분석 규칙을 설정할 수 있습니다.

사용자가 원하는 경우 가상 영역을 설정한 뒤 [이름]필드에 설정한 가상 영역에 대해 이름을 입력할 수 있습니다.

가상 영역 변경하기

설정한 가상 영역의 꼭지점을 움직여 가상 영역의 크기를 변경할 수 있습니다.

다각형을 만들려면 만들어진 사각형의 선을 선택한 후 + 기호가 나타나면 +기호를 클릭하여 꼭지점을 추가합니다. 최대 4개의 점을 추가하여 원하는 다각형을 만들 수 있습니다.

설정한 가상 영역을 이동하려면 마우스로 해당 영역을 클릭하여 드래그합니다. 선을 클릭한 후 꼭지점이 생기면 꼭지점을 이동하여 다각형 가상 영역을 설정할 수 있습니다.

가상 영역 삭제하기

가상 영역을 삭제하려면 화면에서 해당 영역에 마우스 오른쪽 버튼을 클릭한 후 삭제 확인 창에서 [확인] 버튼을 클릭합니다.

영역

가상 영역이 추가되면 순서대로 [영역]의 숫자 버튼 색상이 변경됩니다. 숫자 버튼을 클릭하면 해당 가상 영역이 미리보기 화면에 나타납니다.

이름

[영역]에서 선택한 가상 영역의 이름을 확인하거나 변경할 수 있습니다.

객체

IVA 이벤트를 발생하기 위한 객체 타입을 선택합니다.

예를 들어 '사람'을 선택한 경우, 가상 영역에서 사람을 감지한 경우에만 IVA 이벤트가 발생합니다. 이벤트 발생 시 핸드오버가 동작하도록 하거나 이벤트 동작이 실행하도록 설정할 수 있습니다.

알아두기

  • 객체를 선택한 경우 카메라 영상에서 선택한 객체만을 감지합니다. 객체를 선택하지 않는 경우, 카메라 영상에서 움직임을 감지합니다. 예를 들어 [객체]에서 어떤 옵션도 선택하지 않은 경우, IVA 이벤트 규칙 조건을 만족하는 움직이는 물체를 감지하면 IVA 이벤트가 발생합니다.

카테고리

가상 영역별로 적용할 규칙을 설정할 수 있습니다.

  • 침입: 설정한 영역 내부에 움직이는 물체를 감지하면 이벤트를 발생시킵니다.
  • 침입: 설정한 영역 내부에 [객체]에서 선택한 객체를 감지하면 이벤트를 발생시킵니다.
    • 최소 관찰 시간(초): 감지 후 이벤트를 발생하기 위한 최소 시간을 설정합니다. 설정한 시간보다 더 오래 움직임이 지속될 때 이벤트를 발생시킵니다.
  • 들어감: 움직이는 물체가 사용자가 지정한 영역의 외부에서 내부로 들어올 때 이벤트를 발생시킵니다.
  • 들어감: [객체]에서 선택한 객체가 사용자가 지정한 영역의 외부에서 내부로 들어올 때 이벤트를 발생시킵니다.
  • 나감: 움직이는 물체가 사용자가 지정한 영역의 내부에서 외부로 나갈 때 이벤트를 발생시킵니다.
  • 나감: [객체]에서 선택한 객체가 사용자가 지정한 영역의 내부에서 외부로 나갈 때 이벤트를 발생시킵니다.
  • 나타남(사라짐): 사용자가 지정한 영역 내부에 존재하지 않았던 물체가 영역 라인을 통과하지 않고 영역 내부에 나타나 일정시간 머무르거나, 영역 내부에 존재하던 물체가 사라지면 이벤트를 발생시킵니다.
  • [나타남(사라짐)] 옵션의 경우 다른 옵션과 달리 [객체]에서 선택한 객체가 아닌 움직이는 물체를 감지합니다.
    • 최소 관찰 시간(초): 나타남(사라짐)을 감지하기 위한 물체의 최소 지속 시간을 설정합니다. 가상 영역에서 실행 시간동안 존재하던 물체가 사라지면 이를 사라짐으로 감지하고, 실행 시간 동안 존재하지 않던 물체가 나타나면 나타남으로 감지합니다.
  • 배회: 설정한 가상 영역 내에서 배회하는 움직임을 감지하면 이벤트를 발생시킵니다.
  • 배회: 설정한 가상 영역 내에서 [객체]에서 선택한 객체가 배회하는 것을 감지하면 이벤트를 발생시킵니다.
    • 최소 관찰 시간(초): 배회하는 물체를 감지하고 이벤트를 발생하기 위한 최소 시간을 설정합니다. 설정한 시간보다 더 오래 배회하는 물체를 감지하면 이벤트를 발생시킵니다.
    • 최소 관찰 시간(초): 배회하는 객체를 감지하고 이벤트를 발생하기 위한 최소 시간을 설정합니다. 설정한 시간보다 더 오래 배회하는 객체를 감지하면 이벤트를 발생시킵니다.

알아두기

  • 배회 감지는 다른 IVA 이벤트 감지와 함께 사용할 경우, 단독으로 사용하는 것보다 성능이 다소 저하될 수 있습니다.
  • 다음과 같은 경우 객체가 사라진 것으로 인식하여 배회 감지 기능이 동작하지 않을 수 있습니다.
    • 배회하고 있는 객체가 다른 객체와 겹치는 경우
    • 배회하고 있는 객체가 일정 시간동안 움직이지 않는 경우
    • 배회하고 있는 객체와 근접 영역에 이동체가 있는 복잡한 환경이 발생한 경우

핸드오버

핸드오버는 고정 카메라에서 IVA 이벤트가 발생하면 PTZ 카메라가 해당 화면 영역이나 미리 설정한 프리셋 위치로 이동하는 기능입니다

  • 사용 안 함: 핸드오버 기능을 사용하지 않습니다.
  • 스마트줌: 가상 영역에서 IVA 이벤트가 발생하면, PTZ 카메라가 가상 영역으로 이동하여 가상 영역 중앙 위치를 중심으로 영상을 확대하여 보여줍니다. 고정 카메라와 PTZ 카메라가 연동할 수 있도록 영상 위치 좌표값은 미리 설정해 두어야 하며, [PTZ] > [캘리브레이션] 메뉴에서 설정할 수 있습니다.
  • 핸드오버 번호: 가상 영역에서 IVA 이벤트가 발생하면, PTZ 카메라가 프리셋 위치로 이동합니다. 프리셋 정보를 가진 핸드오버 번호를 리스트 목록에서 선택합니다. 리스트 목록은 [이벤트] > [핸드오버] 메뉴에서 추가하거나 변경할 수 있습니다.

  • 가상 영역 별로 핸드오버 기능을 설정할 수 있으며, 핸드오버 기능은 가상 영역을 먼저 설정한 후 사용할 수 있습니다.

제외 영역

가상선과 가상 영역에서 분석 규칙을 적용할 때 움직임을 감지하지 않을 제외 영역을 설정합니다.

제외 영역 설정하기

미리보기 화면에서 마우스로 꼭지점 4개를 선택하거나 마우스로 드래그하여 사각형을 그리면 제외 영역이 설정됩니다. 제외 영역은 최대 8개까지 설정할 수 있습니다.

제외 영역 변경하기

설정한 제외 영역의 꼭지점을 움직여 제외 영역의 크기를 변경할 수 있습니다.

다각형을 만들려면 만들어진 사각형의 선을 선택한 후 + 기호가 나타나면 +기호를 클릭하면 꼭지점이 추가됩니다. 최대 4개의 점을 추가하여 원하는 다각형을 만들 수 있습니다.

제외 영역을 선택하고 마우스로 드래그하면 제외 영역의 위치를 이동할 수 있습니다.

제외 영역 삭제하기

제외 영역을 삭제하려면 화면에서 해당 영역에 마우스 오른쪽 버튼을 클릭한 후 삭제 확인 창에서 [확인] 버튼을 클릭합니다.

알아두기

공통 설정

가상선과 가상 영역에 공통으로 적용될 민감도와 감지할 물체의 최소, 최대 크기를 설정합니다.

사용 환경에 따라 감지 정확도를 높이고 오감지를 최소화 하기 위해 적용할 수 있는 설정입니다.

민감도

가상선과 가상 영역에 대한 움직임 감지 민감도를 설정합니다. 배경과 물체의 구분이 명확한 환경에서는 민감도를 낮게 설정하고, 어두워서 배경과 물체의 구분이 명확하지 않은 환경에서는 민감도를 높게 설정합니다.

크기

가상선과 가상 영역에서 움직임을 감지할 물체의 최소 크기와 최대 크기를 설정합니다. 오른쪽 하단의 꼭지점을 클릭한 후 마우스를 드래그하여 크기를 변경할 수 있습니다. 크기를 변경하면 [크기]에서 [최소] 값과 [최대] 값도 변경됩니다.
