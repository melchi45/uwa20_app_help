발열 감지

얼굴 온도를 기준으로 발열 상태를 감지할 수 있습니다. 발열 상태를 감지할 때 적용할 기준이나 세부 사항을 설정할 수 있습니다.  

설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오. 발열 감지 이벤트에 대한 설정은 [이벤트]>[이벤트 설정] 페이지의 [발열 감지]에 동일하게 적용됩니다.

발열 감지

발열 감지 사용

발열 감지 이벤트를 사용하려면 [발열 감지 사용]을 선택합니다.

온도 감지

감지한 얼굴 영역 기반으로 카메라가 최고 온도 값을 추출하고 발열 감지 이벤트를 발생시키는데 적용되는 세부 사항을 설정할 수 있습니다.

감지 영역 오버레이

[사용]을 선택하면 감지 영역을 표시합니다. 얼굴 검출이 동작할 해당 영역을 표시하며, 영역 밖의 얼굴은 검출하지 않습니다.

감지 온도

발열 감지 이벤트를 발생시키는데 적용되는 온도를 입력합니다.

최소 관찰 시간 (초)

발열 감지 이벤트가 발생하기 위한 발생 조건의 최소 유지 시간을 설정합니다. [기준 감지 영역]에서 설정한 얼굴 영역의 최고점 온도가 [감지 온도]에서 설정한 온도를 초과하여 유지되는 시간이 [최소 시간 관찰 시간] 설정 값보다 긴 경우 발열 감지 이벤트가 발생한다.

발열 감지 경고음

발열 감지 시 경고음을 발생하려면 [사용]을 선택합니다.

기준 감지 영역

얼굴의 최고점 온도를 기준으로 발열 상태를 감지합니다. [기준 감지 영역]에서 최고점 온도값을 추출할 얼굴 영역을 조정할 수 있습니다.

[초기화]를 클릭하면 설정 내역이 취소되며, 얼굴 영역이 원래의 감지 영역 크기와 위치로 변경됩니다.

  • 수평 비율: 얼굴 영역의 가로 축소 비율을 조정하여, 얼굴 영역의 크기를 조정할 수 있습니다.
  • 수직 비율: 얼굴 영역의 세로 축소 비율을 조정하여, 얼굴 영역의 크기를 조정할 수 있습니다.
  • 위치: 상하좌우 버튼으로 얼굴 영역의 위치를 조정할 수 있습니다.

보정

카메라가 얼굴 온도를 감지할 때 적용할 온도 보정 기능의 상세 항목을 설정할 수 있습니다.

사용자 지정 오프셋 사용

측정 물체의 실제 온도와 카메라가 감지한 온도 사이의 오차를 보정하는 온도값을 설정할 수 있습니다. 온도값을 설정하면, 설정된 온도값만큼 더해진 카메라 감지 온도값이 표시됩니다. 사용자가 지정한 오프셋을 사용하려면 [사용자 지정 오프셋 사용] 체크박스를 선택합니다.

주변 온도 사용

카메라가 설치된 환경의 현재 온도를 설정할 수 있습니다. 주변 온도를 사용하여 카메라의 감지 온도를 보정하려면 [주변 온도 사용] 체크박스를 선택합니다.

주변 습도 사용

카메라가 설치된 환경의 현재 습도를 설정할 수 있습니다. 주변 습도를 사용하여 카메라의 감지 온도를 보정하려면 [주변 습도 사용] 체크박스를 선택합니다.

측정 거리

측정 거리를 사용하여 카메라 감지 온도 보정에 적용할 수 있도록, 피측정자와 카메라 사이의 거리를 설정합니다. 정

블랙바디 보정 사용

블랙바디를 활용하여 온도 보정을 하려면 [블랙바디 보정 사용]을 선택합니다. 블랙바디 보정 사용 시 다음과 같은 옵션을 설정하여 온도 보정을 할 수 있습니다.

  • 블랙바디 온도: 블랙바디 운용 온도 값을 설정합니다.
  • 블랙바디 온도 수정: 입력한 블랙바디 운용 온도 값과 실제 온도와의 오차를 설정합니다.
  • 블랙바디 방사율: 블랙바디와의 운용 방사율값을  설정합니다.
  • 블랙바디와의 거리: 블랙바디와 카메라 사이의 거리를 설정합니다.
  • 블랙바디 설치 높이: 블랙바디의 설치 높이를 설정합니다.  Blackbody의 설치 높이를 설정할 수 있다.
  • 카메라 설치 높이: 카메라의 설치 높이를 설정합니다.
  • 카메라 틸트 각도: 카메라와 블랙바디 사이의 설치 각도 차이를 설정합니다.

블랙바디 오류 경고음

[사용]을 선택하면 카메라가 감지하는 블랙바디의 온도 값과 설정된 블랙바디 온도 값이 다를 경우, 카메라 자체의 경고음이 발생합니다.

캘리브레이션

얼굴 영역의 위치를 열화상 영상과 실화상 영상 좌표에서 공유하기 위해 영상 좌표를 동기화합니다.

카메라 영상 화면에서 마우스로 드래그하여 원하는 위치로 조정하거나 [위치]의 버튼을 이용하여 상하좌우 위치를 세부 조정할 수 있습니다.

위치

버튼을 클릭하여 상하좌우 위치를 조정합니다

공통 설정

얼굴 영역을 카메라가 감지하는데 적용할 공통 설정 항목을 설정할 수 있습니다.

민감도

얼굴 감지 민감도를 설정합니다. 배경과 물체의 구분이 명확한 환경에서는 민감도를 낮게 설정하고, 어두워서 배경과 물체의 구분이 명확하지 않은 환경에서는 민감도를 높게 설정합니다.

크기

발열 상태를 감지할 물체의 최소 크기와 최대 크기를 설정합니다. [초기화]를 클릭하면 설정 내역이 취소되며, 공장 초기화 상태의 최소 크기와 최대 크기로 변경됩니다.

온도

  • [최대 감지 온도 제한]: 설정한 온도를 초과하는 온도는 감지하지 않습니다. 예를 들어 [40]을 입력하는 경우 40도 이상의 온도는 감지하지 않습니다.
  • [최소 감지 온도 제한]: 설정한 온도보다 미만의 온도는 감지하지 않습니다. 예를 들어 [35]을 입력하는 경우 35도 미만의 온도는 감지하지 않습니다.
