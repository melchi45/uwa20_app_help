검색

피플 카운팅은 카메라 영상 분석을 통해 가상의 라인을 지나가는 사람의 수를 카운트하는 것입니다. 피플 카운팅의 '검색'에서는 카운트한 인원 수를 조건별, 시간별로 통계 데이터를 검색하고 확인할 수 있습니다.

알아두기

  • 피플 카운팅의 검색 기능을 사용하려면, 먼저 [통계]>[피플 카운팅]>[설정]에서 [피플 카운팅 사용]을 선택하고 피플 카운팅 룰을 설정해야 합니다.

검색

피플 카운팅 룰별로, 카운트 방식별로 출입한 사람 수의 통계 자료를 보여줍니다.

화면에 [통계]>[피플 카운팅]>[설정]에서 설정한 룰 라인이 화면에 표시됩니다. 또한 룰별로 각 카운트 방식(IN 혹은 OUT)에 따라 카운트한 인원 수 데이터를 표로 보여줍니다. 표의 수치는 오늘 0시부터 현재 시점까지 카운트한 인원수의 합계입니다. 표의 [룰 이름]은 [통계]>[피플 카운팅]>>[설정]에서 설정한 것입니다. [IN]은 룰 라인의 화살표 방향으로 움직인 사람 수를 나타내고, [OUT]은 룰 라인의 화살표 반대 방향으로 움직인 사람의 수를 나타냅니다.

오늘

오늘 0시부터 24시까지 카운트한 인원 수를 시간별로 그래프로 보여줍니다.

룰 이름과 카운트 방식(IN과 OUT)의 조합으로 '룰 이름-IN', '룰 이름-OUT'으로 총 4개 옵션이 있으며, 옵션을 선택하면 해당 그래프만 청록색으로 표시됩니다. 그래프 위에 마우스를 올리면 선택한 시간의 인원 정보를 정확한 수치로 보여주는 풍선 도움말이 나타납니다.

주간

오늘을 기준으로 지난 7일간 날짜별 인원 수 추이를 그래프로 보여줍니다.

룰 이름과 카운트 방식(IN과 OUT)의 조합으로 '룰 이름-IN', '룰 이름-OUT'으로 총 4개 옵션이 있으며, 옵션을 선택하면 해당 그래프만 청록색으로 표시됩니다. 그래프 위에 마우스를 올리면 선택한 날짜의 인원 정보를 정확한 수치로 보여주는 풍선 도움말이 나타납니다.

검색

날짜

[최근]을 선택하면 현재 시간을 기준으로 최근 하루, 일 주, 한 달 단위로 검색할 수 있습니다.

[기간]을 선택하면 일간, 월간, 연간 단위로 기간을 지정하여 검색할 수 있습니다. 달력 아이콘을 선택한 뒤 날짜를 변경할 수 있습니다.

룰

검색할 룰 조건을 선택합니다. 룰(룰 01과 룰 02)과 해당 룰의 카운트 방식(IN과 OUT)을 조합하여 선택할 수 있습니다. 예를 들어 '룰 01'과 'IN'이 만나는 체크박스를 선택하면 '룰 01'에서 지정한 룰 라인의 화살표 방향으로 지나간 인원수를 검색합니다. '룰 02'와 'OUT'이 만나는 체크박스를 선택하면 '룰 02'에서 지정한 룰 라인의 화살표 반대 방향으로 지나간 인원수를 검색합니다.

룰은 [통계]>[피플 카운팅]>[설정]에서 설정할 수 있습니다.

피플 카운팅 데이터 검색하기
  1. [검색]의 [날짜]에서 검색할 날짜를 선택합니다.
  2. [룰]에서 검색할 룰과 각 룰의 카운트 방식을 선택합니다. 예를 들어 '룰 01' 중 지정한 방향으로 움직인 사람 수만 검색하려면 '룰 01'과 'IN'이 만나는 체크박스를 선택합니다.
  3. [검색] 버튼을 클릭합니다. [결과]에 검색 결과가 나타납니다.

결과

피플 카운팅 데이터를 검색한 경우, 결과가 그래프와 표로 나타납니다.

'최근 1일'을 검색한 경우 오늘 0시부터 24시까지 시간별로 데이터를 보여줍니다. '최근 1주'를 검색한 경우 오늘을 기준으로 지난 7일간 데이터를 날짜별로 보여줍니다. '최근 1개월'을 검색한 경우 오늘을 기준으로 지난 30일간 데이터를 날짜별로 보여줍니다.

'기간'을 선택한 후 특정 일, 월, 년을 지정한 경우, 지정한 날의 0시부터 24시까지 시간별 데이터를 보여주며, 지정한 달과 년의 날짜별 데이터를 보여줍니다.

피플 카운팅 데이터는 룰 이름과 카운트 방식(IN과 OUT)의 조합으로 '룰 이름-IN' '룰 이름-OUT'으로 분류되어 나타납니다. 그래프에서 옵션을 선택하면 해당 그래프만 청록색으로 표시됩니다. 인또한 그래프 위에 마우스를 올리면 선택한 시점의 데이터를 정확한 수치로 보여주는 풍선 도움말이 나타납니다.

알아두기

  • 검색되는 데이터는 15분마다 갱신됩니다. 본 검색 결과는 [설정] 페이지에서 보여지는 실시간 결과와 상이할 수 있습니다.

다운로드

검색 실행 후 [다운로드] 버튼을 클릭하면 검색 결과를 다운로드할 수 있습니다. txt파일 형식으로 다운로드할 수 있으며, 다운로드한 파일은 각 브라우저의 기본 저장 경로에 저장됩니다. 파일 이름은 영문자와 숫자를 포함하여 최대 64자까지 입력할 수 있습니다.
