입출력 박스

외부 입출력 장치를 카메라에 연결하여 사용할 수 있습니다.

[입출력 박스] 페이지에서는 카메라에 연결할 입출력 박스의 정보를 입력할 수 있으며, 입출력 박스와 카메라의 연결 상태를 확인할 수 있습니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

입출력 박스

카메라에 연결할 외부 입출력 박스의 IP 타입, 프로토콜 타입, IP 주소, 포트 번호 등을 설정할 수 있습니다.

외부 입출력 박스를 카메라에 연결하여 사용하려면 [입출력 박스 사용]을 선택합니다.

IP 타입

IPv4와 IPv6 중 입출력 박스와 카메라가 통신할 때 사용할 IP 주소 타입을 설정합니다.

HTTP/HTTPS

HTTP와 HTTPS 중 입출력 박스와 카메라가 통신할 때 사용할 통신 방식 선택합니다.

IP 주소

입출력 박스의 IP 주소를 입력합니다.

포트

입출력 박스의 포트 번호를 입력합니다.

RTSP

입출력 박스의 RTSP 포트 번호를 입력합니다.

ID

입출력 장치에 접속하기 위한 ID를 입력합니다.

비밀번호

입출력 장치에 접속하기 위한 비밀번호를 입력합니다.

상태

입출력 장치와 카메라의 연결 상태를 표시합니다. 카메라와 입출력 장치가 정상적으로 연결돼 있으면 [연결됨]이 표시되며, 연결 중인 상태이면 [연결 중]이 나타납니다.
