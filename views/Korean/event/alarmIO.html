알람 입출력

카메라의 입출력 포트를 사용자가 원하는대로 입력 포트나 출력 포트로 설정할 수 있습니다.

입력 포트로 사용할지 출력 포트로 사용할지 선택한 후 각 알람 입출력에 대한 세부 항목을 설정할 수 있습니다.

설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

알아두기

  • 모든 알람 포트를 출력으로 사용하는 경우, 주야간 모드의 [외부 입력] 모드는 비활성화되며, [외부 입력] 모드로 이미 설정돼 있는 경우에는 [자동] 모드로 변경됩니다. 주야간 모드의 외부 입력 모드는 [비디오 및 오디오]>[카메라 설정]>[주야간 모드]에서 확인할 수 있습니다.

알람 입출력

포트 번호를 선택한 후 해당 포트를 입력 포트로 사용할지 출력 포트로 사용할지 선택합니다. 선택을 완료한 후 [적용] 버튼을 클릭하십시오.

[입력]을 선택한 경우, [입력 장치 설정], [이벤트 동작], [이벤트 스케줄]을 설정할 수 있으며, [출력]을 선택한 경우, [알람 출력]을 설정할 수 있습니다.

카메라마다 지원되는 알람 입력/출력 수가 다를 수 있습니다..

입력 장치 설정

알람 입력 장치를 사용할지 여부와 알람 입력시  알람 이벤트를 발생시키기 위한 방식 등을 설정합니다.

입력 장치 설정

알람 입력 장치를 사용하려면 [사용]을 선택합니다.

방식

알람 입력 방식을 선택합니다.

  • N.O. (Normal Open): 알람 입력 센서가 기본적으로 열린 상태이며, 닫힘 상태가 되면 알람으로 인식하고 알람 입력 이벤트를 발생시킵니다.
  • N.C. (Normal Close): 알람 입력 센서가 기본적으로 닫힘 상태이며, 열림 상태가 되면 알람으로 인식하고 알람 입력 이벤트를 발생시킵니다.

이벤트 동작

알람 입력 이벤트 발생시 카메라가 어떤 동작을 할지 설정합니다.

FTP

알람 입력 이벤트가 발생했을 때 영상 이미지를 FTP 서버에 저장하려면 [사용]을 선택합니다. FTP에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

이메일

알람 입력 이벤트가 발생했을 때 영상 이미지를 이메일로 전송하려면 [사용]을 선택합니다. 이메일에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

저장

알람 입력 이벤트가 발생했을 때 (micro)SD 카드 또는 NAS에 영상을 저장하려면 [사용]을 선택합니다. 저장에 관한 상세 설정은 [이벤트]>[저장]에서 할 수 있습니다.

알람 입력 이벤트가 발생했을 때 (micro)SD 카드에 영상을 저장하려면 [사용]을 선택합니다. 저장에 관한 상세 설정은 [이벤트]>[저장]에서 할 수 있습니다.

알람 출력

알람 입력 이벤트가 발생했을 때 특정 알람 번호로 알람이 출력되도록 설정할 수 있습니다. 알람 출력 번호를 선택한 후, 알람을 출력하지 않으려면 [사용 안 함]을 선택하고 알람을 출력하려면 출력 시간을 선택합니다. [항상 동작]을 선택하여 사용자가 알람을 멈추기 전까지 계속 출력되도록 하거나 5, 10, 15초까지 출력하도록 설정할 수 있습니다.

카메라가 지원하는 출력 알람 개수에 따라 알람 출력 옵션이 다르게 표시됩니다. 예를 들어 출력되는 알람이 2개라면 ‘알람 출력 1’, ‘알람 출력 2’와 같이 옵션이 표시됩니다.

프리셋 이동

알람 입력 이벤트가 발생했을 때 카메라 뷰 위치를 미리 설정해 둔 프리셋 번호로 이동하도록 설정할 수 있습니다. 카메라 프리셋은 [PTZ]>[디지털 PTZ]나 [외부 PTZ]의 [프리셋]에서 설정할 수 있습니다.

[PTZ]>[외부 PTZ]에서 [외부 PTZ 사용] 옵션을 선택한 경우, [프리셋 이동] 옵션이 보이지 않습니다.

프리셋 이동

알람 입력 이벤트가 발생했을 때 카메라 뷰 위치를 미리 설정해 둔 위치로 이동하도록 설정할 수 있습니다. 카메라 위치 프리셋은 [PTZ]>[프리셋 설정]의 [프리셋 설정] 영역에서 설정할 수 있습니다.

프리셋 이동

알람 입력 이벤트가 발생했을 때 카메라 위치를 미리 설정해 둔 위치로 이동하도록 설정할 수 있습니다. 카메라 위치 프리셋은 [PTZ]>[PTZ 설정]의 [프리셋] 탭에서 설정할 수 있습니다.

프리셋 이동

알람 입력 이벤트가 발생했을 때 카메라 뷰 위치를 미리 설정해 둔 위치로 이동하도록 설정할 수 있습니다. 카메라 위치 프리셋은 [PTZ]>[PT 설정]의 [프리셋]에서 설정할 수 있습니다.

오디오 클립

알람 입력 이벤트가 발생했을 때 특정 오디오 클립이 재생되도록 설정할 수 있습니다. 오디오 클립 목록에서 원하는 파일을 선택하면 이벤트가 발생할 때 해당 클립이 재생됩니다. 이벤트가 발생해도 오디오 클립이 재생되지 않도록 하려면 [사용 안 함]을 선택합니다. 재생 버튼()을 클릭하면 해당 오디오 파일을 재생하여 미리듣기할 수 있습니다.

오디오 클립은 [비디오 및 오디오]>[오디오 설정]>[오디오 클립]에서 등록할 수 있습니다.

이벤트 동작 스케줄

알람 입력 이벤트가 발생했을 때 [이벤트 동작]에서 설정한 이벤트 동작이 항상 동작하도록 하거나 설정한 시간에만 동작하도록 설정합니다.

항상 동작

알람 입력 이벤트가 발생했을 때 설정한 이벤트 동작을 항상 실행합니다.

설정한 시간에만 동작

설정한 시간에 알람 입력 이벤트가 발생했을 때 설정한 이벤트 동작을 실행합니다. [설정한 시간에만 동작]을 선택하여 나타나는 시간표에는 현재 카메라 시간을 기준으로 한 일요일부터 토요일까지의 날짜가 표시됩니다. 시간표에서 마우스를 클릭하거나 드래그하여 이벤트 동작 시간을 설정할 수 있습니다. 설정된 이벤트 동작 스케줄은 해당 요일과 시간에 반복적으로 실행됩니다.

[1분], [30분], [1시] 버튼을 클릭하여 시간 보기 단위를 변경할 수 있습니다. [초기화] 버튼을 클릭하면 설정된 이벤트 동작 스케줄이 모두 삭제됩니다. 카메라 시간을 확인하거나 변경하려면 [Basic]>[날짜 및 시간]을 참고하십시오.

알람 출력

사용자가 알람을 출력하거나 이벤트가 발생하여 알람으로 출력될 때 알람을 어떻게 제어할 지 설정합니다.

알람 출력 이벤트에 대한 설정은 [이벤트]>[이벤트 설정] 페이지의 [알람 출력]에 동일하게 적용됩니다.

알람 출력 타입의 설정 변경 시 라이브 페이지의 알람 출력 버튼과 이벤트 설정의 알람 출력의 타입이 변경됩니다.

방식

알람 출력 방식을 선택합니다.

  • N.O. (Normal Open): 알람 출력 센서가 기본적으로 열린 상태이며, 닫힘 상태가 되면 알람을 출력합니다.
  • N.C. (Normal Close): 알람 출력 센서가 기본적으로 닫힘 상태이며, 열림 상태가 되면 알람을 출력합니다.

모드

알람을 출력할 때 제어 방식을 설정합니다.

  • 펄스: [실행 시간]에서 설정한 시간 동안 알람이 출력되고 자동으로 알람이 꺼집니다.
  • 활성화/비활성화: 라이브 화면에서 알람 출력 버튼을 누르면 알람이 출력되고 다시 누르면 알람이 꺼집니다.

실행 시간

[모드]에서 [펄스]를 선택했을 때 알람 출력 시간을 설정합니다.
