라이브 화면

카메라가 촬영 중인 라이브 영상을 확인하고, 화면 캡처 등의 여러 가지 기능을 제어할 수 있습니다. 화면 상단의 버튼을 클릭하면 라이브 화면이 나타납니다.

상단의 버튼을 클릭하여 라이브 화면과 재생 화면, 설정 화면으로 이동할 수 있습니다.

  • (라이브): 카메라가 촬영 중인 라이브 화면을 확인하고 다양한 카메라 기능을 제어할 수 있습니다.
  • (재생): (micro)SD 카드 또는 NAS에 녹화해 둔 영상을 검색하고 재생하여 볼 수 있습니다.
  • (재생): (micro)SD 카드에 녹화해 둔 영상을 검색하고 재생하여 볼 수 있습니다.
  • (설정): 카메라의 상세 설정을 변경할 수 있습니다.

알아두기

  • 고해상도의 영상을 라이브 화면에서 재생하는 경우, 고해상도 영상을 최적화하여 보여줄 수 있는 PLUGINFREE 프로파일이 자동으로 생성됩니다. 생성된 PLUGINFREE 프로파일은 [설정]>[Basic]>[비디오 프로파일]에서 확인할 수 있습니다.
  • 라이브 페이지 오른쪽 상단에서 아이콘을 클릭하면, 접속한 IP 주소와 인증 상태를 확인할 수 있습니다. 인증에 성공한 경우 초록색 아이콘으로, 실패한 경우 빨간색 아이콘으로 표시됩니다. 또한 HTTP로 접속하여 해당 사항이 없는 경우는 대시(-)로 표시됩니다.
  • Chrome으로 웹뷰어에 접속하면 재생 화면의 기능을 가장 안정적으로 사용할 수 있습니다.
  • 라이브 페이지에서 영상을 재생할 때 아래와 같은 조건에서 화면에 잔상이 표시될 수 있습니다.
    • 프로파일 변경 시 해상도가 달라지는 경우
    • 프로파일 변경 시 네트워크 지연 발생으로 데이터 수신이 늦어지는 경우
    • 웹 브라우저 창의 크기 변경 및 위치 이동 시
  • 발열 감지 카메라에서 채널 1은 실화상 영상이며, 채널 2는 열화상 영상입니다.

아이콘

라이브 화면 하단의 아이콘은 다음과 같은 기능을 합니다.

특정 브라우저 또는 코덱에서는 일부 기능이 동작하지 않을 수도 있습니다.

아이콘 기능 설명
BestShot

BestShot은 영상에서 감지된 객체(사람, 얼굴, 차량, 번호판)에서 신뢰도가 가장 높은 객체를 이미지로 추출한 것입니다. BestShot의 이미지 형식은 JPEG이며, 객체를 포함하는 일부 영역만 추출하여 이미지를 생성합니다.

라이브 페이지의 BestShot 패널에서는 객체별 BestShot을 확인할 수 있습니다. 또한 각 객체의 세부 속성이 아이콘으로 표시됩니다. 예를 들어 사람의 경우 성별, 상/하의 옷 색상과 길이, 모자나 가방 착용 여부 등 세부 정보를 보여줍니다.

BestShot 패널에 표시될 객체의 종류는 [설정]>[분석]>[객체 감지]>[BestShot]에서 선택할 수 있습니다.

 

비디오 설정

현재 라이브 화면에 적용된 프로파일을 확인하거나 프로파일을 변경할 수 있습니다. 또한 라이브 화면의 디스플레이 설정을 변경할 수 있습니다.

온도

온도 분석 관심 영역별로(A/B/C) 각 영역의 평균(Average), 최소(Maximum), 최대(Maximum) 온도를 보여줍니다.

온도 분석 관심 영역은 [설정]>[분석]>[온도 감지] 메뉴에서 설정할 수 있습니다.

[온도 단위]에서 화씨와 섭씨 중 온도 표시 단위를 선택하면 선택한 단위에 맞게 온도가 표시됩니다.

[스폿 방사율]에서 방사율 정보를 입력할 수 있습니다. [설정]>[비디오 및 오디오]>[온도 설정]>[스폿 방사율]에서 입력한 방사율이 [스폿 방사율]에 기본값으로 표시됩니다.

 

PTZ

 카메라의 줌이나 초점을 조절하고 원하는 프리셋으로 이동할 수 있습니다. [설정]>[PTZ]>[프리셋 설정] 페이지에서 설정한 프리셋 목록이 표시됩니다. [설정]버튼을 클릭한 후 새로운 프리셋 위치를 추가할 수도 있습니다.

PT

 카메라의 팬/틸트 동작을 디지털로 제어할 수있습니다.

PTZ

카메라의 팬/틸트/줌 동작을 디지털로 제어할 수 있습니다.

현재 라이브 화면에 적용된 비디오 프로파일이 디지털 PTZ를 지원하는 경우에만 작동합니다. 디지털 PTZ 제어 프로파일로 변경하려면 [설정]>[Basic]>[비디오 프로파일]>[비디오 프로파일]>[프로파일 타입]에서 [디지털 PTZ 프로파일]을 선택합니다.

[채널 선택]에서 채널 2를 선택한 경우에만 디지털 PTZ 기능이 활성화됩니다.

PTZ 아이콘이 보이지 않으면 [설정]>[PTZ]>[외부 PTZ]의 [외부 PTZ 사용] 옵션이 선택 해제되어 있는지 확인하십시오.

 

PTZ

카메라 RS-485나 RS-422 단자에 연결된 팬/틸트 리시버를 통해 카메라의 팬/틸트/줌 동작을 제어할 수 있습니다.

[채널 선택]에서 채널 1을 선택한 경우에만 외부 PTZ 제어 기능이 동작합니다.

PTZ 아이콘이 보이지 않으면 [설정]>[PTZ]>[외부 PTZ]의 [외부 PTZ 사용] 옵션이 선택되어 있는지 확인하십시오.

 

PTZ

카메라의 팬/틸트/줌 동작을 제어할 수 있습니다. [설정]>[PTZ]>[PTZ 설정] 페이지에서 설정한 프리셋 위치로 이동하거나, 스윙, 그룹, 투어 기능을 시작하고 중지할 수 있습니다.

[포커스 저장]에서 포커스 목록을 확인할 수 있습니다.

채널 선택에서 [PTZ] 채널을 선택한 경우에만 'PTZ' 아이콘이 보입니다.

상태

프로파일 별로 접속 정보와 현재 카메라에 접속한 사용자 접속 정보를 확인할 수 있습니다.

오버레이

카메라가 객체를 감지할 때 영상 화면에 표시할 객체 타입을 선택할 수 있습니다.

AI 카메라의 객체 분석 알고리즘이 작동하는 경우에만 감지한 객체가 영상 화면에 표시되며, AI 알고리즘이 동작하려면 객체 감지, IVA, 피플 카운팅, 대기열 관리, 히트맵 중 하나의 기능이 활성화돼 있어야 합니다.

[채널 선택]에서 채널 1을 선택한 경우에만 오버레이 아이콘이 활성화됩니다.

 
렌즈 모델명

렌즈 종류가 표시됩니다.

채널 선택에서 채널을 선택하면 채널별로 장착된 렌즈 이름을 확인할 수 있습니다.

채널 선택

카메라가 지원하는 채널이 표시됩니다. 채널을 선택하면 각 채널의 라이브 영상을 확인할 수 있습니다.

채널 번호를 선택하면 해당 채널의 영상이 뷰어에 나타납니다. 멀티 뷰 아이콘을 선택하면 카메라가 지원하는 모든 채널의 영상이 한 번에 보여집니다. 단일 화면으로 돌아오려면 채널 번호를 클릭합니다.

고정형 카메라의 채널 번호를 선택하면 해당 영상이 뷰어에 나타납니다. [PTZ]를 선택하면 PTZ 단일 채널 영상이 뷰어에 나타납니다. 멀티 채널 아이콘을 선택하면 고정형 카메라와 PTZ 카메라이 영상이 한 번에 나타납니다. 단일 화면으로 돌아오려면 채널 번호나 [PTZ]를 클릭합니다.

카메라 채널 2~5번을 선택하면 각 채널별 카메라 영상을 보여줍니다. 채널 1을 선택하면 2~5번 채널의 카메라 영상을 하나로 연결하여 파노라마 형식으로 보여줍니다. 채널 6번과 7번을 선택하면 1번 채널 영상을 카메라 PTZ를 조정하면서 확인할 수 있습니다.

  • 채널 1: 라이브 영상을 보여 주는 채널입니다.
  • 채널 2: 디지털 PTZ 영상을 보여주는 채널입니다.

채널 1은 실화상 영상이며, 채널 2는 열화상 영상입니다. 멀티뷰 아이콘을 선택하면 실화상 영상과 열화상 영상을 동시에 확인할 수 있습니다.

 

채널 선택

카메라가 지원하는 채널이 표시됩니다. 채널을 선택하면 각 채널의 라이브 영상을 확인할 수 있습니다.

[Fisheye View] 채널은 피쉬아이 카메라의 원본 영상이며, 이 외 채널은 사용자가 원하는 뷰 모드로 설정하고 추가한 것입니다. 카메라 채널은 [설정]>[Basic] >[비디오 프로파일]에서 추가할 수 있습니다.

전체 화면

라이브 화면을 전체 화면으로 볼 수 있습니다. 원래 웹 브라우저의 크기로 돌아가려면 전체 화면 모드에서 아이콘을 다시 한번 클릭하거나 키보드의 [ESC] 키를 누릅니다.
크기 옵션

한 번씩 클릭할 때마다 다음 모드로 변경됩니다.

  • (화면 맞춤): 카메라 영상을 웹 브라우저의 크기에 맞추어 봅니다.
  • (원본 크기): 카메라 영상의 실제 해상도 크기로 봅니다.
  • (영상 비율 고정): 카메라 영상 해상도의 종횡비를 유지한 채 라이브 화면을 확대 또는 축소하여 웹 브라우저의 크기만큼 봅니다.

캡처

라이브 화면을 캡처하여 PNG 이미지 파일로 저장할 수 있습니다. 캡처한 이미지 파일은 각 브라우저의 기본 저장 경로에 저장됩니다.

와이퍼

 버튼을 클릭하면 와이퍼가 왕복 동작합니다.

워셔

 버튼을 클릭하면 물이 3초 동안 나온 후 자동으로 중지합니다. 이 후 와이퍼가 10초 정도 동작하며 물을 닦아냅니다.

히터

 버튼을 클릭하면 히터가 30초 동안 동작합니다.

저장

라이브 화면을 수동으로 녹화하여 PC에 저장할 수 있습니다. 저장 아이콘을 클릭하여 녹화를 시작하고, 다시 한번 더 클릭하면 녹화가 멈춥니다. 녹화 파일은 .avi의 형식으로 저장할 수 있고, 브라우저에 따라 브라우저 기본 저장 경로에 저장되거나 다른 이름 저장 창에서 파일 경로를 설정할 수 있습니다.

녹화 영상에 비밀번호를 설정하려면 녹화 영상 파일 형식 목록에서 ZIP을 선택한 후 비밀번호를 입력합니다. 다운로드한 영상을 재생할 때 설정한 비밀번호를 입력해야 영상을 재생할 수 있습니다.

녹화 영상 비밀번호 설정 기능은 일부 카메라 모델에서는 지원하지 않습니다.

픽셀 카운트

라이브 화면에서 마우스로 선택한 영역의 영상 픽셀 수를 확인할 수 있습니다. 픽셀 카운트 아이콘을 클릭한 후 원하는 영역을 마우스로 클릭하고 드래그합니다. 선택한 영역이 표시되고 영상의 픽셀 수가 표시됩니다. 픽셀 카운트 아이콘을 한 번 더 클릭하면 픽셀 카운트 기능이 종료됩니다.

마이크

마이크 기능을 사용할 수 있습니다. 마이크 기능을 사용하려면 해당 프로파일에 [오디오 입력] 기능이 활성화되어 있어야 합니다. 오디오 입력 기능을 활성화하려면 [설정]>[Basic]>[비디오 프로파일]>[오디오 입력]에서 [사용]을 선택합니다.

채널 1을 선택한 경우에만 마이크 기능을 사용할 수 있습니다.

알람 출력

원하는 알람 출력 번호를 클릭하면 이미 설정한 대로 알람을 출력합니다. 알람은 [설정]>[이벤트]>[알람 입출력] 페이지에서 설정할 수 있습니다. 알람 출력 개수는 카메라마다 다릅니다.

스피커

라이브 화면의 오디오 음량을 조절할 수 있습니다. 아이콘을 클릭하여 오디오를 활성화한 후 음량 크기를 조절할 수 있습니다.

채널 1을 선택한 경우에만 스피커 기능을 사용할 수 있습니다.

재생/정지

오디오 클립 목록에서 원하는 오디오 클립을 선택한 후, 오디오를 재생하거나 정지할 수 있습니다. 라이브 영상을 모니터링하다가 사용자가 원하는 오디오를 출력할 수 있습니다.

오디오 클립은 [설정]>[비디오 및 오디오]>[오디오 설정]>[오디오 클립]에서 등록할 수 있습니다.

채널을 변경하려면

  • 원하는 채널 번호를 선택합니다. 해당 채널의 영상이 뷰어에 나타납니다.
  • 여러 화면을 한 번에 보려면 멀티 뷰 아이콘을 클릭합니다.
  • 단일 화면으로 돌아오려면 채널 번호를 클릭합니다.

영상을 캡처하려면

  • 캡처할 장면에서 캡처 아이콘()을 클릭합니다.
  • 캡처가 저장되면 알림 메시지가 나타납니다. 각 브라우저의 저장 경로에 캡처 이미지가 저장됩니다.

Windows 7 이상에서 Internet Explorer 브라우저로 캡쳐가 안 되는 경우, Internet Explorer 브라우저를 관리자 권한으로 실행하십시오.

영상을 녹화하려면

  • 저장 아이콘()을 클릭합니다.
  • 수동 녹화를 종료하려면 저장 아이콘()을 한 번 더 클릭합니다.

프리셋을 추가하려면 [설정] 버튼을 클릭한 후 프리셋 이름을 입력하고 번호를 설정한 뒤 [적용] 버튼을 클릭합니다. 또한 [설정]>[PTZ]> [디지털 PTZ]>[프리셋] 탭에서 프리셋을 추가하거나 변경할 수 있습니다.

PTZ

카메라의 팬/틸트/줌 동작을 제어하거나 미리 설정해 둔 PTZ 프리셋 위치로 이동이 가능하며 그룹 동작을 제어할 수 있습니다.

분할 화면 선택

4분할 뷰에서 개별 영상의 번호를 선택합니다. 1은 4분할 뷰의 왼쪽 상단, 2는 오른쪽 상단, 3은 왼쪽 하단, 4는 오른쪽 하단의 영상 뷰입니다.

버튼을 클릭하고 드래그하여 카메라 영상 뷰를 조정합니다. 줌 막대에서 버튼을 클릭하면 화면이 확대되고 버튼을 클릭하면 화면이 축소됩니다.

[설정]>[Basic]>[비디오 프로파일]>[비디오 프로파일]>[피쉬아이 뷰 타입]에서 [왜곡 보정 뷰]를 선택한 후 [뷰]에서 [4분할 뷰]를 선택한 경우에만 [분할 화면 선택] 옵션이 나타납니다.

프리셋

프리셋 목록에서 원하는 프리셋 번호를 선택한 후 [이동] 버튼을 클릭하면 카메라 뷰가 프리셋 위치로 이동합니다. 프리셋 위치를 설정하려면 PTZ 제어 화면에서 아이콘을 클릭하여 카메라 뷰 위치를 조정한 후 [설정] 버튼을 클릭합니다. [번호]와 [이름] 필드에서 프리셋 번호와 이름을 각각 설정한 뒤 [적용] 버튼을 클릭합니다.

[홈] 버튼을 클릭하면 카메라 뷰가 홈 위치로 이동합니다. 홈 위치는 [설정]>[PTZ]>[디지털 PTZ]>[프리셋] 탭에 설정할 수 있습니다.

채널 선택

채널 선택

채널 번호 버튼을 선택하면 고정 카메라 영상을 확인할 수 있습니다. [PTZ] 버튼을 클릭하면 PTZ 카메라 영상을 단일 화면에서 확인할 수 있습니다. 분할 화면 버튼을 클릭하면, 왼쪽 화면에는 고정 카메라 영상이, 오른쪽 화면에서는 PTZ 영상이 동시에 나타납니다. 단일 화면으로 이동하려면 채널 번호 버튼이나 PTZ 버튼을 클릭합니다.

고정 카메라와 PTZ 카메라 연동

분할 화면 상태에서 고정 카메라 영상 화면에 마우스를 올려 놓으면 마우스 커서가 돋보기 모양으로 나타납니다. 고정 카메라 영상 화면에서 특정 지점을 클릭하면, 클릭한 지점이 오른쪽 PTZ 화면 중앙 + 표시에 나타납니다.

고정 카메라 영상 화면에서 마우스를 드래그하여 영역을 설정하면, PTZ 영상 화면에 해당 영역이 확대되어 나타납니다.
