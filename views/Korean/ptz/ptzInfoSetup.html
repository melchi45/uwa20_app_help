PTZ 설정

카메라 PTZ 프리셋 위치를 설정하고, 스윙, 그룹, 투어, 자동 실행 등 세부 기능을 설정할 수 있습니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

프리셋

PTZ 카메라의 프리셋 위치를 설정할 수 있습니다.

프리셋 설정하기

  1. 아래와 같이 카메라의 위치를 조정합니다.
    • 아이콘을 마우스로 드래그하면 카메라가 팬과 틸트로 이동합니다.
    • 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 아이콘을 클릭하여 화면을 확대하거나 아이콘을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
    • 아이콘을 클릭하여 포커스를 가까이(near)이동하거나 버튼을 클릭하여 포커스를 멀리(far)이동합니다.
    • 카메라 포커스를 자동으로 한 번 맞추려면 아이콘을 클릭합니다.
  2. [프리셋 설정] 버튼을 클릭합니다.
  3. [프리셋 설정] 대화 상자에서 [번호]에서 원하는 프리셋 번호를 선택합니다.
  4. [이름]에 해당 프리셋의 이름을 입력합니다. 프리셋 이름은 영어 대소문자와 숫자만 입력할 수 있습니다.
  5. [확인] 버튼을 클릭합니다.

추가한 프리셋이 [프리셋 설정]목록에 나타납니다. 프리셋 목록에서 해당 프리셋 세부 내용을 설정할 수 있습니다.

프리셋 삭제하기

  • [프리셋 설정]의 프리셋 목록에서 삭제할 프리셋의 체크박스를 선택한 후 [삭제] 버튼을 클릭합니다.

홈으로 이동하기

  • [홈 이동] 버튼을 클릭합니다.

홈 설정하기

  1. 아래와 같이 홈으로 설정할 카메라 위치를 조정합니다.
    • 아이콘을 마우스로 드래그하면 카메라가 팬과 틸트로 이동합니다.
    • 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 아이콘을 클릭하여 화면을 확대하거나 아이콘을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
    • 버튼을 클릭하여 포커스를 가까이(near)이동하거나 버튼을 클릭하여 포커스를 멀리(far)이동합니다.
    • 카메라 포커스를 자동으로 한 번 맞추려면 버튼을 클릭합니다.
  2. [홈 설정] 버튼을 클릭합니다.

알아두기

  • 홈 위치 설정 후 카메라가 재부팅되면 화면이 홈 포지션에 위치합니다. 만약 다른 시퀀스(프리셋, 스윙, 그룹, 투어, 추적)가 동작 중이라면 카메라가 재부팅되더라도 홈 위치가 아닌 동작 중이던 시퀀스 기능이 실행됩니다.

워셔 실행하기

  • [워셔 실행] 버튼을 클릭합니다.

외부에 부착된 워셔가 동작하며 카메라 렌즈를 세척합니다. 워셔를 실행하려면 워셔 위치가 먼저 설정돼 있어야 합니다. 워셔 위치를 설정하려면 [워셔 위치 설정]을 클릭하십시오.

워셔 위치 설정하기

  • [워셔 위치 설정] 버튼을 클릭합니다.

워셔 위치가 프리셋 88번으로 설정됩니다. [프리셋 설정] 목록에 워셔 위치가 추가된 것을 확인할 수 있습니다.

프리셋 설정

프리셋 목록을 확인하고 각 프리셋 별로 상세 설정을 수정하거나 삭제할 수 있습니다.

번호

[프리셋 설정] 대화 상자에서 입력한 프리셋 번호입니다.

이름

[프리셋 설정] 대화 상자에서 입력한 프리셋 이름입니다.

연결 동작

카메라가 해당 프리셋 위치로 이동한 후 실행할 카메라의 동작을 설정합니다. 스윙 기능이 실행 중인 경우, 연결 동작 기능이 작동하지 않습니다.

  • 사용 안 함: 카메라가 해당 프리셋 위치로 이동한 후 아무런 동작을 하지 않습니다.
  • 자동 실행: [자동 실행] 탭에서 설정한 동작이 자동으로 실행됩니다. 자동 실행 [모드]에서 [사용 안 함]을 선택하거나 자동 실행 동작이 없는 경우에는 프리셋 실행 상태가 유지됩니다. 그룹이나 투어 기능으로 프리셋이 실행되면 '자동 실행' 기능이 작동하지 않습니다.
  • 자동 추적: [추적 시간] 컬럼에서 설정한 시간만큼 자동 추적을 실행합니다.
  • 자동 추적 > 홈 이동: 자동 추적 동작 실행 후 PTZ 홈 위치로 이동합니다. 홈 위치를 설정하는 방법은 홈 설정하기를 참고하십시오.
  • 자동 추적 > 자동 실행: 자동 추적 동작 실행 후 사전에 지정한 동작을 자동 실행합니다. 자동 실행할 동작을 설정하는 방법은 자동 실행을 참고하십시오.
  • 영상 분석: 영상 분석 동작이 실행됩니다.
추적 시간

[연결 동작]에서 [자동 추적]을 선택한 경우, 자동 추적을 실행할 추적 시간을 설정합니다.

이동

설정 페이지로 이동하여 세부 옵션을 확인하거나 설정할 수 있습니다.

  • 카메라 설정: [비디오 및 오디오] > [카메라 설정] 페이지로 이동합니다. 선택한 프리셋 주변 환경에 적합하도록 카메라 설정을 변경할 수 있습니다.
  • 움직임 감지: [분석] > [움직임 감지] 페이지로 이동합니다. [움직임 감지] 페이지에서 [타입] 옵션 중 [프리셋]을 선택하면 프리셋별로 움직임 감지 이벤트를 각각 다르게 설정할 수 있습니다.
  • IVA: [분석] > [IVA] 페이지로 이동합니다. [IVA] 페이지에서 [타입] 옵션 중 [프리셋]을 선택하면 프리셋별로 IVA(지능형 영상 분석) 이벤트를 각각 다르게 설정할 수 있습니다.

마지막 위치

카메라 전원을 껐다 켜면 저장된 마지막 위치로 이동하도록 설정할 수 있습니다. 카메라 마지막 위치를 저장할지 여부를 선택하고 저장 간격을 선택합니다.

마지막 위치 저장

마지막 위치를 저장하려면 [사용]을 선택합니다.

시간

현재 위치를 마지막 위치로 저장할 저장 간격을 설정합니다. 단위는 '초'입니다. 저장 간격을 필드에 입력한 후 [적용]을 클릭하십시오.

스윙

스윙은 2개의 프리셋 구간을 카메라가 왕복으로 기능입니다.

알아두기

  • 스윙 기능을 사용하려면 최소 2개 이상의 프리셋이 추가되어 있어야 합니다. [프리셋] 탭에서 프리셋을 추가하거나 수정할 수 있습니다.

스윙 설정

시작 프리셋 번호

스윙 동작을 시작하는 시작 프리셋 번호를 선택합니다.

종료 프리셋 번호

스윙 동작을 종료하는 종료 프리셋 번호를 선택합니다.

속도

시작 프리셋에서 종료 프리셋으로 이동할 때 이동 속도입니다.

큰 숫자로 설정할 수록 시작 프리셋 위치에서 종료 프리셋 위치로 빠르게 이동합니다.

전환 간격

시작 프리셋과 종료 프리셋 위치에서 카메라가 각각 멈춰서 대기하는 시간입니다.

예를 들어 '1'을 입력한 경우, 시작 프리셋 위치에서 '1초'를 체류한 뒤 종료 프리셋 위치로 카메라가 이동합니다.

스윙 모드

시작 프리셋에서 종료 프리셋으로 왕복하며 이동할 때 카메라의 팬이나 틸트 동작만으로 이동할지 팬과 틸트 모든 동작으로 이동할지 선택합니다.

스윙 설정하기

  1. [시작 시간 프리셋 번호] 컬럼에서 스윙을 시작할 프리셋 번호를 선택합니다.
  2. [종료 시간 프리셋 번호] 컬럼에서 스윙을 종료할 프리셋 번호를 선택합니다.
  3. [속도] 컬럼에서 시작 프리셋에서 종료 프리셋으로 이동할 이동 속도를 입력합니다.
  4. [전환 간격] 컬럼에서 시작 프리셋에서 얼마나 오랫동안 머물다가 종료 프리셋으로 이동할지 프리셋 유지 시간을 입력합니다.
  5. [스윙 모드] 컬럼에서 스윙 동작 모드를 설정합니다. 예를 들어 팬 방향으로만 프리셋으로 이동하려면 [팬]을 선택합니다.

스윙 시작하기

  • 스윙 동작을 시작하려면 [시작] 버튼을 클릭합니다.

스윙 중지하기

  • 스윙 동작을 중지하려면 [중지] 버튼을 클릭합니다.

그룹

그룹 기능은 이미 지정한 여러 개의 프리셋을 그룹화하여 연속적으로 호출하는 기능입니다.

6개의 그룹까지 설정할 수 있으며 그룹마다 128개 프리셋까지 저장할 수 있습니다.

알아두기

  • 그룹 기능을 사용하려면 프리셋이 추가되어 있어야 합니다. [프리셋] 탭에서 프리셋을 추가할 수 있습니다.

그룹 설정

프리셋 번호

그룹에 저장할 프리셋 번호를 선택합니다.

속도

프리셋간 이동 속도, 즉 다음 프리셋 위치로 이동할 때 카메라의 움직임 속도를 입력합니다.

큰 숫자로 설정할 수록 현재 프리셋 위치에서 다음 프리셋 위치로 빠르게 이동합니다.

전환 간격

각 프리셋 위치에서의 체류 시간을 입력합니다.

예를 들어 1번 프리셋에서 전환 간격을 '1'로 선택하면 1번 프리셋 위치에서 1초 동안 체류한 후 다음 프리셋 위치로 이동합니다.

그룹 1 설정하기

  1. 그룹 1을 설정하려면 그룹 드롭다운 목록에서 '1'을 선택합니다.
  2. [프리셋 번호] 컬럼에서 '그룹 1'에 추가할 프리셋 번호를 선택합니다.
  3. [속도] 컬럼에서 설정 중인 프리셋 위치에서 다음 프리셋 위치로 카메라가 이동할 속도를 입력합니다.
  4. [전환 간격] 컬럼에서 설정 중인 프리셋 위치에 카메라가 얼마나 체류한 후 다음 프리셋 위치로 이동할지 체류 시간을 설정합니다.
  5. [적용] 버튼을 클릭합니다.

그룹 설정 값 삭제하기

  1. 그룹 드롭다운 목록에서 기존에 설정한 값을 초기화할 그룹 번호를 선택합니다.
  2. [삭제] 버튼을 클릭합니다.

그룹 동작 시작하기

  1. 그룹 드롭다운 목록에서 동작할 그룹 번호를 선택합니다.
  2. [시작] 버튼을 클릭합니다.

그룹 동작 중지하기

  1. 현재 작동 중인 그룹을 중지하려면 [중지] 버튼을 클릭합니다.

투어

투어 기능은 이미 지정한 프리셋의 집합인 그룹을 순차적으로 연속 호출하는 기능입니다.

알아두기

  • 투어 기능을 사용하려면 그룹이 설정돼 있어야 합니다. [그룹] 탭에서 그룹을 설정할 수 있습니다.

투어 설정

그룹 번호

연속적으로 호출될 그룹 번호를 순서대로 선택합니다.

전환 간격

각 그룹에서의 체류 시간을 입력합니다.

예를 들어 1번 그룹에서 전환 간격을 '1'로 선택하면 1번 그룹에서 1초 동안 체류한 후 다음 그룹으로 이동합니다.

투어 설정하기

그룹 1, 그룹 2, 그룹 3을 순차적으로 호출하여 카메라를 이동하려면 다음과 같이 설정합니다.

  1. 그룹 번호 드롭다운 목록에서 '1'을 선택합니다.
  2. [전환 간격] 컬럼에서 '그룹 1'에서 카메라가 체류할 시간을 설정합니다.
  3. 그룹 번호 드롭다운 목록에서 '2'를 선택합니다.
  4. [전환 간격] 컬럼에서 '그룹 2'에서 카메라가 체류할 시간을 설정합니다.
  5. 그룹 번호 드롭다운 목록에서 '3'를 선택합니다.
  6. [전환 간격] 컬럼에서 '그룹 3'에서 카메라가 체류할 시간을 설정합니다.
  7. [적용] 버튼을 클릭합니다.

투어 설정값 초기화하기

  • [삭제] 버튼을 클릭하면, 설정한 투어 값이 모두 삭제되고 초기화됩니다.

투어 동작 시작하기

  • [시작] 버튼을 클릭합니다. 설정한 그룹을 순차적으로 호출하며 카메라가 이동합니다.

투어 동작 중지하기

  • [중지] 버튼을 클릭합니다. 투어 동작이 중지합니다.

추적

추적 기능은 카메라의 움직임 경로를 저장한 후 그 경로대로 카메라가 움직이도록 하는 기능입니다.

추적 설정

추적 경로를 설정하거나 추적 기능을 시작하고 종료할 수 있습니다. 추적 경로를 설정할 때 미리 설정해 둔 프리셋 위치를 추적 경로에 포함할 수 있습니다. 프리셋 위치는 [프리셋] 탭에서 설정할 수 있습니다.

추적 경로 설정하기

  1. 설정할 추적 번호를 선택합니다.
  2. [설정] 버튼을 클릭한 후, 다음과 같은 방법으로 이동 경로를 설정합니다.
    • 영상 화면에서 방향 버튼을 이용하여 팬, 틸트, 줌을 조정하여 저장하고자 하는 이동 경로를 설정합니다.
    • 미리 설정해 둔 위치를 이동 경로에 포함하려면 [프리셋 목록]에서 원하는 프리셋 위치 번호를 선택하고 [이동]을 클릭합니다.
  3. [설정] 버튼을 한 번 더 클릭하면, 해당 추적 번호로 추적 경로가 저장됩니다.

추적 동작 시작하기

  1. 동작할 추적 번호를 선택합니다.
  2. [시작] 버튼을 클릭합니다. 해당 추적 번호로 설정해 둔 이동 경로를 따라 카메라가 이동합니다.

추적 동작 중지하기

  1. 현재 작동 중인 추적 동작을 중지하려면[중지] 버튼을 클릭합니다.

자동 실행

자동 실행은 사용자가 일정 시간동안 카메라를 조작하지 않을 경우 사전에 지정한 동작을 자동으로 실행하는 기능입니다.

모드

자동으로 실행할 기능을 선택한 후 세부 설정을 지정할 수 있습니다.

  • 사용 안 함: 자동 실행 기능을 사용하지 않습니다.
  • 홈: 홈 위치로 카메라가 이동합니다. 홈 위치는 [프리셋] 탭에서 설정할 수 있습니다.
  • 프리셋: 해당 프리셋 번호로 이동합니다. 프리셋으로 이동할 시간과 이동할 프리셋 위치 번호를 설정할 수 있습니다.
  • 스윙: 스윙 동작을 자동으로 실행합니다. 스윙 동작이 자동으로 실행되는 시간과 스윙 모드를 설정할 수 있습니다.
  • 그룹: 그룹 동작을 자동으로 실행합니다. 그룹 동작이 자동으로 실행되는 시간과 그룹 번호를 설정할 수 있습니다. 그룹 상세 동작은 [그룹] 탭에서 설정할 수 있습니다.
  • 투어: 투어동작을 자동으로 실행합니다.
  • 추적: 추적 동작을 자동으로 실행합니다. 추적 동작이 자동으로 실행되는 시간과 추적 번호를 설정할 수 있습니다. 추적 상세 기능은 [추적] 탭에서 설정할 수 있습니다.
  • 오토팬: 팬 방향으로 360도 회전하는 동작으로 자동으로 실행합니다. 자동으로 실행되는 시간과 카메라가 팬 방향으로 회전하는 속도와 틸트 고정 각도를 설정할 수 있습니다. 예를 들어 틸트 각도를 '30'으로 설정한 경우 틸트 각도는 30도로 유지됩니다.
  • 스케줄: 설정한 요일과 시간에 설정한 동작이 자동으로 작동하도록 설정할 수 있습니다. 분 단위까지 설정할 수 있으며, 기본 설정은 홈 포지션입니다.

스케줄에 따라 카메라 자동 실행하도록 설정하기

  1. [모드]에서 [스케줄]을 선택합니다.
  2. 카메라가 자동으로 동작할 시간을 클릭합니다.
  3. [스케줄] 대화 상자의 [모드]에서 자동으로 실행될 동작을 선택합니다. 선택한 모드에 따라 세부 옵션이 다릅니다. 각 옵션을 설정합니다.
  4. [확인] 버튼을 클릭합니다.

자동 실행 동작이 설정된 시간이 시간표에 표시됩니다. [초기화]버튼을 클릭하면 설정한 시간이 모두 삭제됩니다.
