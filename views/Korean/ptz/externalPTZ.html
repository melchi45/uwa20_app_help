외부 PTZ

카메라 RS-485나 RS-422 단자에 연결된 팬/틸트 리시버를 통해 PTZ를 제어할 수 있도록 설정할 수 있습니다. 프리셋을 설정하거나, 카메라와 연결된 팬/틸트 리시버와의 통신 모드를 설정할 수 있습니다.

채널 선택

카메라 채널을 선택한 후 채널별로 외부 PTZ 세부 항목을 설정할 수 있습니다.

채널 1은 라이브 영상을 보여 주는 채널이며, 채널 2는 디지털 PTZ 영상을 보여 주는 채널입니다. [외부 PTZ]은 채널 1에서만 설정할 수 있습니다.

외부 PTZ

외부 PTZ 사용

카메라 RS-485나 RS-422 단자에 연결된 팬/틸트 리시버를 통해 PTZ를 제어할 수 있도록 외부 PTZ를 사용하려면 [외부 PTZ 사용]을 선택합니다.

알아두기

  • [외부 PTZ 사용]을 선택하면 외부 PTZ 기능만 사용할 수 있으며, [외부 PTZ 사용]을 선택하지 않으면 디지털 PTZ 기능만 사용할 수 있습니다.
  • [외부 PTZ 사용]을 선택하면 라이브 페이지에서 1번 채널을 선택한 후 PTZ를 제어할 수 있습니다.
  • [외부 PTZ 사용] 옵션을 변경하면 카메라가 재부팅됩니다.

PTZ 조정하기

카메라의 팬, 틸트, 및 줌 동작을 조정할 수 있습니다.

PTZ 조정하기

  • 아이콘을 마우스로 드래그하면 카메라가 팬과 틸트로 이동합니다.
  • 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 아이콘을 클릭하여 화면을 확대하거나 아이콘을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
  • 아이콘을 클릭하여 포커스를 가까이(near)이동하거나 아이콘을 클릭하여 포커스를 멀리(far)이동합니다.
  • 카메라 포커스를 자동으로 한 번 맞추려면 버튼을 클릭합니다.

프리셋

프리셋은 카메라의 뷰를 설정해 두는 것을 의미합니다. 128개의 프리셋을 설정할 수 있습니다.

프리셋 설정하기

  1. PTZ 제어 화면에서 아이콘을 클릭하여 카메라 영상을 확대하거나, 아이콘을 클릭하고 드래그하여 카메라 뷰를 조정합니다.
  2. [번호]에서 원하는 프리셋 번호를 선택합니다.
  3. [이름]에 해당 프리셋의 이름을 입력합니다.
  4. [추가] 버튼을 클릭합니다.

프리셋으로 이동하기

  1. [번호]에서 원하는 프리셋 번호를 선택합니다.
  2. [이동] 버튼을 클릭하면 설정해 둔 위치로 카메라의 뷰가 이동합니다.

프리셋 삭제하기

  1. [번호]에서 삭제할 프리셋 번호를 선택합니다.
  2. [삭제] 버튼을 클릭하면 프리셋이 삭제됩니다.

시리얼 포트 설정

외부 팬/틸트 리시버와의 연결값을 설정합니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

프로토콜

리시버와 카메라가 통신할 프로토콜을 선택합니다.

카메라 ID

카메라 ID가 표시됩니다. 고정 값으로 변경할 수 없습니다.

전송 속도

RS-485 통신 속도를 설정합니다.

데이터 비트

데이터 비트를 설정합니다.

패리티 비트

데이터 전송 처리 에러를 확인하기 위한 패리터 비트를 설정합니다.

정지 비트

한 문자의 종료를 표시할 정지 비트를 설정합니다.

ONVIF 포커스 이동

외부 카메라 포커스를 ONVIF 프로토콜로 조절하도록 설정하려면 [사용함]을 선택합니다.
