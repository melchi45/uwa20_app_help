PT 설정

카메라 팬/틸트 프리셋 위치를 설정하고, 그룹, 자동 실행의 세부 기능을 설정할 수 있습니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

프리셋

프리셋은 카메라의 뷰 위치를 설정해 두는 것을 의미합니다. 카메라 프리셋을 설정한 뒤 해당 뷰로 쉽게 이동할 수 있습니다. 300개의 프리셋을 설정할 수 있습니다.

프리셋 설정하기

  1. PT 제어 화면에서 아이콘을 클릭하고 드래그하여 카메라 뷰를 조정합니다.
  2. [프리셋 설정] 버튼을 클릭합니다.
  3. [프리셋 설정] 대화 상자에서 [번호]에서 원하는 프리셋 번호를 선택하고, [이름]에 해당 프리셋의 이름을 입력합니다.
  4. [확인] 버튼을 클릭합니다.

추가한 프리셋이 [프리셋 설정] 목록에 나타납니다. 프리셋 목록에서 해당 프리셋 세부 내용을 설정할 수 있습니다.

홈 위치로 이동하기

  • [홈 이동] 버튼을 클릭합니다.

홈 설정하기

  1. PT 제어 화면에서 아이콘을 클릭하고 드래그하여 원하는 홈 위치에 카메라 뷰가 향하도록 조정합니다.
  2. [홈 설정] 버튼을 클릭합니다.

알아두기

  • 홈 위치 설정 후 카메라가 재부팅되면 카메라 화면이 홈 포지션에 위치합니다.

프리셋 설정

프리셋 목록을 확인하고 각 프리셋별로 상세 설정을 수정하거나 삭제할 수 있습니다.

번호

프리셋 설정 시 입력한 입력한 프리셋 번호입니다.

이름

프리셋 설정 시 입력한 입력한 프리셋 이름입니다.

연결 동작

카메라가 해당 프리셋 위치로 이동한 후 실행할 카메라의 동작을 설정합니다.

  • 사용 안 함: 카메라가 해당 프리셋 위치로 이동한 후 아무런 동작을 하지 않습니다.
  • 자동 실행: [자동 실행] 탭에서 설정한 동작이 자동으로 실행됩니다. 자동 실행 [모드]에서 [사용 안 함]을 선택하거나 자동 실행 동작이 없는 경우에는 프리셋 실행 상태가 유지됩니다.
  • 영상 분석: 영상 분석 동작이 실행됩니다.
이동

설정 페이지로 이동하여 세부 옵션을 확인하거나 설정할 수 있습니다.

  • 카메라 설정: [비디오 및 오디오] > [카메라 설정] 페이지로 이동합니다. 선택한 프리셋 주변 환경에 적합하도록 카메라 설정을 변경할 수 있습니다.
  • 움직임 감지: [분석] > [움직임 감지] 페이지로 이동합니다. [움직임 감지] 페이지에서 [타입] 옵션 중 [프리셋]을 선택하면 프리셋별로 움직임 감지 이벤트를 각각 다르게 설정할 수 있습니다.
  • IVA: [분석] > [IVA] 페이지로 이동합니다. [IVA] 페이지에서 [타입] 옵션 중 [프리셋]을 선택하면 프리셋별로 IVA(지능형 영상 분석) 이벤트를 각각 다르게 설정할 수 있습니다.

프리셋으로 이동하기

  1. [번호] 컬럼이나 [이름] 컬럼에서 원하는 프리셋을 클릭합니다. 미리 설절해 둔 프리셋 위치로 카메라 뷰가 이동합니다.

프리셋 삭제하기

  1. [번호] 컬럼 옆의 체크 상자에서 삭제할 프리셋을 선택합니다.
  2. [삭제] 버튼을 클릭하면 프리셋이 삭제됩니다.

그룹

여러 개의 프리셋을 그룹화하여 그룹으로 지정된 프리셋을 연속적으로 호출할 수 있습니다.

그룹 설정하기

  1. [그룹 설정] 번호 목록에서 그룹 번호를 선택합니다.
  2. [프리셋 번호]에서 프리셋 번호를 선택한 후 [속도]에서 프리셋 간 이동 속도를 입력하고 [전환 간격]에서 해당 프리셋에서 머물 전환 간격을 초 단위로 입력합니다.
  3. 2번 과정을 반복하여 프리셋 번호를 2개 이상 입력합니다.
  4. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭합니다.

그룹 작동하기

  1. [그룹 설정] 번호 목록에서 작동할 그룹 번호를 선택합니다.
  2. [시작] 버튼을 클릭하면 해당 그룹에 설정된 프리셋이 전환 간격에 맞춰 변경됩니다.
  3. [중지] 버튼을 클릭하면 그룹으로 작동 중인 프리셋이 멈춥니다.

그룹 삭제하기

  1. [그룹 설정] 번호 목록에서 삭제할 그룹을 선택합니다.
  2. [삭제] 버튼을 클릭하면 그룹이 삭제됩니다.

자동 실행

사용자가 일정 시간동안 카메라를 조작하지 않을 경우 사전에 지정한 동작을 자동으로 실행합니다.

모드

  • 사용 안 함: 자동 실행 기능을 사용하지 않습니다.
  • 홈: 홈 위치로 카메라가 이동합니다.
  • 프리셋: 해당 프리셋 번호로 이동합니다. 프리셋으로 이동할 시간과 이동할 프리셋 위치 번호를 설정할 수 있습니다.
  • 그룹: 그룹 동작을 자동으로 실행합니다. 그룹 동작이 자동으로 실행되는 시간과 그룹 번호를 설정할 수 있습니다. 그룹 상세 동작은 [그룹] 탭에서 설정할 수 있습니다.
  • 스케줄: 설정한 요일과 시간에 설정한 동작이 자동으로 작동하도록 설정할 수 있습니다. 분 단위까지 설정할 수 있으며, 기본 설정은 홈 포지션입니다.
  • 오토팬: 팬 방향으로 360도 회전하는 동작으로 자동으로 실행합니다. 자동으로 실행되는 시간과 카메라가 팬 방향으로 회전하는 속도와 틸트 고정 각도를 설정할 수 있습니다. 예를 들어 틸트 각도를 '30'으로 설정한 경우 틸트 각도는 30도로 유지됩니다.

시간

어느 정도의 시간동안 카메라를 조작하지 않으면 카메라가 자동으로 홈 위치로 이동할지 시간을 선택합니다.

[모드]에서 [홈]을 선택한 경우, '시간' 옵션이 활성화됩니다.
