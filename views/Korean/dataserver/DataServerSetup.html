데이터 서버 설정

Retail Insight 서버를 등록하면, 등록된 서버로 카메라의 데이터 정보(피플 카운팅, 큐 매니지먼트, 히트맵 정보)를 서버로 전송합니다. [데이터 서버 설정] 페이지에서는 Retail Insight 서버 정보를 입력하고 등록할 수 있습니다. 입력을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

데이터 서버 설정

데이터 서버 사용

카메라의 데이터 정보를 Retail Insight 서버에 전송하려면 [데이터 서버 사용]을 선택합니다.

로그인

Retail Insight 서버를 등록하기 위해 서버 정보를 입력할 수 있습니다.

IP

Retail Insight 서버의 IP 주소를 입력합니다.

알아두기

  • [IP] 입력 필드 옆에 서버와의 연결 상태를 나타내는 메시지가 나타납니다.
    • 접속 중: Retail Insight 서버에 접속을 시도 중일 때 나타납니다. 페이지 하단의 [적용] 버튼을 클릭하거나 [데이터 서버 사용]을 선택하면 서버에 접속을 시도합니다.
    • 성공: Retail Insight 서버에 접속되었을 때 나타납니다.
    • 실패: Retail Insight 서버에 접속하지 못했을 때 나타납니다. 메시지가 계속 나타나는 경우 [데이터 서버 설정] 페이지에서 입력한 Retail Insight 서버 정보가 맞는지 확인하십시오.

포트

Retail Insight 서버의 포트값을 입력합니다.

포트값은 80 또는 1024~65535 사이의 값을 사용할 수 있습니다.

ID

Retail Insight 서버에 로그인할 계정의 ID를 입력합니다.

ID는 최대 8자까지 입력할 수 있으며, 특수문자 #%&+=\:<>"'는 사용할 수 없습니다.

비밀번호

Retail Insight 서버에 로그인할 계정의 비밀번호를 입력합니다.

비밀번호는 최소 8자에서 최대 20자까지 입력할 수 있으며, 특수문자 #%&+=\:<>"'는 사용할 수 없습니다.

매장 이름

매장 이름을 입력합니다.

매장 이름은 Retail Insight 서버에서 카메라를 구분하기 위해 사용됩니다.
