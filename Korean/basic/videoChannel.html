비디오 채널

다양한 종류의 피쉬아이 뷰 타입 중 특정 타입으로 고정된 채널을 추가할 수 있습니다.

설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

비디오 채널

비디오 채널을 추가하면 라이브 화면에서 특정 뷰 타입을 선택하여 볼 수 있습니다. 다양한 피쉬아이 뷰 타입이 존재하는데, [비디오 채널]에서 원하는 뷰 타입을 추가하면 [라이브] 페이지에서 선택한 뷰 타입의 프로파일만 선택하여 볼 수 있습니다.

비디오 채널 추가하기

  1. [추가] 버튼을 클릭합니다.
  2. [피쉬아이 뷰 타입] 컬럼의 드롭다운 메뉴에서 원하는 뷰 타입을 선택합니다.

[모두 삭제] 버튼을 클릭하면 설정한 모든 비디오 채널 뷰 타입이 삭제됩니다.

알아두기

  • 비디오 채널은 2개까지만 설정할 수 있습니다.
  • [비디오 채널]에서 피쉬아이 뷰 타입을 추가해도 [비디오 프로파일]에 해당 뷰 타입을 지원하는 프로파일이 없을 경우 [라이브] 페이지에서 재생시킬 수 없습니다.
  • 비디오 채널 설정을 변경하면 카메라가 재부팅됩니다.
