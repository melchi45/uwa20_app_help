PT 제한

카메라의 팬과 틸트 움직임이 제한된 범위에서만 움직이도록 설정할 수 있습니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

알아두기

  • [팬 및 틸트 범위 제한]에서 [사용]을 선택하여, PT 제한 기능을 사용하면 제한된 범위에서만 프리셋을 설정할 수 있습니다.
  • PT 제한 기능을 사용하기 이 전에 설정한 팬과 틸트 움직임 범위를 초과하여 설정한 프리셋이 있으면 프리셋은 초기 설정대로 작동합니다.

PTZ 조정하기

카메라의 팬, 틸트, 및 줌 동작을 조정할 수 있습니다.

PTZ 조정하기

  • 아이콘을 마우스로 드래그하면 카메라가 팬과 틸트로 이동합니다.
  • 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 아이콘을 클릭하여 화면을 확대하거나 아이콘을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
  • 아이콘을 클릭하여 포커스를 가까이(near)이동하거나 아이콘을 클릭하여 포커스를 멀리(far)이동합니다.
  • 카메라 포커스를 자동으로 한 번 맞추려면 버튼을 클릭합니다.

팬/틸트 제한

팬이나 틸트 동작 범위를 제한하려면 [사용]을 선택합니다. 설정된 범위 내에서만 팬과 틸트가 동작합니다.

모드

제한 범위를 설정할 동작을 선택합니다.

틸트 각도

[모드]에서 [틸트 제한]을 선택한 경우 나타납니다. 틸트 움직임 각도를 설정합니다.

동작

팬과 틸트가 동작할 최대 범위를 설정합니다.

팬과 틸트 동작 제한 범위 설정하기

  1. [시작] 버튼을 클릭하여 설정을 시작합니다.
  2. [들어감] 버튼을 클릭하면 카메라 영상 화면에 시작점 위치가 표시됩니다. PTZ를 조정하면서 제한 범위의 시작점 위치를 설정합니다.
  3. [들어감] 버튼을 클릭하면 카메라 영상 화면에 마지막점 위치가 표시됩니다. PTZ를 조정하면서 제한 범위의 마지막점 위치를 설정합니다.
  4. [들어감] 버튼을 한 번 더 클릭합니다.

[종료] 버튼을 클릭하면 현재 설정 중인 값이 모두 삭제됩니다. [시작] 버튼을 클릭하고 동작 제한 범위를 다시 설정할 수 있습니다.

팬/틸트 속도 기울기

카메라의 팬/틸트 동작 제어시 속도 옵션을 선택합니다.

카메라의 팬/틸트 동작을 제어할 때 중심점에서 멀어지면 동작 속도가 빨라지게 됩니다. [지수]를 선택하면 곡선의 형태와 같이 속도가 증가합니다. 즉 느리게 증가하다가 갈수록 빠르게 증가합니다. [선형]을 선택하면 직선의 형태와 같이 속도가 증가하기 때문에 일정한 속도로 빨라집니다.

줌 연동 설정

줌 배율에 따라서 팬, 틸트 이동 속도를 조정합니다. 줌 배율이 증가하면 패, 틸트 이동 속도가 감소합니다.

모드

줌 배율에 따라 팬, 틸트의 수동 조작 속도가 변경되도록 옵션을 선택할 수 있습니다.

팬/틸트 기울기 초기화

카메라의 팬과 틸트 모터 위치를 초기화할지 여부를 설정하고 초기화 주기와 시간을 설정할 수 있습니다.

예를 들어 [사용]을 선택한 후 [실행 시간]은 '2일'을, [초기화 시간]은 '9'로 선택한 경우, 2일마다 오전 9시에 팬/틸트 위치가 초기화됩니다. 카메라 설정 시간이 2000년 1월 1일인 경우 2000년 1월 3일 9시에 초기화 동작이 시작되며, 이 후 2000년 1월 5일, 7일과 같이 2일마다 오전 9시 정각에 팬/틸트 위치가 초기화됩니다.

실행 시간

초기화 주기를 설정합니다. 예를 들어 '2일'을 선택한 경우 2일마다 초기화가 진행됩니다.

초기화 시간

초기화 동작 시간을 설정합니다. 0시부터 23시까지 정각 시간을 선택할 수 있습니다.

알아두기

  • ±10분 정도의 오차가 발생할 수 있습니다.
  • 팬/틸트 초기화 동작 후 이전 PTZ 동작 상태를 유지합니다. 예를 들면 프리셋 동작 중 초기화 동작을 한 경우, 초기화 완료 후 프리셋 동작으로 복귀합니다.
