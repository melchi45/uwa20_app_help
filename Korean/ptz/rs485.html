RS-485

카메라의 RS-485 단자에 연결된 컨트롤러를 통하여 카메라의 PTZ를 제어할 수 있도록 컨트롤러와의 연결 값을 설정합니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

PTZ 조정하기

카메라의 팬, 틸트, 및 줌 동작을 조정할 수 있습니다.

PTZ 조정하기

  • 아이콘을 마우스로 드래그하면 카메라가 팬과 틸트로 이동합니다.
  • 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 아이콘을 클릭하여 화면을 확대하거나 아이콘을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
  • 아이콘을 클릭하여 포커스를 가까이(near)이동하거나 아이콘을 클릭하여 포커스를 멀리(far)이동합니다.
  • 카메라 포커스를 자동으로 한 번 맞추려면 버튼을 클릭합니다.

RS-485 설정

컨트롤러와 PTZ 카메라와의 연결 값을 설정합니다.

프로토콜

컨트롤러와 PTZ 카메라가 통신할 프로토콜을 선택합니다.

카메라 ID

카메라 ID를 선택합니다.

전송 속도

RS-485 통신 속도를 설정합니다.

종료

컨트롤러와 PTZ 카메라 통신 신호의 감쇠를 방지하기 위해서, 컨트롤러와 카메라간 통신 루프에서 가장 먼 경로에 있는 카메라에 대해 종단을 할지 여부를 설정할 수 있습니다.

알아두기

  • 본 조작을 위해서는 네트워크 카메라와 컨트롤러가 정상적으로 연결되어 있어야 합니다. 또한, 컨트롤러는 카메라를 제어할 수 있도록 시리얼 포트가 설정되어 있어야 합니다.
