프리셋 설정

카메라의 프리셋을 설정하고 삭제할 수 있으며, 현재 위치를 카메라의 마지막 위치로 저장할 지 여부를 선택하고 저장 간격을 설정할 수 있습니다.

프리셋

프리셋은 카메라의 뷰를 설정해 두는 것을 의미합니다. 카메라 프리셋을 설정한 뒤 해당 뷰로 쉽게 이동할 수 있습니다. 300개의 프리셋을 설정할 수 있습니다.

프리셋 설정하기

  1. 아래와 같이 화면을 조정하여 카메라 뷰를 조정합니다.
    • [줌]에서 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 버튼을 클릭하여 화면을 확대하거나 버튼을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
    • [포커스]에서 버튼이나 버튼을 클릭하여 카메라 초점을 조절합니다.
    • 카메라 초점을 자동으로 한 번 맞추려면 [포커스]에서 버튼을 클릭합니다.
  2. [프리셋 설정] 버튼을 클릭합니다.
  3. [프리셋 설정] 대화 상자에서 [번호]에서 원하는 프리셋 번호를 선택합니다.
  4. [이름]에 해당 프리셋의 이름을 입력합니다.
  5. [확인] 버튼을 클릭합니다.

추가한 프리셋이 [프리셋 설정]목록에 나타납니다. 프리셋 목록에서 해당 프리셋 세부 내용을 설정할 수 있습니다.

홈 위치로 이동하기

  • [홈 이동] 버튼을 클릭합니다.

홈 설정하기

  1. 아래와 같이 홈 위치를 카메라 뷰가 향하도록 화면을 조정합니다.
    • [줌]에서 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 버튼을 클릭하여 화면을 확대하거나 버튼을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
    • [포커스]에서 버튼이나 버튼을 클릭하여 카메라 초점을 조절합니다.
    • 카메라 초점을 자동으로 한 번 맞추려면 [포커스]에서 버튼을 클릭합니다.
  2. [홈 설정] 버튼을 클릭합니다.

알아두기

  • 홈 위치 설정 후 카메라가 재부팅되면 화면이 홈 포지션에 위치합니다. 마지막 위치와 홈 위치를 같이 설정한 경우, 카메라 재부팅되면 마지막 위치로 이동합니다.

프리셋 설정

설정한 프리셋 목록이 나타납니다.

[프리셋 설정] 대화 상자에서 입력한 번호와 프리셋 이름이 [번호]와 [이름] 컬럼에 각각 나타납니다.

프리셋으로 이동하기

  1. [번호] 컬럼 옆의 체크 상자에서 원하는 프리셋을 선택합니다.
  2. [이동] 버튼을 클릭하면 설정해 둔 위치로 카메라의 뷰가 이동합니다.

프리셋 삭제하기

  1. [번호] 컬럼 옆의 체크 상자에서 삭제할 프리셋을 선택합니다.
  2. [삭제] 버튼을 클릭하면 프리셋이 삭제됩니다.

마지막 위치

카메라 전원을 껐다 켜면 저장된 마지막 위치로 이동하도록 설정할 수 있습니다. 카메라 마지막 위치를 저장할 지 여부를 선택하고 저장 간격을 선택합니다.

마지막 위치

마지막 위치를 저장하려면 [사용]을 선택합니다.

시간

현재 위치를 마지막 위치로 저장할 저장 간격을 설정합니다.

자동 실행

사용자가 일정 시간동안 카메라를 조작하지 않을 경우 사전에 지정한 동작을 자동으로 실행합니다.

모드

  • 사용 안 함: 자동 실행 기능을 사용하지 않습니다.
  • 홈: 홈 위치로 카메라가 이동합니다.

시간

어느 정도의 시간동안 카메라를 조작하지 않으면 카메라가 자동으로 홈 위치로 이동할지 시간을 선택합니다.

[모드]에서 [홈]을 선택한 경우, '시간' 옵션이 활성화됩니다.
