디지털 PTZ 설정

카메라를 물리적으로 상하로 이동하거나(틸트) 좌우로 이동하여(팬) 뷰를 변경하는 것이 아니라 디지털 줌을 실행한 뒤 카메라의 뷰를 이동하도록 설정할 수 있습니다. 프리셋을 설정하거나, 프리셋을 그룹으로 설정하여 카메라 뷰를 변경할 수 있습니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

알아두기

  • 디지털 PTZ를 설정하려면, [비디오 프로파일]>[프로파일 타입]에서 [디지털 PTZ 프로파일]을 선택해야 합니다.
  • 3M Live view 화각은 미리 보기 화면에서 맞춘 화각 보다 좀 더 넓은 화각을 보여줍니다.

채널 선택

카메라 채널을 선택한 후 채널별로 PTZ 관련 세부 항목을 설정할 수 있습니다.

디지털 PTZ메뉴에서 설정된 프리셋은 라이브화면에서 확인할 수 있습니다.

채널 2~5번은 장착된 개별 카메라 영상이고, 채널 1은 2~5번 채널의 카메라 영상을 하나로 연결하여 파노라마 형식으로 보여주는 영상입니다. 채널 6번과 7번은 채널 1 영상에 대해 디지털 PTZ가 적용된 영상입니다.

'디지털 PTZ 설정'은 채널 6번과 7번에만 설정할 수 있습니다.

프리셋

프리셋은 카메라의 뷰를 설정해 두는 것을 의미합니다. 128개의 프리셋을 설정할 수 있습니다.

프리셋 설정하기

  1. PTZ 제어 화면에서 아이콘을 클릭하여 카메라 영상을 확대한 후
  2. PTZ 제어 화면에서
    • 아이콘을 클릭하고 드래그하여 카메라 뷰를 조정합니다.
    • 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 아이콘을 클릭하여 화면을 확대하거나 아이콘을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
  3. [번호]에서 원하는 프리셋 번호를 선택합니다.
  4. [이름]에 해당 프리셋의 이름을 입력합니다.
  5. [추가] 버튼을 클릭합니다.

아이콘을 클릭하면 화면에서 이동하는 물체를 자동으로 추적하며, 아이콘이 로 변경됩니다. 아이콘을 다시 한 번 더 클릭하면 자동 추적이 중지됩니다.

프리셋으로 이동하기

  1. [번호]에서 원하는 프리셋 번호를 선택합니다.
  2. [이동] 버튼을 클릭하면 설정해 둔 위치로 카메라의 뷰가 이동합니다.

프리셋 삭제하기

  1. [번호]에서 삭제할 프리셋 번호를 선택합니다.
  2. [삭제] 버튼을 클릭하면 프리셋이 삭제됩니다.

홈 위치 설정하기

  1. PTZ 제어 화면에서 아이콘을 클릭하여 카메라 영상을 확대한 후 아래와 같이 PTZ를 조정하여 카메라 뷰를 조정합니다.
    • 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 버튼을 클릭하여 화면을 확대하거나 버튼을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
    • 버튼이나 버튼을 클릭하여 카메라 초점을 조절합니다.
  2. 홈 포지션 설정 아이콘 클릭합니다.

홈 위치로 이동하기

  1. 홈으로 이동하기 아이콘 을을 클릭합니다.

분할 화면 선택하기

  1. [분할 화면 선텍]에서 4분할 뷰의 개별 영상 번호를 선택합니다. 1은 4분할 뷰의 왼쪽 상단, 2는 오른쪽 상단, 3은 왼쪽 하단, 4는 오른쪽 하단의 영상 뷰입니다.

그룹

여러 개의 프리셋을 그룹화하여 그룹으로 지정된 프리셋을 연속적으로 호출할 수 있습니다.

그룹 설정하기

  1. [그룹 번호]에서 그룹 번호를 선택합니다.
  2. [프리셋 목록]에서 [추가] 버튼을 클릭한 후 프리셋 목록이 나타나면 [프리셋 번호]에서 프리셋 번호를 선택하고 [전환 간격]에 해당 프리셋에서 머물 전환 간격을 초 단위로 입력합니다.
  3. 2번 과정을 반복하여 프리셋 번호를 2개 이상 입력합니다.
  4. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭합니다.

그룹 작동하기

  1. [그룹 번호]에서 작동할 그룹을 선택합니다.
  2. [시작] 버튼을 클릭하면 해당 그룹에 설정된 프리셋이 전환 간격에 맞춰 변경됩니다.
  3. [중지] 버튼을 클릭하면 그룹으로 작동 중인 프리셋이 멈춥니다.

그룹 삭제하기

  1. [그룹 번호]에서 삭제할 그룹을 선택합니다.
  2. [삭제] 버튼을 클릭하면 그룹이 삭제됩니다.
