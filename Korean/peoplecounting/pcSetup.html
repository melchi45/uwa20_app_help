설정

피플 카운팅은 카메라 영상 분석을 통해 가상의 라인을 지나가는 사람의 수를 카운트하는 것입니다. 피플 카운팅의 '설정'에서는 사람의 수를 카운트하는 조건과 피플 카운팅 정확도를 높이기 위한 세부 사항을 설정합니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

설정

피플 카운팅 사용

피플 카운팅 기능을 사용하려면 [피플 카운팅 사용]을 선택합니다.

설정

인원수를 카운트할 룰을 설정합니다. 피플 카운팅 룰은 2개까지 설정할 수 있습니다.

룰 설정하기

카운팅 룰을 설정하려면 [룰]에서 체크박스를 선택한 후 [이름] 필드에 룰 이름을 입력합니다. 룰 이름은 영문자와 숫자만 15자까지 입력할 수 있습니다. 입력한 이름은 [통계]>[피플 카운팅]>[검색]에 나타납니다.

알아두기

  • 룰 이름을 변경하면, 기존 룰에 대한 카운팅 통계 데이터는 초기화되고, 다시 카운트가 시작됩니다.

카운트 방식 설정하기

피플 카운팅에서 인원을 카운트 하는 방식에는 IN 카운트와 OUT 카운트가 있습니다.

[룰]에서 체크박스를 선택하면 영상에 룰 라인이 나타납니다. 라인의 화살표 방향으로 움직이는 인원은 IN 카운트이며, 화살표 반대 방향으로 움직이는 인원은 OUT 카운트가 됩니다. 룰 라인의 화살표를 클릭하면 화살표 방향을 변경할 수 있습니다.

룰 라인 설정하기

피플 카운팅 기능은 룰 라인을 지나가는 사람을 카운트합니다. 룰 라인의 위치를 변경하려면 영상에 표시된 룰 라인을 클릭한 후 마우스로 드래그합니다. 룰 라인 양쪽 끝에 위치한 꼭지점을 마우스로 드래그하여 라인의 길이를 조절할 수 있습니다.

룰을 선택하여 활성화되면 영상 하단에 실시간 카운팅 데이터가 나타납니다. 실시간 카운팅은 매일00시 기준으로 초기화됩니다.

피플 카운팅 룰 활성화 및 비활성화하기

카운팅 룰을 활성화하려면 [룰]에서 원하는 룰의 체크박스를 선택합니다.

카운팅 룰을 비활성화하려면 [룰]의 원하는 룰의 체크박스를 선택 해제합니다. 피플 카운팅 룰 설정은 그대로 남아 있으므로, 체크상자를 다시 선택하면 동일한 설정으로 룰 기능이 활성화됩니다.

제외 영역

사용자가 지정한 영역에서는 출입하는 사람의 수를 카운트하지 않도록 설정합니다.

제외 영역 설정하기

영상 위에 마우스로 꼭지점 4개를 선택하면 제외 영역이 설정되며 [영역]의 숫자 버튼 색상이 변경됩니다. 제외 영역은 최대 4까지 설정할 수 있습니다.

제외 영역 변경하기

제외 영역의 꼭지점이나 선을 클릭한 뒤 움직여 제외 영역의 크기를 조절할 수 있습니다. 제외 영역을 이동하려면 마우스로 해당 영역을 클릭하여 드래그합니다.

제외 영역 삭제하기

제외 영역을 삭제하려면 화면에서 해당 영역에 마우스 오른쪽 버튼을 클릭한 후 삭제 확인 창에서 [확인] 버튼을 클릭합니다.

영역

[영역]의 숫자 버튼을 클릭하면, 해당 영역이 영상에서 선택됩니다. 선택된 영역은 테두리가 굵은 흰색으로 나타납니다.

캘리브레이션

피플 카운팅 정확도를 향상시키기 위해 카메라 높이나 카운트할 사람의 크기를 설정합니다. 카메라 높이나 사람의 크기 중 하나만 선택하여 적용할 수 있습니다.

카메라 높이

실제 카메라가 설치된 높이를 지정합니다. + 나 - 표시를 클릭하여 값을 변경하거나 카메라 높이를 수치로 입력할 수 있습니다. 입력할 수 있는 값은 최소 2.5m, 최대 4.5m입니다. 카메라 설치 높이를 설정하면, 설치 높이를 기반으로 위치별로 카운트할 사람의 크기를 자동으로 추정합니다.

크기

카운트할 사람의 크기를 직접 지정합니다. 영상 화면에 표시된 상자의 오른쪽 하단 모서리를 마우스로 드래그하여 크기를 조절할 수 있습니다.

알아두기

  • [피플 카운팅] > [캘리브레이션]에서 설정한 값은 [피플 카운팅]과 [히트맵] 기능에 동일하게 적용됩니다.
  • 캘리브레이션은 피플 카운팅 정확도 향상에 필요합니다. 부정확한 캘리브레이션 정보는 피플 카운팅 정확도에 영향을 미칠 수 있습니다.

피플 카운팅 실시간 데이터

설정한 룰과 각 룰의 카운트 방식에 따라 실시간으로 카운트한 인원수 합계가 표로 나타납니다.

리포트

사용

주기적으로 피플 카운팅 데이터를 이메일과 FTP로 수신하려면 [사용]을 선택합니다.

FTP와 이메일에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

스케줄

매일 동일한 시간에 피플 카운팅 데이터를 수신하려면 [일간]을 선택한 후 시간을 선택합니다.

매주 동일한 요일과 시간에 피플 카운팅 데이터를 수신하려면 [주간]을 선택한 후 요일과 시간을 선택합니다.

파일 이름

FTP 서버나 이메일로 피플 카운팅 데이터를 보낼 파일 이름을 입력하고 파일 형식을 선택합니다. 파일 이름은 영문자와 숫자를 포함하여 최대 64자까지 입력할 수 있습니다.

모든 데이터 삭제

피플 카운팅 데이터를 모두 삭제하려면 [모든 데이터 삭제]를 클릭합니다.

알아두기

  • 다음의 경우, 변경된 영상 화면에 맞추어 룰을 다시 설정하고, [모든 데이터 삭제]를 클릭하여 피플 카운팅 검색에 사용되는 이전 데이터를 삭제하는 것을 권장합니다.
    • [비디오 및 오디오] > [비디오 설정] > [비디오 회전]에서 설정 값을 변경한 경우
    • [비디오 및 오디오] > [카메라 설정] > [스페셜] > [DIS]에서 설정 값을 변경한 경우

알아두기

  • 피플 카운팅 기능은 실시간 정확도가 아닌 통계적 데이터 제공을 목적으로 하고 있으며, 이동하는 사물이나 동물도 측정 인원수에 포함될 수 있습니다.
  • 카메라 설치 직후 안정화되는 동안에는 오동작할 수 있습니다. 안정화 시간은 최대 50~60분까지 걸릴 수 있습니다.
  • 피플 카운팅을 구동하기 위한 카메라는 렌즈를 지면에 직하방식으로 설치해야 합니다.(그림1 참고)
  • 다음과 같은 경우에는 측정된 데이터의 오차가 커질 수 있습니다.
    • 시야가 충분히 확보되지 않는 공간에 설치한 경우
    • 야간 및 저조도 환경(그림2 참고)
    • 급격한 조명 변화가 발생하는 환경이나 직사광선, 일몰, 일출, 직접 조명 등과 같이 강한 빛에 영향을 받는 경우(그림3 참고)
    • 출입문 주변과 같이 고정된 위치에 지속적으로 움직임이 나타나는 경우(그림4 참고)
    • 여러 명의 사람이 모여서 이동하는 경우
    • 이동체가 설정한 룰 주변에 머무르는 경우
    • 피쉬아이 카메라의 경우, 영상 외곽 부분에 룰을 설정한 경우(그림5 참고)
  • 캘리브레이션에서 물체 크기를 실제 크기보다 크게 지정할 경우 실제 인원수 보다 적게 측정될 수 있습니다. 반대로 실제 크기보다 작게 지정할 경우 실제 인원수 보다 많이 측정될 수 있습니다.
