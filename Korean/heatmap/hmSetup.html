설정

히트맵은 화면에서의 움직임 빈도를 색상으로 보여줍니다. 히트맵의 '설정'에서는 배경 영상의 색상을 설정하고 히트맵 데이터를 수집할 때 제외할 영역을 설정할 수 있습니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

알아두기

  • [통계]>[피플 카운팅]>[설정] > [캘리브레이션]에서 설정한 카메라의 높이나 사람 크기는 히트맵 기능에 동일하게 적용됩니다.

설정

히트맵 사용

히트맵 기능을 사용하려면 [히트맵 사용]을 선택합니다.

설정

히트맵 배경 영상 색상을 설정합니다.

배경 컬러

히트맵 데이터를 보여 주는 배경 영상의 색상을 선택합니다. [컬러]를 선택하면 배경 영상이 컬러 영상으로 나타나고 [흑백]을 선택하면 배경 영상이 흑백 영상으로 나타납니다.

제외 영역

사용자가 지정한 영역에서의 움직임은 히트맵 정보로 수집되지 않습니다.

제외 영역 설정하기

영상 위에 마우스로 꼭지점 4개를 선택하면 제외 영역이 설정되며 [영역]의 숫자 버튼 색상이 변경됩니다. 제외 영역은 최대 4까지 설정할 수 있습니다.

제외 영역 변경하기

제외 영역의 꼭지점이나 선을 클릭한 뒤 움직여 제외 영역의 크기를 변경할 수 있습니다. 제외 영역을 이동하려면 마우스로 해당 영역을 클릭하여 드래그합니다.

제외 영역 삭제하기

제외 영역을 삭제하려면 화면에서 해당 영역에 마우스 오른쪽 버튼을 클릭한 후 삭제 확인 창에서 [확인] 버튼을 클릭합니다.

영역

[영역]의 숫자 버튼을 클릭하면, 해당 영역이 영상에서 선택됩니다. 선택된 영역은 테두리가 굵은 흰색으로 나타납니다.

리포트

사용

주기적으로 히트맵 데이터를 이메일과 FTP로 수신하려면 [사용]을 선택합니다.

FTP와 이메일에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

스케줄

매일 동일한 시간에 히트맵 데이터를 수신하려면 [일간]을 선택한 후 시간을 선택합니다.

매주 동일한 요일과 시간에 히트맵 데이터를 수신하려면 [주간]을 선택한 후 요일과 시간을 선택합니다.

파일 이름

FTP 서버나 이메일로 히트맵 데이터를 보낼 파일 이름을 입력하고 파일 형식을 선택합니다. 파일 이름은 영문자와 숫자를 포함하여 최대 64자까지 입력할 수 있습니다.

모든 데이터 삭제

히트맵 통계 데이터를 모두 삭제하려면 [모든 데이터 삭제]를 클릭합니다.
