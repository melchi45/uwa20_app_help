검색

대기열 관리는 카메라 영상 분석을 통해 지정한 영역에 체류한 평균 인원수와 누적 체류 시간을 모니터링할 수 있는 기능입니다. 대기열 관리의 '검색' 페이지에서는 대기 중인 인원수 및 대기 시간 통계 데이터를 검색하고 확인할 수 있습니다.

알아두기

  • 대기열 관리의 검색 기능을 사용하려면, 먼저 [통계]>[대기열 관리]>[설정]에서 [대기열 관리 사용]을 선택하고 대기열 관리 영역을 설정해야 합니다.

검색

화면에 [통계]>[대기열 관리]>[설정]에서 설정한 대기열 룰이 나타납니다.  

점유율

대기열 룰별로 실시간 분석 자료를 보여줍니다.

오늘

평균 사람 수

오늘 0시부터 24시까지 각 대기열 룰의 평균 인원 수를 시간별로 그래프로 보여줍니다.

그래프 오른쪽 상단에 위치한 대기열 룰 이름을 선택하면 해당 그래프만 청록색으로 표시됩니다. 그래프 위에 마우스를 올리면 선택한 시간의 인원 정보를 정확한 수치로 보여주는 풍선 도움말이 나타납니다.

누적 시간

대기한 총 대기 누적 시간을 대기열 룰별, 감지 레벨별(높음, 중간)로 각각 분석하여 그래프로 보여줍니다.

감지 레벨은 [설정] 페이지의 [감지 레벨]에서 지정한 것으로, 대기열의 인원수에 따라 '높음'과 '중간'으로 대기 정도를 분류합니다.

예를 들어 'Queue 1- 높음'은 'Queue 1' 대기열에 감지 레벨 '높음' 수준으로 대기 인원이 많았을 때의 총 누적 시간을 나타내며 'Queue 1- 중간'은 'Queue 1' 대기열에 감지 레벨 '중간' 수준으로 대기 인원이 많았을 때의 총 누적 시간을 나타냅니다.

주간

오늘을 기준으로 지난 7일간 날짜별 대기열별 평균 인원 수 추이를 그래프로 보여줍니다.

그래프 위에 마우스를 올리면 선택한 날짜의 인원 정보를 정확한 수치로 보여주는 풍선 도움말이 나타납니다.  

검색

날짜

[최근]을 선택하면 현재 시간을 기준으로 최근 하루, 일 주, 한 달 단위로 검색할 수 있습니다.

[기간]을 선택하면 일간, 월간, 연간 단위로 기간을 지정하여 검색할 수 있습니다. 달력 아이콘을 선택한 뒤 날짜를 변경할 수 있습니다.

룰

검색할 조건을 선택합니다. 대기열 룰과 분석 조건(평균 인원수, 높음이나 중간의 감지 레벨)을 조합하여 선택할 수 있습니다. 예를 들어 'Queue 1'과 '평균'이 만나는 체크박스를 선택하면 'Queue 1' 대기열의 평균 인원수를 검색합니다. 'Queue 1'과 '높음'이 만나는 체크박스를 선택하면 'Queue 1' 대기열에서 '높음' 수준의 인원이 대기한 총 누적 시간을 검색합니다. 'Queue 1'과 만나는 '평균' '높음'을 모두 선택하여 검색할 수도 있습니다.

룰은 [통계]>[대기열 관리]>[설정]에서 설정할 수 있습니다.

대기열 데이터 검색하기
  1. [검색]의 [날짜]에서 검색할 날짜를 선택합니다.
  2. [룰]에서 검색할 대기열과 분석 옵션을 선택합니다. 예를 들어 'Queue 1' 대기열의 평균 대기 인원수와 '높음' 수준의 인원이 대기했던 총 대기 누적 시간을 검색하려면 'Queue 1'과 '평균'이 만나는 체크박스와 'Queue 1'과 '높음'이 만나는 체크박스를 선택합니다.
  3. [검색] 버튼을 클릭합니다. [결과]에 검색 결과가 나타납니다.

결과

대기열 데이터를 검색한 경우, 결과가 그래프와 표로 나타납니다.

'최근 1일'을 검색한 경우 오늘 0시부터 24시까지 시간별로 데이터를 보여줍니다. '최근 1주'를 검색한 경우 오늘을 기준으로 지난 7일간 데이터를 날짜별로 보여줍니다. '최근 1개월'을 검색한 경우 오늘을 기준으로 지난 30일간 데이터를 날짜별로 보여줍니다.

'기간'을 선택한 후 특정 일, 월, 년을 지정한 경우, 지정한 날의 0시부터 24시까지 시간별 데이터를 보여주며, 지정한 달과 년의 날짜별 데이터를 보여줍니다.

대기열 데이터는 대기열 이름과 분석 방식(평균 인원수, 대기 수준 높음, 대기 수준 중간)의 조합으로 '대기열 이름-평균', '대기열 이름-높음', '대기열 이름-중간'으로 분류되어 나타납니다. 그래프에서 옵션을 선택하면 해당 그래프만 청록색으로 표시됩니다.  또한 그래프 위에 마우스를 올리면 선택한 시점의 데이터를 정확한 수치로 보여주는 풍선 도움말이 나타납니다.  

다운로드

검색 실행 후 [다운로드] 버튼을 클릭하면 검색 결과를 다운로드할 수 있습니다. xlsx와 txt파일 형식으로 다운로드할 수 있으며, 다운로드한 파일은 각 브라우저의 기본 저장 경로에 저장됩니다. 파일 이름은 영문자와 숫자를 포함하여 최대 64자까지 입력할 수 있습니다.
