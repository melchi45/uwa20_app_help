핸드 오버

핸드 오버는 이벤트가 발생했을 때 PTZ 카메라에 설정해 둔 PTZ 프리셋 위치로 이동하는 기능입니다. 이 페이지에서는 프리셋 위치로 이동할 카메라와 프리셋 위치를 설정할 수 있습니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

수신 카메라

수신용 PTZ 카메라를 추가하거나 삭제할 수 있습니다. 수신 카메라는 최대 32대까지 등록할 수 있습니다.

수신 카메라로 등록한 카메라의 정보가 표시됩니다.

수신 카메라 추가하기

  1. [추가] 버튼을 클릭합니다.
  2. [카메라 추가] 창에서 핸드 오버를 수신할 PTZ 카메라 정보를 입력합니다.
    • 번호: 수신 카메라 번호를 설정합니다. [분석] 하위 메뉴에서 핸드 오버 수신 카메라를 설정할 때 수신 카메라 번호를 이용합니다.
    • IP 타입: IPv4와 IPv6 중 IP 주소 타입을 설정합니다.
    • IP 주소: 수신 카메라의 IP 주소를 입력합니다.
    • 포트: 수신 카메라의 포트 번호를 입력합니다.
    • 사용자: 수신 카메라에 접속하기 위한 접속 ID를 입력합니다.
    • 비밀번호: 수신 카메라에 접속하기 위한 비밀번호를 입력합니다.
    • 프리셋 번호: 수신 카메라가 이동할 프리셋 위치 번호를 입력합니다. 수신을 위한 PTZ 카메라에 프리셋 번호를 미리 설정해 두어야 하며 핸드 오버 메뉴에서는 미리 설정해 둔 PTZ 프리셋 번호를 입력합니다.
  3. [적용]을 클릭합니다.

수신 카메라 삭제하기

  1. 수신 카메라 목록에서 체크박스를 클릭하여 삭제할 카메라를 선택합니다.
  2. [삭제] 버튼을 클릭한 후 확인 창이 나타나면 [확인]을 클릭합니다.

핸드오버

고정 카메라에 이벤트가 발생하면 PTZ 카메라가 미리 설정해 둔 프리셋 위치로 이동하도록 핸드오버 번호를 설정할 수 있습니다.

프리셋은 PTZ 카메라의 팬, 틸트 위치와 줌 배율을 미리 설정해 두는 것으로, PTZ 프리셋은 [PTZ] > [PTZ 설정] > [프리셋] 탭 > [프리셋 설정] 메뉴에서 설정할 수 있습니다.

설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

알아두기

  • 이 페이지에서 설정한 핸드오버 번호는 '핸드오버 번호' 목록으로 나타납니다. 핸드오버 번호 목록은 [분석] 하위 메뉴의 [움직임 감지], [IVA], 및 [탬퍼링] 각 페이지에서 확인할 수 있습니다.

핸드오버 번호 설정

핸드오버 번호를 추가하고, 각 번호별로 프리셋 정보를 설정할 수 있습니다.

핸드오버 번호 추가하기

  1. [추가] 버튼을 클릭합니다.
  2. [추가] 창에서 다음 항목을 설정합니다.
    • 번호: 핸드오버 번호를 선택합니다. '번호' 항목에서 지정한 번호는 [분석] 하위 메뉴의 [움직임 감지], [IVA], [탬퍼링 감지] 각 페이지에서 [핸드오버] > [핸드오버 번호] 목록으로 나타납니다.
    • CH: PTZ 카메라의 채널 번호는 자동으로 표기되며, 수정할 수 없습니다.
    • 프리셋 번호: 프리셋 번호를 입력합니다. 프리셋 번호는 [PTZ] > [PTZ 설정] > [프리셋] 탭 > [프리셋 설정]에서 미리 설정해 두어야 합니다.
  3. [적용]을 클릭합니다.

핸드오버 번호 삭제하기

  1. 핸드오버 번호 목록에서 체크박스를 클릭하여 삭제 항목을 선택합니다.
  2. [삭제] 버튼을 클릭한 후 확인 창이 나타나면 [확인]을 클릭합니다.
