PTRZ 설정

PTRZ는 PTZ 기능을 가진 카메라가 아니더라도 카메라를 상하좌우로 움직이거나 영상을 확대 축소할 수 있으며, 또한 카메라를 회전할 수 있는 기능입니다. 특히 천장, 벽면, 모퉁이에 카메라를 설치하는 경우 카메라 위치를 상하좌우로 조절하여 최적의 뷰를 확보하는데 유용한 기능입니다.

'PTRZ 설정' 페이지에서는 PTRZ 기능을 이용하여 카메라의 위치를 조정하고 포커스를 설정할 수 있습니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

알아두기

  • PTRZ 기능은 카메라 초기 설정시 원하는 시야를 맞추거나 영상 화면을 수평으로 정렬하는데 사용하는 것을 권장합니다. (웹뷰어는 카메라 설치 후에 접속하는 걸로 판단해서 "PTRZ 기능은 카메랄 설치시 원하는 시야를 맞추거나..."로 썼다가 수정했어요. 의견 부탁드립니다. )

PTRZ 설정

카메라 뷰를 조정하려면 버튼을 마우스로 클릭한 후 상하좌우로 드래그합니다. 카메라 위치가 팬과 틸트로 이동합니다.

또한 카메라를 시계 방향으로 회전하려면 버튼을 클릭합니다. 버튼 클릭을 해제할 때가지 카메라가 시계 방향으로 계속 회전합니다. 카메라를 시계 반대 방향으로 회전하려면 버튼을 클릭합니다. 버튼 클릭을 해제할 때까지 시계 반대 방향으로 계속 회전합니다.

자동 회전

카메라를 회전하여 영상 화면을 수평으로 맞추려면 [자동 회전] 버튼을 클릭합니다.

줌과 포커스 설정

카메라의 줌과 포커스를 조절하고 설정할 수 있습니다.

포커스를 초기화하려면 [초기화] 버튼을 클릭합니다. 포커스가 기본 위치로 조절됩니다.

포커스

화면에 제시된 포커스 버튼을 눌러 수동으로 포커스를 조절합니다. 버튼에 표시된 값은 포커스 이동 간격을 뜻합니다. 1은 포커스를 한번에 1만큼 이동하고, 10은 포커스를 10만큼 이동하고, 100은 포커스를 100만큼 이동시킵니다. ‘-‘ 값은 포커스를 가까이(near) 이동하고, ‘+’ 값은 멀리(far) 이동합니다.

줌

줌 배율을 수동으로 조절합니다. 버튼에 표시된 값은 줌 이동 간격을 뜻합니다. 1은 줌을 한번에 1만큼, 10은 줌을 10만큼, 100은 줌을 100만큼 이동시킵니다. ‘-‘ 값은 줌아웃(Zoom out, wide)하고, ‘+’ 값은 줌인(Zoom in, tele)합니다.

심플 포커스

화면 위에 사용자가 지정한 위치를 기준으로 포커스를 자동으로 조절합니다. 화면 위에 마우스로 클릭하고 드래그 하여 포커스를 맞출 위치를 설정한 후, [심플 포커스] 버튼을 클릭하면 화면에 N(Near)<->F(Far)이 표시되고 사용자가 지정한 위치를 기준으로 포커스가 자동으로 조절됩니다.

알아두기

  • 포커스 조절 중에 피사체에 급격한 변화(급격한 움직임, 사라짐)가 있을 경우, 포커스 조절 중에 급격한 휘도의 변화가 있을 경우, 영상의 콘트라스트가 낮은 경우, 강한 광원이 주위에 있을 경우에 심플 포커스 기능이 제대로 작동하지 않을 수 있습니다.
