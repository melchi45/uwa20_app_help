위치 설정

카메라 영상에 방향 표시 정보가 나타나도록 설정할 수 있습니다.

북쪽/북동쪽/동쪽/남동쪽/남쪽/남서쪽/서쪽/북서쪽 총 8 방향을 설정할 수 있으며, 이 후 PTZ 조작으로 카메라 팬(Pan) 위치가 변경되면 이에 맞추어 현재 영상 화면에서 방향 정보가 업데이트됩니다.

알아두기

  • OSD 크기나 언어 설정을 변경하면 방향 표시 설정값(방위각, 설명 문구, 위치 정보)이 모두 초기화됩니다. OSD 크기는 [비디오 및 오디오]>[카메라 설정]>[OSD]에서, 언어 설정은  [시스템]>[제품 정보]에서 설정할 수 있습니다.

방향 표시기

방향 표시기 사용

방향 표시 기능을 사용하려면 [방향 표시기 사용]을 선택합니다. 8개 방향 중 설정한 모든 방향 정보가 표시됩니다.

[방향 표시기 사용]은 방향 표시 전체 기능 사용 여부를 설정하는 옵션이며, 개별 방향 표시 여부는[방향 표시]에서 설정할 수 있습니다.

방향 표시 설정하기

  1. 다음과 같이 카메라 PTZ를 조정하여, 카메라 방향을 설정할 때 기준이 될 영상 화면을 지정합니다.
    [방향 표시기 사용] 옵션을 선택하지 않아야 PTZ 조그 패널이 활성화됩니다.
    • 아이콘을 마우스로 드래그하면 카메라가 팬과 틸트로 이동합니다.
    • 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 아이콘을 클릭하여 화면을 확대하거나 아이콘을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
    • 아이콘을 클릭하여 포커스를 가까이(near)이동하거나 아이콘을 클릭하여 포커스를 멀리(far)이동합니다.
    • 카메라 포커스를 자동으로 한 번 맞추려면 아이콘을 클릭합니다.
  2. [방향 표시기 사용]을 선택합니다.
  3. 카메라 영상 화면에서 방향 표시 화살표를 선택합니다.
  4. 3번에서 선택한 방향 표시가 영상 화면에 나타나도록 하려면 [방향 표시]에서 [사용]을 선택합니다.
  5. 다음과 같이 세부 정보를 설정할 수 있습니다.
    • 방위각 정보를 보여주려면 [방위각]에서 [사용]을 선택합니다. 방향 화살표 옆에 NE, E, SE, S, SW, W, NW와 같이 방위각 정보가 표시됩니다.
    • 방향 설명 문구를 [텍스트]에서 입력합니다.
    • 방향 표시(방향 표시 화살표, 방위각, 텍스트)의 위치를 [X]와 [Y]에서 각각 설정합니다.  
  6. 설정한 방향 표시를 카메라 영상 화면에서 확인하려면 [미리보기]를 클릭합니다. 설정 화면으로 다시 돌아가려면 [뒤로]를 클릭합니다.
  7. 설정을 적용하려면 페이지 아래의 [적용] 버튼을 합니다.

방향을 설정한 뒤 팬(Pan) 조정으로 카메라가 바라보는 방향이 변경되면, 이에 맞추어 현재 영상 화면에서 방향 정보가 업데이트됩니다.

방향 표시

개별 방향 표시 여부를 설정합니다.

영상 화면에서 화살표를 선택한 뒤 [사용]을 선택하면, 해당 화살표 및 방향 정보가 카메라 영상에 표시됩니다.

방위각

[사용]을 선택하면 해당 방향에 대해 방위각 정보를 표시합니다.

북쪽 방향은 N으로 표시되며, 북동쪽, 동쪽, 남동쪽, 남쪽, 남서쪽, 서쪽,북서쪽은 각각 NE, E, SE, S, SW, W, NW으로 표시됩니다.

[비디오 및 오디오]>[카메라 설정]에서 (북쪽 설정)버튼을 클릭하면 원하는 방향을 북쪽으로 설정할 수 있습니다.

텍스트

해당 방향에 대해 설명 문구를 입력할 수 있습니다.

알아두기

  • 텍스트는 최대 12자까지 입력할 수 있으며, a~z, A~Z, 0~9, 특수 문자 . -  # * + - / ( ) _ ,와 빈 칸만 입력할 수 있습니다.
  • 텍스트의 크기, 색상 및 투명도는 [비디오 및 오디오]>[카메라 설정]>[OSD]의 [크기], [컬러], [OSD 텍스트 투명도]에서 각각 변경할 수 있습니다.
  • 언어 설정을 한글이나 중문으로 설정한 상태에서 입력한 텍스트는 다른 언어로 언어 설정을 변경하면 한글이나 중문으로 입력된 텍스트가 모두 삭제됩니다.

X/Y

카메라 영상 화면에서 방향을 지시하는 화살표가 표시될 위치를 설정합니다.

X/Y 좌표값은 현재 카메라 영상 화면을 기준으로 합니다.

미리보기

설정한 카메라 방향 정보를 카메라 영상에서 확인하려면 [미리보기]를 클릭합니다. 설정 상태로 돌아오려면 [뒤로]를 클릭합니다.
