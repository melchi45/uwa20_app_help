온도 감지

사용자가 지정한 관심 영역에서 급격한 온도 변화가 발생한 경우, 온도 감지 이벤트를 발생시킬 수 있습니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오. 온도 감지 이벤트에 대한 설정은 [이벤트]>[이벤트 설정] 페이지의 [온도 감지]에 동일하게 적용됩니다.

온도 감지

온도 변화 감지 사용

온도 변화 감지 이벤트를 사용하려면 [온도 변화 감지 사용]을 선택합니다.

온도 감지 사용

온도 감지 이벤트를 사용하려면 [온도 감지 사용]을 선택합니다.

영역

온도 변화량을 감지할 영역을 설정합니다. 설정한 영역의 오른쪽 아래 꼭지점을 움직여 관심 영역의 크기를 변경할 수 있으며, 영역을 이동하려면 마우스로 해당 영역을 클릭하여 드래그합니다. 영역을 삭제하려면 화면에서 해당 영역에 마우스 오른쪽 버튼을 클릭한 후 삭제 확인 창에서 [확인] 버튼을 클릭합니다.

영역은 3개까지 설정할 수 있으며, 영역별로 온도 변화 감지 규칙을 설정할 수 있습니다.

영역

온도를 감지할 영역을 설정합니다. 설정한 영역의 오른쪽 아래 꼭지점을 움직여 관심 영역의 크기를 변경할 수 있으며, 영역을 이동하려면 마우스로 해당 영역을 클릭하여 드래그합니다. 영역을 삭제하려면 화면에서 해당 영역에 마우스 오른쪽 버튼을 클릭한 후 삭제 확인 창에서 [확인] 버튼을 클릭합니다.

영역은 3개까지 설정할 수 있으며, 영역별로 온도 감지 규칙을 설정할 수 있습니다.

온도 변화 감지

[영역]에서 온도 변화를 감지할 영역을 선택한 후 아래 옵션을 설정합니다.

  • 온도 차이: 어느 정도 이상의 온도 변화가 발생할 때 이를 온도 변화로 감지할 지, 감지를 위한 온도 변화량을 선택합니다. 섭씨와 화씨 중 온도 타입을 선택할 수 있으며, 온도 감소와 증가 모두를 포함합니다. 예를 들어 '섭씨 20도'를 선택한 경우, 섭씨 20도 이상의 온도 변화가 발생하면, 카메라가 온도 변화 상태로 감지합니다.
  • 온도 타입:[영역]에서 설정한 영역에는 다양한 범위의 온도가 존재할 수 있습니다. 다양한 온도 중 온도 변화량을 관찰할 때 기준이 될 온도를 '최저' '평균' '최고' 중에서 선택합니다. 예를 들어 [영역]에서'3'을선택하고 [온도 차이]에서는 '섭씨 20도'를 선택한 후, [온도 타입]에서 '최저'를 선택한 경우, 영역 3에서 관찰되는 최저 온도를 기준으로 '섭씨 20도' 이상의 온도 변화가 발생하면 카메라가 온도 변화 상태로 감지합니다.

온도 타입

관찰할 온도 타입을 선택합니다.

[영역]에서 설정한 영역에서는 3가지 유형의 온도가 제공됩니다. 선택한 영역에서 '최저' '평균' '최고' 온도 중 온도 변화를 분석할 때 기준이 될 온도 타입을 선택합니다.

감지 조건

관찰할 온도 조건을 선택합니다.

관찰 온도가 특정 온도 이상이나 이하로 변하는 것을 감지하려면 [이상] 이나 [이하]를 각각 선택합니다. 관찰 온도가 일정값 이상으로 증가하거나 감소하는 것(온도 변화량)을 감지하려면 [증가]나 [감소]를 각각 선택합니다.

감지 온도

온도 값을 입력합니다.

[감지 조건]에서 [이상]이나 [이하]를 선택한 경우, 감지하려는 온도를 입력합니다. 또한 [감지 조건]에서 [증가]나 [감소]를 선택한 경우, 감지하려는 온도변화량(증가량 또는 감소량)을 입력합니다.

최소 관찰 시간 (초)

온도 변화를 감지하고 이를 이벤트로 발생시키기까지 최소한의 시간을 설정합니다. 온도 변화가 최소 관찰 시간 동안 지속되어야 온도 감지 이벤트가 발생합니다. 1~60초 사이의 값을 설정할 수 있습니다. 입력 필드에 직접 입력하거나 슬라이드 막대를 움직여 값을 변경할 수 있습니다.

최소 관찰 시간 (초)

온도를 감지하고 이를 이벤트로 발생시키기까지 최소한의 시간을 설정합니다. [감지 타입] 설정에 따라, 최소 관찰 시간 동안 이상이나 이하의 특정 온도가 지속되거나, 증가하거나 감소한 온도 변화가 최소 관찰 시간 동안 지속되어야 온도 감지 이벤트가 발생합니다. 1~60초 사이의 값을 설정할 수 있습니다. 입력 필드에 직접 입력하거나 슬라이드 막대를 움직여 값을 변경할 수 있습니다.

온도 감지 이벤트 설정 예(1)

영역 1에서 최대 온도 값이 섭씨 100도 이상되는 경우가 3초 이상 지속될 경우 이벤트 알림을 발생하려면 다음과 같이 설정합니다.

  • 영역: 영역 1을 선택합니다.
  • 온도 타입: '최저'를 선택합니다.
  • 감지 조건: '이상'을 선택합니다.
  • 감지 온도: '100'을 입력합니다.
  • 최소 관찰 시간: '3'을 입력합니다.

이벤트 알림 동작과 일정은 [이벤트 동작] 메뉴와 [이벤트 동작 스케줄] 메뉴에서 각각 설정할 수 있습니다.

온도 감지 이벤트 설정 예(2)

영역 3내에서 최소 온도 값이 기준 온도 대비 섭씨 30도 이상 증가하고 이 상태가 5초 이상 지속될 경우에 이벤트 알림을 발생하려면 다음과 같이 설정합니다.

  • 영역: 영역 3을 선택합니다.
  • 온도 타입: '최저'를 선택합니다.
  • 감지 조건: '증가'를 선택합니다.
  • 감지 온도: '30'을 입력합니다.
  • 최소 관찰 시간: '5'을 입력합니다.

이벤트 알림 동작과 일정은 [이벤트 동작] 메뉴와 [이벤트 동작 스케줄] 메뉴에서 각각 설정할 수 있습니다.

온도 오버레이

카메라 영상 화면에 표시할 항목을 선택합니다. [영역]에서 영역을 선택한 뒤 영역별로 표시 항목을 선택할 수 있습니다.

  • 영역: 영역 번호를 화면에 표시할지 여부를 선택합니다.
  • 평균: 설정된 영역의 평균 온도를 화면에 표시할지 여부를 선택합니다.
  • 최저: 설정된 영역의 최저 온도를 화면에 표시할지 여부를 선택합니다.
  • 최고: 설정된 영역의 최고 온도를 화면에 표시할지 여부를 선택합니다.

선택한 항목은 [라이브] 영상 화면 상단에도 오버레이되어 나타납니다.

영역 방사율

각 영역의 방사율 값을 설정할 수 있습니다. 알맞은 방사율 값을 설정해야 측정 영역의 온도를 보다 정확하게 얻을 수 있습니다.

알아두기

방사율은 [비디오 및 오디오]>[온도 설정]>[스폿 방사율]의 표를 참고하십시오. 방사율은 표면 처리, 색상, 측정 당시의 온도, 대기 온도 등 다양한 변수에 따라 달라지므로 반복된 실험을 통해 적절한 값을 찾아야 합니다. 정확한 방사율 값을 입력하면 보다 정확한 온도 측정값을 얻을 수 있습니다.

이벤트 동작

FTP

온도 감지 이벤트가 발생했을 때 영상 이미지를 FTP 서버에 저장하려면 [사용]을 선택합니다. FTP에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

이메일

온도 감지 이벤트가 발생했을 때 영상 이미지를 이메일로 전송하려면 [사용]을 선택합니다. 이메일에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

저장

온도 감지 이벤트가 발생했을 때 (micro)SD 카드 또는 NAS에 영상을 저장하려면 [사용]을 선택합니다. 저장에 관한 상세 설정은 [이벤트]>[저장]에서 할 수 있습니다.

알람 출력

온도 감지 이벤트가 발생했을 때 특정 알람 번호로 알람이 출력되도록 설정할 수 있습니다. 알람 출력 번호를 선택한 후, 알람을 출력하지 않으려면 [사용 안 함]을 선택하고 알람을 출력하려면 출력 시간을 선택합니다. [항상 동작]을 선택하여 사용자가 알람을 멈추기 전까지 계속 출력되도록 하거나 5, 10, 15초까지 출력하도록 설정할 수 있습니다. 알람 출력에 관한 상세 설정은 [이벤트]>[알람 출력]에서 할 수 있습니다.

카메라가 지원하는 출력 알람 개수에 따라 알람 출력 옵션이 다르게 표시됩니다. 예를 들어 출력되는 알람이 2개라면 ‘알람 출력 1’, ‘알람 출력 2’와 같이 옵션이 표시됩니다.

프리셋 이동

온도 감지 이벤트가 발생했을 때 카메라 뷰 위치를 미리 설정해 둔 프리셋 번호로 이동하도록 설정할 수 있습니다. 카메라 위치 프리셋은 [PTZ]>[디지털 PTZ]나 [외부 PTZ]의 [프리셋]에서 설정할 수 있습니다.

프리셋 이동

온도 감지 이벤트가 발생했을 때 카메라 뷰 위치를 미리 설정해 둔 위치로 이동하도록 설정할 수 있습니다. 카메라 위치 프리셋은 [PTZ]>[프리셋 설정]의 [프리셋 설정] 영역에서 설정할 수 있습니다.

프리셋 이동

온도 감지 이벤트가 발생했을 때 카메라 위치를 미리 설정해 둔 위치로 이동하도록 설정할 수 있습니다. 카메라 위치 프리셋은 [PTZ]>[PTZ 설정]의 [프리셋] 탭에서 설정할 수 있습니다.

프리셋 이동

온도 감지 이벤트가 발생했을 때 카메라 뷰 위치를 미리 설정해 둔 위치로 이동하도록 설정할 수 있습니다. 카메라 위치 프리셋은 [PTZ]>[PT 설정]의 [프리셋]에서 설정할 수 있습니다.

이벤트 동작 스케줄

온도 감지 이벤트가 발생했을 때 [이벤트 동작]에서 설정한 이벤트 동작이 항상 실행하도록 하거나 정해진 시간에만 실행하도록 설정합니다.

항상 동작

온도 감지 이벤트가 발생했을 때 설정한 이벤트 동작을 항상 실행합니다.

설정한 시간에만 동작

설정한 시간에 온도 감지 이벤트가 발생했을 때 설정한 이벤트 동작을 실행합니다. [설정한 시간에만 동작]을 선택하여 나타나는 시간표에는 현재 카메라 시간을 기준으로 한 일요일부터 토요일까지의 날짜가 표시됩니다. 시간표에서 마우스를 클릭하거나 드래그하여 이벤트 동작 시간을 설정할 수 있습니다. 설정된 이벤트 동작 스케줄은 해당 요일과 시간에 반복적으로 실행됩니다.

[1분], [30분], [1시] 버튼을 클릭하여 시간 보기 단위를 변경할 수 있습니다. [초기화] 버튼을 클릭하면 설정된 이벤트 동작 스케줄이 모두 삭제됩니다. 카메라 시간을 확인하거나 변경하려면 [Basic]>[날짜 및 시간]을 참고하십시오.
