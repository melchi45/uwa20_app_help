자동 추적

자동 추적 기능은 영상을 분석하여 움직이는 물체를 인지하고 추적하는 기능입니다. 자동 추적 사용을 위한 세부 내용을 설정할 수 있습니다.

설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

알아두기

카메라 자동 추적을 실행할 수 있는 방법은 다음과 같습니다.

  • 자동 추적 아이콘 을 클릭합니다. 전체 영역에서 추적 대상을 분석 및 선정하고 추적합니다.
  • 라이브 페이지의 영상 화면에서 마우스 오른쪽 버튼을 클릭하면 클릭한 영역을 중심으로 움직임을 분석한 후 자동 추적을 시작합니다.
  • [PTZ] > [PTZ 설정] > [프리셋] > [프리셋 설정] > [연결 동작]에서 '자동 추적'을 선택한 경우, 해당 프리셋으로 카메라가 이동하고 자동 추적 기능이 자동으로 실행됩니다.

PTZ 조정하기

카메라의 팬, 틸트, 및 줌 동작을 조정할 수 있습니다.

PTZ 조정하기

  • 아이콘을 마우스로 드래그하면 카메라가 팬과 틸트로 이동합니다.
  • 줌 막대를 조절하여 영상 배율을 조정합니다. 혹은 아이콘을 클릭하여 화면을 확대하거나 아이콘을 클릭하여 화면을 축소하여 영상 배율을 선택합니다.
  • 아이콘을 클릭하여 포커스를 가까이(near)이동하거나, 아이콘을 클릭하여 포커스를 멀리(far)이동합니다.
  • 자동 추적을 시작하려면 아이콘을 클릭합니다. 아이콘을 클릭하면 자동 추적이 멈춥니다.
  • 카메라 포커스를 자동으로 한 번 맞추려면 아이콘을 클릭합니다.

제외 영역

추적 대상 감지 제외 영역 목록이 표시됩니다. 영역은 최대 8개까지 설정할 수 있습니다.

제외 영역 설정하기

  1. 카메라 영상 화면에 마우스를 드래그하여 제외 영역을 설정합니다.
  2. [자동 추적] 대화 상자에서 제외 영역 이름을 입력하고 [확인] 버튼을 클릭합니다.

제외 영역 삭제하기

  1. 제외 영역 목록에서 삭제할 번호를 선택합니다.
  2. [삭제] 버튼을 클릭합니다.

알아두기

  • 자동 추적 제외 기능은 [자동 추적 설정] > [영역 동작]을 선택했을 때만 작동됩니다.
  • 자동 추적 제외 영역은 초기 대상 감지 영역에서 제외된다는 의미이며, 추적 중인 대상이 제외 영역에 들어가는 경우는 계속 추적합니다. 즉, 설정한 영역은 추적 중인 대상이 영역 내부로 진입할 때 추적을 멈추도록 하는 역할을 하지는 않습니다.

설정

카메라 높이

카메라가 높이를 설정합니다. 카메라가 설치된 실제 높이에 맞게 입력할수록 자동 추적 기능이 좀 더 정확하게 작동합니다.

줌

자동 추적 시 줌 동작을 사용할지 여부를 선택합니다.

대상 크기 유지

줌 사용을 선택한 경우 추적 대상에 대하여 화면 안에 유지되길 원하는 크기를 지정합니다. 전체 영상 화면 크기에 대비하여 자동 추적 대상의 크기를 조정하고 추적 대상의 크기를 유지하면서 자동 추적을 실행합니다.

[줌]에서 [사용]을 선택한 경우에만 작동합니다.

지시자 표시

영상 화면에 추적하고 있는 대상을 표시해주는 지시자를 사용할지 여부를 선택합니다.

지시사 표시를 [사용]으로 선택한 경우 녹화 영상에서도 지시자가 표시됩니다.

제외 영역

설정한 영역에서 감지한 물체는 자동 추적에서 제외합니다. 제외 영역을 설정하는 방법은 '자동 추적 제외 영역 설정하기'을 참고 하십시오.

[사용]을 선택한 경우 설정한 모든 제외 영역이 사용됩니다.

지정한 영역에서는 움직임이 감지되어도, 자동 추적이 시작되지 않습니다.

자동 종료

자동 추적 기능의 자동 종료 여부를 설정합니다. [사용]를 선택하면 일정 기간(60초) 동안 추적 대상이 감지되지 않을 경우 종료되어 자동 추적 동작을 더이상 유지하지 않습니다. [사용]을 선택하지 않으면 추적 대상 탐색을 통한 자동 추적 동작을 계속 유지할 수 있습니다.

알아두기

  • 자동 추적 기능이 정상적으로 작동하기 위한 조건은 아래와 같습니다.
    • 설치 높이: 최소 2.5m 이상, 최대 30m 이하
    • 설치 형태: 기울어짐 없이 수평이 유지된 실내 또는 실외 천장형
    • 추적 대상: 지평면 상에서 이동하는 직립 보행 중인 사람(높이 약 170cm)
  • 다음의 상황에서는 대상을 추적할 수 없거나 오동작이 발생할 수 있습니다.
    • 촬영 영역에 다수의 이동체가 존재하는 경우
    • 다른 이동체나 시설로 가려지거나 겹치는 경우
    • 이동체와 배경과의 대비차가 거의 없는 경우
    • 너무 작거나 너무 큰 물체
    • 너무 빠르거나 너무 느리게 움직이는 경우
    • 이동체가 설치된 카메라의 바로 아래를 지나는 경우
    • 이동체의 이동 경로가 설치 높이를 기준으로 하는 지면이 아닌 경우
    • 강한 바람, 눈, 비와 같은 악천후 환경
    • 촬영 영역이 어두운 경우
    • 급격한 조명 변화가 있는 경우
    • 촬영 영역 또는 카메라를 향한 빛 깜박임이 있는 경우
    • 빛 반사 또는 라이트 등 카메라를 향한 빛이 있는 경우
    • 카메라 흔들림 등의 이유로 영상이 흔들리는 경우
    • 렌즈 커버 및 조광기 커버가 더럽혀져 있거나 젖은 경우
    • 카메라 렌즈의 축에 변화가 있는 경우

이벤트 동작

이메일

자동 추적 이벤트가 발생했을 때 영상 이미지를 이메일로 전송하려면 [사용]을 선택합니다. 이메일에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

저장

자동 추적 이벤트가 발생했을 때 (micro)SD 카드 또는 NAS에 영상을 저장하려면 [사용]을 선택합니다. 저장에 관한 상세 설정은 [이벤트]>[저장]에서 할 수 있습니다.

자동 추적 이벤트가 발생했을 때 (micro)SD 카드에 영상을 저장하려면 [사용]을 선택합니다. 저장에 관한 상세 설정은 [이벤트]>[저장]에서 할 수 있습니다.

알람 출력

자동 추적 이벤트가 발생했을 때 특정 알람 번호로 알람이 출력되도록 설정할 수 있습니다. 알람 출력 번호를 선택한 후, 알람을 출력하지 않으려면 [사용 안 함]을 선택하고 알람을 출력하려면 출력 시간을 선택합니다. [항상 동작]을 선택하여 사용자가 알람을 멈추기 전까지 계속 출력되도록 하거나 5, 10, 15초까지 출력하도록 설정할 수 있습니다. 알람 출력에 관한 상세 설정은 [이벤트]>[알람 출력]에서 할 수 있습니다.

심플 포커스

자동 추적 이벤트가 발생했을 때 심플 포커스를 실행시켜 포커스를 자동으로 맞춰줍니다. 이벤트 발생 시 심플 포커스는 한 번만 실행됩니다.

이벤트 동작 스케줄

자동 추적 이벤트가 발생했을 때 [이벤트 동작]에서 설정한 이벤트 동작이 항상 동작하도록 하거나 설정한 시간에만 동작하도록 설정합니다.

항상 동작

자동 추적 이벤트가 발생했을 때 설정한 이벤트 동작을 항상 실행합니다.

설정한 시간에만 동작

설정한 시간에 자동 추적 이벤트가 발생했을 때 설정한 이벤트 동작을 실행합니다. [설정한 시간에만 동작]을 선택하여 나타나는 시간표에는 현재 카메라 시간을 기준으로 한 일요일부터 토요일까지의 날짜가 표시됩니다. 시간표에서 마우스를 클릭하거나 드래그하여 이벤트 동작 시간을 설정할 수 있습니다. 설정된 이벤트 동작 스케줄은 해당 요일과 시간에 반복적으로 실행됩니다.

[1분], [30분], [1시] 버튼을 클릭하여 시간 보기 단위를 변경할 수 있습니다. [초기화] 버튼을 클릭하면 설정된 이벤트 동작 스케줄이 모두 삭제됩니다. 카메라 시간을 확인하거나 변경하려면 [Basic]>[날짜 및 시간]을 참고하십시오.
