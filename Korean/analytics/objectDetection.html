개체 감지

카메라 영상에서 사용자가 설정한 개체를 감지하면 개체 감지 이벤트를 발생시킬 수 있습니다. 카메라가 감지하는 개체 타입은 사람, 차량, 얼굴, 번호판으로 4가지 타입이 있습니다.

설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오. 개체 감지 이벤트에 대한 설정은 [이벤트]>[이벤트 설정] 페이지의 [개체 감지]에 동일하게 적용됩니다.

채널 선택

카메라 채널을 선택한 후 채널별로 개체 감지 세부 항목을 설정할 수 있습니다.

채널 1은 라이브 영상을 보여 주는 채널이며, 채널 2는 디지털 PTZ 영상을 보여 주는 채널입니다. [개체 감지]는 채널 1에만 설정할 수 있습니다.

개체 감지

개체 감지 사용

개체 감지 이벤트를 사용하려면 [개체 감지 사용]을 선택합니다.

[개체 감지 사용]을 선택하면, [개체]에서 선택한 개체가 [최소 관찰 시간]에서 설정한 시간보다 더 오래 화면에서 감지되면 개체 감지 이벤트가 발생합니다.

개체

카메라가 감지할 개체 유형을 설정합니다.

개체

사람, 자동차, 얼굴, 번호판 4가지 유형 중 카메라가 감지할 개체 유형을 선택합니다.

최소 관찰 시간(초)

감지 후 이벤트를 발생하기 위한 최소시간을 설정합니다. 설정한 시간보다 더 오래 움직임이 지속될 때 이벤트를 발생시킵니다.

제외 영역

[개체]에서 선택한 개체를 감지하지 않고 제외할 영역을 설정합니다.

제외 영역 설정하기

미리보기 화면에서 마우스로 꼭지점 4개를 선택하거나 마우스로 드래그하여 사각형을 그리면 제외 영역이 설정됩니다. 제외 영역은 최대 8개까지 설정할 수 있습니다.

제외 영역 변경하기

설정한 제외 영역의 꼭지점을 움직여 제외 영역의 크기를 변경할 수 있습니다.

다각형을 만들려면 만들어진 사각형의 선을 선택한 후 + 기호가 나타나면 +기호를 클릭하면 꼭지점이 추가됩니다. 최대 4개의 점을 추가하여 원하는 다각형을 만들 수 있습니다.

제외 영역을 선택하고 마우스로 드래그하면 제외 영역의 위치를 이동할 수 있습니다.

제외 영역 삭제하기

제외 영역을 삭제하려면 화면에서 해당 영역에 마우스 오른쪽 버튼을 클릭한 후 삭제 확인 창에서 [확인] 버튼을 클릭합니다.

영역

제외 영역이 추가되면 순서대로 [영역]의 숫자 버튼 색상이 변경됩니다. 숫자 버튼을 클릭하면 해당 제외 영역이 미리보기 화면에서 선택됩니다.

알아두기

  • [개체 감지]의 [제외 영역] 탭에서 설정한 값은 [IVA]>[제외 영역] 값과 연동합니다. 즉 [개체 감지]의 [제외 영역] 탭에서 제외 영역을 설정하면 [IVA]>[제외 영역]에도 동일한 제외 영역이 생성됩니다.

공통 설정

개체를 감지할 때 민감도와 감지할 개체의 최소 크기와 최대 크기를 설정합니다.

사용 환경에 따라 감지 정확도를 높이고 오감지를 최소화 하기 위해 적용할 수 있는 설정입니다.

민감도

개체 감지 민감도를 설정합니다. 배경과 물체의 구분이 명확한 환경에서는 민감도를 낮게 설정하고, 어두워서 배경과 물체의 구분이 명확하지 않은 환경에서는 민감도를 높게 설정합니다.

크기

감지할 개체의 최소 크기와 최대 크기를 설정합니다. 오른쪽 하단의 꼭지점을 클릭한 후 마우스를 드래그하여 크기를 변경할 수 있습니다. 크기를 변경하면 [크기]에서 [최소] 값과 [최대] 값도 변경됩니다.

제외 영역

AI 카메라는 각 개체별로 세부 속성을 분석한 데이터를 카메라와 연동하는 프로그램 등에서 활용할 수 있도록 전송합니다. 이 때 제외 영역으로 설정된 영역에서 감지한 개체 정보는 기본적으로 전송하지 않습니다. 제외 영역에서 감지한 개체의 정보와 각 개체의 세부 속성을 전송하려면 [감지 제외 영역의 개체 데이터 사용]을 선택합니다.  

카메라가 전송하는 개체별 세부 속성 정보는 "AI Camera Metadata Specification" 문서를 참조하십시오.

알아두기

  • [개체 감지]의 [공통 설정] 탭에서 설정한 값은 [IVA]>[공통 설정] 값과 연동합니다. 즉 [개체 감지]의 [공통 설정] 탭에서 값을 변경하면 [IVA]>[공통 설정] 값도 동일하게 변경됩니다.
  • 사용자가 지정한 최소 크기보다 작은 개체나 최대 크기보다 큰 개체는 감지하지 않습니다. 크고 작은 노이즈에 의한 오감지를 피하기 위하여 설치 환경에 적합한 최소/최대 감지 크기를 설정하십시오. 단, 동일한 위치에서의 동일한 움직임이라도 감지되는 크기는 다소 차이가 있을 수 있기 때문에 편차를 고려하여 여유를 포함하여 최소/최대 크기 제한을 설정하십시오.

BestShot

BestShot은 AI 카메라가 감지한 개체(사람, 얼굴, 차량, 번호판)의 정보를 분석하여 가장 신뢰할 수 있는 개체를 이미지로 생성한 것입니다. 이미지는 JPEG 형식이며, 카메라 영상 화면 전체가 아닌 해당 개체를 포함하는 일부 영역만 잘라 이미지로 추출합니다.

카메라와 연동하는 프로그램에서 활용할 수 있도록 BestShot의 이미지 데이터를 전송할 수 있습니다.

BestShot

BestShot을 생성할 개체 타입을 선택합니다.

선택한 개체의 BestShot은 웹뷰어 라이브 페이지의 왼쪽에 위치한 BestShot 패널에서 확인할 수 있습니다. BestShot 이미지 하단에는 각 개체의 세부 속성 정보가 아이콘으로 표시됩니다. 예를 들어 '자동차' 개체의 경우, 차량 타입(자동차, 버스, 트럭, 오토바이, 기차, 자전거)과 차량 색상 정보가 BestShot 이미지 하단에 표시됩니다.

이벤트 동작

FTP

개체 감지 이벤트가 발생했을 때 영상 이미지를 FTP 서버에 저장하려면 [사용]을 선택합니다. FTP에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

이메일

개체 감지 이벤트가 발생했을 때 영상 이미지를 이메일로 전송하려면 [사용]을 선택합니다. 이메일에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

저장

개체 감지 이벤트가 발생했을 때 (micro)SD 카드 또는 NAS에 영상을 저장하려면 [사용]을 선택합니다. 저장에 관한 상세 설정은 [이벤트]>[저장]에서 할 수 있습니다.

개체 감지 이벤트가 발생했을 때 (micro)SD 카드에 영상을 저장하려면 [사용]을 선택합니다. 저장에 관한 상세 설정은 [이벤트]>[저장]에서 할 수 있습니다.

알람 출력

개체 감지 이벤트가 발생했을 때 특정 알람 번호로 알람이 출력되도록 설정할 수 있습니다. 알람 출력 번호를 선택한 후, 알람을 출력하지 않으려면 [사용 안 함]을 선택하고 알람을 출력하려면 출력 시간을 선택합니다. [항상 동작]을 선택하여 사용자가 알람을 멈추기 전까지 계속 출력되도록 하거나 5, 10, 15초까지 출력하도록 설정할 수 있습니다. 알람 출력에 관한 상세 설정은 [이벤트]>[알람 출력]에서 할 수 있습니다.

카메라가 지원하는 출력 알람 개수에 따라 알람 출력 옵션이 다르게 표시됩니다. 예를 들어 출력되는 알람이 2개라면 ‘알람 출력 1’, ‘알람 출력 2’와 같이 옵션이 표시됩니다.

프리셋 이동

움직임 감지 이벤트가 발생했을 때 카메라 뷰 위치를 미리 설정해 둔 프리셋 번호로 이동하도록 설정할 수 있습니다. 카메라 위치 프리셋은 [PTZ]>[디지털 PTZ]나 [외부 PTZ]의 [프리셋]에서 설정할 수 있습니다.

[PTZ]>[외부 PTZ]에서 [외부 PTZ 사용] 옵션을 선택한 경우, [프리셋 이동] 옵션이 보이지 않습니다.

이벤트 동작 스케줄

개체 감지 이벤트 동작의 작동 시간을 설정합니다. [이벤트 동작]에서 설정한 이벤트 동작이 항상 동작하도록 하거나 설정한 시간에만 동작하도록 설정합니다.

항상 동작

개체 감지 이벤트가 발생했을 때 설정한 이벤트 동작을 항상 실행합니다.

설정한 시간에만 동작

설정한 시간에 개체 감지 이벤트가 발생했을 때 설정한 이벤트 동작을 실행합니다. [설정한 시간에만 동작]을 선택하여 나타나는 시간표에는 현재 카메라 시간을 기준으로 한 일요일부터 토요일까지의 날짜가 표시됩니다. 시간표에서 마우스를 클릭하거나 드래그하여 이벤트 동작 시간을 설정할 수 있습니다. 설정된 이벤트 동작 스케줄은 해당 요일과 시간에 반복적으로 실행됩니다.

[1분], [30분], [1시] 버튼을 클릭하여 시간 보기 단위를 변경할 수 있습니다. [초기화] 버튼을 클릭하면 설정된 이벤트 동작 스케줄이 모두 삭제됩니다. 카메라 시간을 확인하거나 변경하려면 [Basic]>[날짜 및 시간]을 참고하십시오.
