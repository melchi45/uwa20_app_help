라이브 화면

카메라가 촬영 중인 라이브 영상을 확인하고, 화면 캡처 등의 여러 가지 기능을 제어할 수 있습니다. 화면 상단의 버튼을 클릭하면 라이브 화면이 나타납니다.

상단의 버튼을 클릭하여 라이브 화면과 재생 화면, 설정 화면으로 이동할 수 있습니다.

  • (라이브): 카메라가 촬영 중인 라이브 화면을 확인하고 다양한 카메라 기능을 제어할 수 있습니다.
  • (재생): (micro)SD 카드 또는 NAS에 녹화해 둔 영상을 검색하고 재생하여 볼 수 있습니다.
  • (재생): (micro)SD 카드에 녹화해 둔 영상을 검색하고 재생하여 볼 수 있습니다.
  • (설정): 카메라의 상세 설정을 변경할 수 있습니다.

알아두기

  • Chrome으로 웹뷰어에 접속하면 재생 화면의 기능을 가장 안정적으로 사용할 수 있습니다.
  • 라이브 페이지에서 영상을 재생할 때 아래와 같은 조건에서 화면에 잔상이 표시될 수 있습니다.
    • 프로파일 변경 시 해상도가 달라지는 경우
    • 프로파일 변경 시 네트워크 지연 발생으로 데이터 수신이 늦어지는 경우
    • 웹 브라우저 창의 크기 변경 및 위치 이동 시

아이콘

라이브 화면 하단의 아이콘은 다음과 같은 기능을 합니다.

특정 브라우저 또는 코덱에서는 일부 기능이 동작하지 않을 수도 있습니다.

아이콘 기능 설명

비디오 설정

현재 라이브 화면에 적용된 프로파일을 확인하거나 프로파일을 변경할 수 있습니다. 또한 라이브 화면의 디스플레이 설정을 변경할 수 있습니다.

PT

 카메라의 팬/틸트 동작을 디지털로 제어할 수있습니다.

PTZ

카메라의 팬/틸트/줌 동작을 디지털로 제어할 수 있습니다. 현재 라이브 화면에 적용된 비디오 프로파일이 디지털 PTZ를 지원하는 경우에만 작동합니다.

디지털 PTZ 제어 프로파일로 변경하려면 [설정]>[Basic]>[비디오 프로파일]>[비디오 프로파일]>[프로파일 타입]에서 [디지털 PTZ 프로파일]을 선택합니다.

PTZ

카메라 RS-485나 RS-422 단자에 연결된 팬/틸트 리시버를 통해 카메라의 팬/틸트/줌 동작을 제어할 수 있습니다. [설정]>[PTZ]>[외부 PTZ] 페이지에서 설정한 프리셋 위치로 이동하거나, 스윙, 그룹, 투어, 추적 기능을 시작하고 중지할 수 있습니다.

PTZ

카메라의 팬/틸트/줌 동작을 제어할 수 있습니다. [설정]>[PTZ]>[PTZ 설정] 페이지에서 설정한 프리셋 위치로 이동하거나, 스윙, 그룹, 투어, 추적 기능을 시작하고 중지할 수 있습니다.

채널 선택에서 [PTZ] 채널을 선택한 경우에만 'PTZ' 아이콘이 보입니다.

상태

프로파일 별로 접속 정보와 현재 카메라에 접속한 사용자 접속 정보를 확인할 수 있습니다.
렌즈 모델명

카메라에 장착된 렌즈 모델 이름이 표시됩니다. 채널 선택에서 채널을 선택하면 채널별로 장착된 렌즈 이름을 확인할 수 있습니다.

채널 선택

카메라 채널 번호가 표시됩니다.

채널 번호를 선택하면 해당 채널의 영상이 뷰어에 나타납니다. 4분할 화면 아이콘을 선택하면 4개 채널의 영상이 한 번에 보여집니다. 단일 화면으로 돌아오려면 채널 번호를 클릭합니다.

고정형 카메라의 채널 번호를 선택하면 해당 영상이 뷰어에 나타납니다. [PTZ]를 선택하면 PTZ 단일 채널 영상이 뷰어에 나타납니다. 멀티 채널 아이콘을 선택하면 고정형 카메라와 PTZ 카메라이 영상이 한 번에 나타납니다. 단일 화면으로 돌아오려면 채널 번호나 [PTZ]를 클릭합니다.

카메라 채널 2~5번을 선택하면 각 채널별 카메라 영상을 보여줍니다. 채널 1을 선택하면 2~5번 채널의 카메라 영상을 하나로 연결하여 파노라마 형식으로 보여줍니다. 채널 6번과 7번을 선택하면 1번 채널 영상을 카메라 PTZ를 조정하면서 확인할 수 있습니다.

전체 화면

라이브 화면을 전체 화면으로 볼 수 있습니다. 원래 웹 브라우저의 크기로 돌아가려면 전체 화면 모드에서 아이콘을 다시 한번 클릭하거나 키보드의 [ESC] 키를 누릅니다.
크기 옵션

한 번씩 클릭할 때마다 다음 모드로 변경됩니다.

  • (화면 맞춤): 카메라 영상을 웹 브라우저의 크기에 맞추어 봅니다.
  • (원본 크기): 카메라 영상의 실제 해상도 크기로 봅니다.
  • (영상 비율 고정): 카메라 영상 해상도의 종횡비를 유지한 채 라이브 화면을 확대 또는 축소하여 웹 브라우저의 크기만큼 봅니다.

캡처

라이브 화면을 캡처하여 PNG 이미지 파일로 저장할 수 있습니다. 캡처한 이미지 파일은 각 브라우저의 기본 저장 경로에 저장됩니다.

저장

라이브 화면을 수동으로 녹화하여 PC에 저장할 수 있습니다. 저장 아이콘을 클릭하여 녹화를 시작하고, 다시 한번 더 클릭하면 녹화가 멈춥니다. 녹화 파일은 .avi의 형식으로 저장할 수 있고, 브라우저에 따라 브라우저 기본 저장 경로에 저장되거나 다른 이름 저장 창에서 파일 경로를 설정할 수 있습니다.

픽셀 카운트

라이브 화면에서 마우스로 선택한 영역의 영상 픽셀 수를 확인할 수 있습니다. 픽셀 카운트 아이콘을 클릭한 후 원하는 영역을 마우스로 클릭하고 드래그합니다. 선택한 영역이 표시되고 영상의 픽셀 수가 표시됩니다. 픽셀 카운트 아이콘을 한 번 더 클릭하면 픽셀 카운트 기능이 종료됩니다.

마이크

마이크 기능을 사용할 수 있습니다. 마이크 기능을 사용하려면 해당 프로파일에 [오디오 입력] 기능이 활성화되어 있어야 합니다. 비디오 입력 기능을 활성화하려면 [설정]>[Basic]>[비디오 프로파일]>[오디오 입력]에서 [사용함]을 선택합니다.

채널 1을 선택한 경우에만 마이크 기능을 사용할 수 있습니다.

알람 출력

원하는 알람 출력 번호를 클릭하면 이미 설정한 대로 알람을 출력합니다. 알람은 [설정]>[이벤트]>[알람 출력] 페이지에서 설정할 수 있습니다. 알람 출력 개수는 카메라마다 다릅니다.

스피커

라이브 화면의 오디오 음량을 조절할 수 있습니다. 아이콘을 클릭하여 오디오를 활성화한 후 음량 크기를 조절할 수 있습니다.

채널 1을 선택한 경우에만 스피커 기능을 사용할 수 있습니다.

채널을 변경하려면

  • 원하는 채널 번호를 선택합니다. 해당 채널의 영상이 뷰어에 나타납니다.
  • 4분할 화면을 보려면 4분할 화면 아이콘을 클릭합니다.
  • 단일 화면으로 돌아오려면 채널 번호를 클릭합니다.

영상을 캡처하려면

  • 캡처할 장면에서 캡처 아이콘()을 클릭합니다.
  • 캡처가 저장되면 알림 메시지가 나타납니다. 각 브라우저의 저장 경로에 캡처 이미지가 저장됩니다.

Windows 7 이상에서 Internet Explorer 브라우저로 캡쳐가 안 되는 경우, Internet Explorer 브라우저를 관리자 권한으로 실행하십시오.

영상을 녹화하려면

  • 저장 아이콘()을 클릭합니다.
  • 수동 녹화를 종료하려면 저장 아이콘()을 한 번 더 클릭합니다.

사용자 PC에 수동 녹화를 .avi 파일로 저장할 수 있습니다. 경로를 지정하고 영상을 저장하십시오.

피쉬아이 뷰 모드에서 녹화를 하면 사용자가 보고 있는 왜곡보정된 뷰가 아닌 원본 이미지로 저장됩니다.

전체 화면으로 전환하려면

  • 전체 화면 아이콘()을 선택합니다. 뷰어 화면이 전체 화면으로 변경됩니다.
  • 전체 화면 모드를 종료하려면 전체 화면 아이콘()을 한 번 더 클릭하거나 키보드의 [Esc] 키를 클릭합니다.

오디오를 사용하려면

  • 오디오 아이콘()을 클릭합니다.
  • 음량 조절 바를 이용하여 오디오 음량을 조절합니다.

오디오 동작 중 PC의 오디오 잭을 연결하거나 분리할 때 소리가 나오지 않는 경우, 오디오 아이콘()을 클릭하여 활성화하십시오.

스피커를 사용하려면

  • 스피커 아이콘()을 클릭합니다.

픽셀 수를 카운트하려면

  • 픽셀 카운트 아이콘()을 클릭합니다.
  • 영상 위에 마우스를 드래그하여 영역을 설정합니다. 해당 영역의 픽셀 수가 화면에 표시됩니다.

컬러 팔레트

카메라 영상을 시각적으로 분석하기 용이하도록 컬러 팔레트를 제공합니다. 원하는 컬러 팔레트를 선택하면 카메라 영상 화면에 선택한 컬러 팔레트가 적용되어 나타납니다.

각 컬러 팔레트 옵션별 특징은 [설정] > [비디오 및 오디오] > [카메라 설정] > [컬러 팔레트]를 참고하십시오.

비디오 설정

프로파일

현재 라이브 화면에 적용된 비디오 프로파일 이름과 상세 정보가 표시됩니다. [프로파일] 드롭다운 버튼을 누르면 현재 웹뷰어에서 사용할 수 있는 비디오 프로파일 목록이 나타나며 원하는 비디오 프로파일을 선택하면 라이브 화면에 바로 적용됩니다. 선택한 비디오 프로파일의 해상도, 코덱, 프레임 레이트, 목표 비트 레이트를 확인할 수 있습니다.

표시

라이브 화면의 대비, 밝기, 윤곽, 컬러 레벨을 설정할 수 있으며, 설정값을 입력하면 라이브 화면에 바로 적용됩니다. 버튼을 클릭하면 설정한 모든 값이 초기화됩니다.

PT

카메라의 팬/틸트 동작을 제어하거나 미리 설정해 둔 팬/틸트 프리셋 위치로 이동이 가능하며 그룹 동작을 제어할 수 있습니다.

프리셋

프리셋 목록에서 원하는 프리셋 번호를 선택한 후 [이동] 버튼을 클릭하면 카메라 뷰가 프리셋 위치로 이동합니다. 프리셋 위치를 설정하려면 PT 제어 화면에서 아이콘을 클릭하여 카메라 뷰 위치를 조정한 후 [설정] 버튼을 클릭합니다. [이름]과 [번호] 필드에 프리셋 이름과 번호를 각각 설정한 뒤 [적용] 버튼을 클릭합니다.

[홈] 버튼을 클릭하면 카메라 뷰가 홈 위치로 이동합니다. 홈 위치는 [설정]>[PTZ]>[PT 설정]>[프리셋] 탭에 설정할 수 있습니다.

그룹

그룹 목록에서 원하는 그룹 번호를 선택한 후 [시작] 버튼을 클릭하면 설정한 그룹 동작이 작동합니다. 그룹 동작을 중지하려면 [중지] 버튼을 클릭합니다. 그룹 목록은 [설정]>[PTZ]>[PT 설정]>[그룹] 탭에서 추가하거나 변경할 수 있습니다.

PTZ

카메라의 줌이나 초점을 조절할 수 있습니다.

줌 조절하기

줌 영역에서 아이콘을 클릭하면 화면이 확대되고 아이콘을 클릭하면 화면이 축소됩니다.

초점 조절하기

아이콘이나 아이콘을 클릭하여 카메라의 초점을 조절합니다.

카메라 초점을 자동으로 한 번 맞추려면 아이콘을 클릭합니다.

프리셋 이동하기

프리셋 목록에서 원하는 프리셋을 선택한 후 [이동] 버튼을 클릭하면 원하는 프리셋 위치로 이동합니다.

프리셋을 추가하려면 [설정] 버튼을 클릭한 후 프리셋 이름을 입력하고 번호를 설정한 뒤[적용] 버튼을 클릭합니다. 또한 [설정]>[PTZ]>[프리셋 설정] 탭에서 프리셋을 추가하거나 변경할 수 있습니다.

PTZ

카메라의 팬/틸트/줌 동작을 제어하거나 자동 추적 기능을 사용할 수 있습니다.

  • 줌 확대: UI 우측의 바를 위로 끌어올리거나 아이콘을 누르면 화면의 배율을 확대합니다. 바 위치가 중앙에서 멀수록 줌 확대 속도가 빨라집니다.
  • 줌 축소: UI 우측의 바를 아래로 끌어내리거나 아이콘을 누르면 화면의 배율을 축소합니다. 바 위치가 중앙에서 멀수록 줌 축소 속도가 빨라집니다.
  • 화면 이동: 커서가 있는 방향으로 이동합니다.

채널 선택에서 PTZ 채널을 선택한 경우에만 PTZ 기능이 활성화됩니다.

줌 동작 제어하기

줌 막대에서 아이콘을 클릭하면 화면이 확대되고 버튼을 클릭하면 화면이 축소됩니다.

팬/틸트 동작 제어하기

팬/틸트 영역에서 아이콘을 클릭하여 카메라의 방향을 이동합니다.

초점 조절하기

아이콘이나 아이콘을 클릭하여 카메라의 초점을 조절합니다.

카메라 초점을 자동으로 한 번 맞추려면 아이콘을 클릭합니다.

자동 추적

라이브 화면에서 움직이는 사물이나 사람을 자동으로 추적할 수 있습니다. 자동 추적 아이콘을 클릭하면 자동 추적 기능이 실행되며 아이콘이 로 변경됩니다. 자동 추적 아이콘을 한 번 더 클릭하면 추적 기능이 중지됩니다.

또한 영상 화면에서 마우스 오른쪽 버튼을 클릭하면 클릭한 영역을 중심으로 움직임을 분석한 후 자동 추적을 시작합니다.

PTZ 기능 사용하기

미리 설정한 위치로 카메라가 이동하도록 하거나, 스윙, 그룹, 투어, 추적 기능 시작하고 중지할 수 있습니다. 관련 설정은 [설정]>[PTZ]>[PTZ 설정] 페이지를 참고 하십시오.

영역 줌

영역 줌은 특정 영역만 확대하여 볼 수 있는 기능입니다.

아이콘을 클릭한 뒤 카메라 영상 화면에서 마우스를 드래그하여 영역을 선택하면, 선택한 영역을 기준으로 줌 기능이 작동하고 카메라 위치도 선택한 영역으로 이동합니다. 아이콘을 클릭하여 이 전 선택 영역과 설정으로 이동할 수 있으며, 이 전으로 이동한 경우 이동 전 영역과 설정으로 다시 이동하려면 아이콘을 클릭합니다. 아이콘을 클릭하면 줌 배율이 1배로 변경되고 카메라 위치도 원래 위치로 이동합니다.

영역 줌 설정을 종료하려면 아이콘을 한 번 더 클릭합니다. 현재 설정한 영역 줌은 그대로 유지되고 영역 줌 설정 기능은 종료됩니다.

분할 화면 선택

4분할 뷰에서 개별 영상의 번호를 선택합니다. 1은 4분할 뷰의 왼쪽 상단, 2는 오른쪽 상단, 3은 왼쪽 하단, 4는 오른쪽 하단의 영상 뷰입니다.

버튼을 클릭하고 드래그하여 카메라 영상 뷰를 조정합니다. 줌 막대에서 버튼을 클릭하면 화면이 확대되고 버튼을 클릭하면 화면이 축소됩니다.

[설정]>[Basic]>[비디오 프로파일]>[비디오 프로파일]>[피쉬아이 뷰 타입]에서 '왜곡 보정 뷰'를 선택한 후 [뷰]에서 '4분할 뷰'를 선택한 경우에만 '분할 화면 선택' 옵션이 나타납니다.

채널 선택

채널 선택

채널 번호 버튼을 선택하면 고정 카메라 영상을 확인할 수 있습니다. [PTZ] 버튼을 클릭하면 PTZ 카메라 영상을 단일 화면에서 확인할 수 있습니다. 분할 화면 버튼을 클릭하면, 왼쪽 화면에는 고정 카메라 영상이, 오른쪽 화면에서는 PTZ 영상이 동시에 나타납니다. 단일 화면으로 이동하려면 채널 번호 버튼이나 PTZ 버튼을 클릭합니다.

고정 카메라와 PTZ 카메라 연동

분할 화면 상태에서 고정 카메라 영상 화면에 마우스를 올려 놓으면 마우스 커서가 돋보기 모양으로 나타납니다. 고정 카메라 영상 화면에서 특정 지점을 클릭하면, 클릭한 지점이 오른쪽 PTZ 화면 중앙 + 표시에 나타납니다.

고정 카메라 영상 화면에서 마우스를 드래그하여 영역을 설정하면, PTZ 영상 화면에 해당 영역이 확대되어 나타납니다.

알아두기

  • '고정 카메라와 PTZ 카메라 연동'은 [설정]>[PTZ]>[캘리브레이션]에서 고정 카메라와 PTZ 카메라 영상 위치 좌표값을 설정한 경우에만 작동합니다.
