Configuration PT

Vous pouvez configurer la position préréglée pour les fonctions panoramique/inclinaison ainsi que les caractéristiques détaillées d’exécution groupée, automatique. Lorsque vous terminez le réglage, cliquez sur le bouton [Appliquer] ci-dessous.

Préréglage

Un préréglage signifie définir une position de vue de la caméra. Lorsque vous définissez un préréglage de la caméra, vous pouvez facilement accéder à cette position. Au plus 300 préréglages peuvent être définis.

Configuration d’un préréglage

  1. Cliquez et faites glisser l’icône sur l’écran de contrôle PT pour ajuster la position de la vue caméra.
  2. Cliquez sur le bouton [Configuration de préréglage].
  3. Dans la fenêtre de dialogue [Configuration de préréglage] sélectionnez le numéro préréglé souhaité dans [Numéro] et entrez le nom du préréglage dans le champ [Nom].
  4. Cliquez sur le bouton [OK].

Le préréglage ajouté sera affiché dans la liste [Configuration préréglée]. Vous pouvez configurer les détails du préréglage dans la liste des préréglages.

Se déplacer en position initiale

  • Cliquez sur le bouton [Aller en position initiale].

Configuration de la position initiale

  1. Cliquez et glisser l’icône sur l’écran de contrôle PT pour ajuster la direction de la vue caméra sur la position initiale souhaitée.
  2. Cliquez sur le bouton [Configuration de position initiale].

Remarque

  • L’écran de la caméra sera situé en position initiale après le redémarrage de la caméra si la position initiale est configurée.

Configuration préréglée

Vous pouvez afficher la liste des préréglages et modifier ou supprimer les paramètres détaillés de chaque préréglage.

Numéro

C'est le numéro de préréglage saisi dans la boîte de dialogue [Préréglage].

Nom

C'est le nom de préréglage saisi dans la boîte de dialogue [Configuration préréglée].

Mesures de suivi

Définissez l'action de la caméra à exécuter après que la caméra se soit déplacée vers la position prédéfinie correspondante.

  • Désactiver : Aucune action ne sera exécutée après que la caméra se soit déplacée vers la position prédéfinie correspondante.
  • Exécution automatique : Une action définie à partir de l'onglet [Exécution automatique] sera exécutée automatiquement. Lorsque [Désactiver] est sélectionné dans le [Mode] Exécuter automatiquement ou qu'il n'y a pas d'action de démarrage automatique, l'état d'exécution prédéfini sera conservé.
  • Analyses vidéo : L'action d'analyse vidéo sera exécutée.
Aller au menu

Vous pouvez passer à la page de configuration et vérifier ou définir une option détaillée.

  • Configuration de la caméra : Allez à la page [Vidéo & Audio] > [Réglage caméra]. Vous pouvez modifier le réglage de la caméra pour l'adapter à l'environnement du préréglage sélectionné.  
  • Détection de mouvement : Allez à la page [Analyse] > [Détection de mouvement]. Lorsque [Préréglage] est sélectionné parmi les options [Type] de la page de détection de mouvement de l'option, vous pouvez définir un événement de détection de mouvement différent pour chaque préréglage.
  • IVA: Allez à la page [Analyse] > [IVA]. Lorsque [Préréglage] est sélectionné parmi les options [Type] de la page IVA, vous pouvez définir un événement IVA (analyse vidéo intelligente) différent pour chaque préréglage.

Accéder aux préréglages

  1. Cliquez sur votre préréglage souhaité dans la [Numéro] ou [Nom]. La vue de la caméra est alors déplacée sur la position de préréglage désignée.

Effacer un préréglage

  1. Sélectionnez le préréglage que vous souhaitez supprimer dans la case à cocher en regard de la colonne [Numéro].
  2. Lorsque vous cliquez sur le bouton [Supprimer], le préréglage sera supprimé.

Groupe

La fonction de groupe est une fonction utilisée pour grouper un certain nombre de préréglages précédemment désignés et accéder séquentiellement aux préréglages placés dans un groupe.

Groupe de paramètres 1

  1. Pour définir le groupe 1, sélectionnez « 1 » dans le menu déroulant du groupe.
  2. Sélectionnez le numéro prédéfini à ajouter au « Groupe 1 » dans la colonne [Préréglage n°].
  3. Saisissez la vitesse de la caméra pour passer de la position préréglée en cours à la position préréglée suivante dans la colonne [Vitesse].
  4. Définissez la durée restante pour décider combien de temps la caméra reste en position préréglée définie à partir de la colonne [Temps de passage] jusqu'à la position prédéfinie suivante.
  5. Cliquez sur le bouton [Appliquer].

Suppression de la valeur de réglage de groupe

  1. Sélectionnez un numéro de groupe pour effacer la valeur précédemment définie dans le menu déroulant du groupe.
  2. Cliquez sur le bouton [Supprimer].

Démarrer l'action de groupe

  1. Sélectionnez un numéro de groupe à exécuter dans le menu déroulant du groupe.
  2. Cliquez sur le bouton [Démarrer].

Arrêter l'action de groupe

  1. Pour arrêter le groupe en cours d'exécution, cliquez sur le bouton [Arrêter].

Démarrer automatique

L'exécution automatique est une fonction permettant d'exécuter automatiquement une action spécifiée à l'avance si l'utilisateur n'utilise pas la caméra pendant un certain temps.  

Mode

  • Désactiver : La fonction d'exécution automatique n'est pas utilisée.
  • Accueil : La caméra se déplace vers sa position d'origine. Vous pouvez définir la position d'origine depuis l’onglet [Préréglage].
  • Préréglage : Se déplace vers le numéro de préréglage approprié. Vous pouvez régler l'heure à laquelle la caméra se déplace vers le préréglage et le numéro de position préréglée où la caméra se déplace.
  • Groupe : Exécute l'action de groupe automatiquement. Vous pouvez définir l'heure d'exécution automatique d'une action de groupe et le numéro de groupe. Vous pouvez définir l'action de groupe détaillée depuis l’onglet [Groupe].
  • Panoramique automatique : Exécute l'action de pivoter à 360 degrés dans la direction panoramique automatiquement. Vous pouvez définir l'heure d'exécution automatique de l'action, la vitesse de rotation de la caméra dans la direction de panoramique et l'angle fixe de l'inclinaison. Par exemple, si « 30 » est réglé pour la plage d'inclinaison, la plage d'inclinaison sera maintenue à 30 degrés.
  • Calendrier : Vous pouvez définir un réglage pour que l'action fonctionne automatiquement au jour et à l’heure définis.

Temps

Détermine le temps d’attente si la caméra est inactive avant d’amener la caméra automatiquement sur sa position initiale.

Cette option « Temps » est activée en sélectionnant [Home] dans le [Mode].
