Configuration

La gestion des files d'attente est une fonction utilisée pour surveiller le nombre moyen de personnes séjournant dans une zone spécifiée et la durée totale du séjour grâce à une analyse des images de la caméra. Définissez la règle de file d'attente que vous souhaitez analyser et les éléments détaillés pour améliorer la précision de l'analyse dans la page « Configuration » de la gestion des files d'attente. Lorsque vous avez complété le réglage, cliquez sur le bouton [Appliquer] en bas de la page.

Configuration

Activer la gestion des files d'attente

Pour utiliser la fonction de gestion de la file d'attente, sélectionnez [Activer la gestion de la file d'attente].

Règle

Définissez la règle de file d'attente pour laquelle vous souhaitez analyser le nombre de files d'attente ou l'heure de mise en file d'attente. Jusqu'à 3 règles de file d'attente peuvent être définies.

Définir la règle de gestion des files

Pour définir la règle de gestion de la file d'attente, cochez la case en regard de la colonne [N°]. Lorsque la règle de gestion de la file d'attente est affichée sur l'écran, ajustez la taille et la forme de la zone de gestion de la file d'attente en faisant glisser le coin de la zone avec la souris. Saisissez le nom de la zone dans le champ [Nom]. Vous pouvez saisir jusqu'à 15 caractères, y compris les caractères alphabétiques et les chiffres, pour le nom de la règle. Le nom de la règle saisi est affiché dans [Statistiques]>[Gestion des files d'attente]>[Rechercher].

Activer et désactiver l'analyse de gestion de la file d'attente pour chaque règle

Pour activer l'analyse de gestion de la file d'attente, cochez la case devant la colonne [N°] correspondante.

Pour désactiver l'analyse de la gestion des files d'attente, désélectionnez la case devant la colonne [N°] correspondante. La configuration d'analyse de gestion de la file d'attente pour chaque règle reste intacte, ce qui signifie que si vous cochez à nouveau la case, la fonction d'analyse de gestion de la file d'attente sera activée dans la même configuration.

Evénements de file d'attente

Définissez les conditions pour créer un événement de file d'attente.

Un événement de file d'attente différent peut être défini pour chaque file d'attente définie dans [Règle].

Niveau de détection

Le niveau de détection (niveau de file d’attente) affiche le nombre de personnes restant dans la règle définie. Une alarme est générée si le niveau actuel est supérieur au niveau de détection cible.

Le niveau de détection est classé en « Élevé » et « Moyen », en fonction du nombre de personnes dans une règle de file d'attente donnée. Lorsque [Maximum] et [Élevé] sont définis pour le nombre de personnes, la valeur « Moyen » est automatiquement définie.

« Élevé » pour le niveau de détection fait référence à l'état dans lequel le nombre de personnes dépassant la valeur « Élevé » et inférieure à la valeur « Maximum » est détecté. « Moyen » pour le niveau de détection fait référence à l'état dans lequel le nombre de personnes dépassant la valeur « Moyen » et inférieure à la valeur « Élevé » est détecté.

Par exemple, si 20 est défini pour la valeur [Maximum] et 18 est défini pour la valeur [Élevé], 9 sera automatiquement défini pour la valeur « Moyen ». Si le nombre de personnes faisant la queue dans la zone de file d'attente concernée est supérieur à 18 et inférieur à 20, le niveau de détection sera « Élevé ». Si le nombre de personnes faisant la queue dans la zone de file d'attente concernée est supérieur à 9 et inférieur à 18, le niveau de détection sera « Moyen ».

Durée minimale

Le temps minimum de mise en file d'attente pour créer un événement de file d'attente par le niveau de détection peut être défini. Un événement de file d'attente est créé lorsque le nombre de personnes correspondant [Niveau de détection] « Élevé » aux files d'attente est supérieur à l'heure saisie dans [Élevé] de [Durée minimum].  

En d'autres termes, lorsque « 18 » est défini pour [Niveau de détection] « Élevé » et « 60 secondes » de [Durée minimum] sur « Élevé », un événement de file d'attente est créé si 18 personnes ou plus font la queue dans la zone de file d'attente pendant 60 secondes ou plus.

Rapport

Activer

Pour recevoir périodiquement des données de file d'attente par e-mail et FTP, sélectionnez [Activer].

Les réglages détaillés du FTP et de l’e-mail peuvent être réalisés depuis [Événement]>[FTP/E-mail].

Calendrier

Pour recevoir les données de file d'attente à la même heure tous les jours, sélectionnez [Quotidien] et choisissez une heure.

Pour recevoir des données de file d'attente chaque semaine au même jour et heure, sélectionnez [Hebdomadaire] et sélectionnez un jour de la semaine et une heure.

Nom de fichier

Saisissez le nom de fichier pour l'envoi de données de file d'attente via un serveur FTP ou un e-mail et sélectionnez le format de fichier. Vous pouvez saisir jusqu'à 64 caractères, y compris des caractères alphabétiques et des chiffres, pour le nom de fichier.

Action de l'événement

FTP

Pour enregistrer une image sur un serveur FTP lorsqu'un événement de gestion de file d'attente est créé, sélectionnez [Activer]. Les réglages détaillés pour le FTP peuvent être effectués depuis [Événement]>[FTP/E-mail].

E-mail

Pour envoyer une image par e-mail lorsqu'un événement de gestion de file d'attente est créé, sélectionnez [Activer]. Les réglages détaillés pour l'e-mail peuvent être effectués depuis [Événement]>[FTP/E-mail].

Émission d'alerte

Il est possible de définir le numéro d’émission d'alerte pour une alarme spécifique lorsqu'un événement de gestion de file d'attente est créé. Après avoir sélectionné un numéro d’émission d'alerte, sélectionnez [Désactiver] si vous ne souhaitez pas utiliser l'alarme. Si vous souhaitez utiliser l'alarme, sélectionnez l'heure d’émission. Vous pouvez régler l'alarme pour qu'elle sonne continuellement jusqu'à ce qu'elle soit désactivée par l'utilisateur en sélectionnant [Toujours] ou régler l'alarme sur 5, 10 ou 15 secondes. Les réglages détaillés pour l’émission d'alerte peuvent être effectués depuis [Événement]>[Émission d'alerte].

Aller à préréglage

Vous pouvez définir la position de vue de la caméra pour passer à un numéro prédéfini lorsqu'un événement de gestion de file d'attente est créé. Le préréglage de la caméra peut être défini dans [PTZ]>[PTZ numérique] ou [Préréglage] dans [PTZ externe].

Aller à préréglage

Vous pouvez définir la position de vue de la caméra pour passer à une position prédéfinie lorsqu'un événement de gestion de file d'attente est créé. Le préréglage de la caméra peut être défini dans la section [Configuration préréglée] dans [PTZ]>[Configuration préréglée].

Aller à préréglage

Vous pouvez définir la position de la caméra pour passer à une position prédéfinie lorsqu'un événement de gestion de file d'attente est créé. Le préréglage de la caméra peut être défini dans la partie [Préréglage] dans [PTZ]>[Configuration PTZ].

Supprimer toutes les données

Pour supprimer toutes les données d'analyse de file d'attente, cliquez sur [Supprimer toutes les données].

Remarque

  • Dans les cas suivants, il est recommandé de réinitialiser les règles pour qu'elles correspondent à l'écran vidéo modifié et cliquer sur [Supprimer toutes les données] pour supprimer les anciennes données utilisées pour la recherche dans la gestion des files d'attente.
    • Si vous avez modifié les paramètres dans [Vidéo et audio] > [Configuration vidéo] > [Rotation vidéo]
    • Si vous modifiez les paramètres dans [Vidéo et audio] > [Réglage caméra] > [Spécial] > [DIS]

Remarque

  • La fonction Gestion des files d’attente vise à fournir des données statistiques, et non des données en temps réel, et peut inclure des objets en mouvement ou des animaux dans le nombre total d’unités mesurées.
  • Après l’installation de la caméra, il se peut qu’elle continue à ne pas fonctionner correctement jusqu’à ce qu’elle soit stabilisée. Le temps de stabilisation peut prendre de 50 à 60 minutes.
  • Si vous utilisez la fonction Gestion des files d’attente dans un environnement avec un temps d’attente court, elle peut générer un grand nombre d’événements.
  • Une caméra pour effectuer la gestion des files d’attente nécessite que l’objectif soit installé au sol en prise directe. (Figure 1)
  • Dans les cas suivants, les erreurs dans les données mesurées sont susceptibles d’augmenter :
    • Si elle est installée dans un espace dont le champ de vision est insuffisant
    • Heures de nuit ou faible luminosité (Figure 2)
    • Dans un environnement dont la lumière change soudainement, ou en cas d’exposition à une lumière forte, comme la lumière directe du soleil, le coucher ou le lever du soleil, l’éclairage direct, etc (Figure 3)
    • En cas de mouvements incessants à l’emplacement fixe, par exemple à proximité de la porte (Figure 4)
    • Si plusieurs personnes se déplacent dans un groupe
    • Si un objet en mouvement est proche de la règle définie,
    • Si la règle est définie sur le bord extérieur de l’image pour une caméra Fisheye (Figure 5)
