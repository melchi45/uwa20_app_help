Canal vidéo

Vous pouvez ajouter un canal fixe en tant que type spécifique parmi une variété de types d’affichage Fisheye.

Cliquez sur le bouton [Appliquer] ci-dessous lorsque la configuration est terminée.

Canal vidéo

L’ajout d’un canal vidéo vous permet de sélectionner et d’afficher un type de vue spécifique sur l’écran live (direct). Il existe toute une variété de types d’affichage Fisheye. Si vous ajoutez un certain type de vue dans [Canal vidéo], vous pouvez sélectionner et afficher uniquement le profil du type de vue sélectionné sur la page [Live].

Ajout d’un canal vidéo

  1. Cliquez sur le bouton [Ajouter].
  2. Sélectionnez le type de vue souhaité dans le menu déroulant de la colonne [Type d’affichage Fisheye].

Si vous cliquez sur le bouton [Supprimer tout], tous les types de vue du canal vidéo précédemment définis seront supprimés.

Remarque

  • Jusqu’à 2 canaux vidéo peuvent être définis.
  • Lors de la configuration du type d’affichage du canal vidéo, [Panorama simple] et [Panorama double] ne peuvent pas être ajoutés l’un à l’autre. De plus, [Quadravision] ne peut être ajouté à aucun des types suivants : [Q1], [Q2], [Q3], et [Q4].
  • Même si vous ajoutez un type d’affichage Fisheye dans [Canal vidéo], si aucun profil ne prend en charge le type de vue dans [Profil vidéo], la lecture ne s’effectuera pas sur la page [Live].
  • Si les paramètres du canal vidéo sont modifiés, la caméra redémarre.
