Configuración de cámara

Puede cambiar los ajustes de la cámara para capturar la mejor imagen posible en el ambiente donde esté instalada la cámara. Los predefinidos de imagen adecuados para diversos entornos se proporcionan de forma predeterminada; el usuario también puede especificar la configuración de la cámara directamente. Se puede comprobar en la pantalla de vista previa de la cámara cómo se muestra la imagen de la cámara de acuerdo con la configuración. Cuando haya completado el ajuste, haga clic en el botón [Aplicar] en la parte de abajo de la página. Si el tiempo de tiempo de espera (240 segundos) ha sido sobrepasado sin hacer clic en el botón [Aplicar] después de cambiar el ajuste, se restaurará la configuración anterior.

Tipo

Para definir la misma detección de movimiento para todos los preajustes, seleccione [Global]. Para definir la misma detección de movimiento para todos los preajustes de PTZ, seleccione [Preajuste]. La opción [Preajuste] estará activada solo cuando un preajuste se defina desde la página Configuración de preajustes.

Modo preestablecido de imagen

Varios predefinidos de imagen se proporcionan para los propósitos específicos. Seleccione un predefinido de imagen de acuerdo con el entorno en el que se utiliza la cámara.

  • Enfoque de definición: Este es el modo por defecto cuando la cámara se envía desde la fábrica.
  • Movimiento: Esto se utiliza principalmente para la captura definida de una persona en movimiento. El movimiento se selecciona cuando hay muchos objetos en movimiento, como por una carretera o en un museo. El ruido puede aumentar la sensibilidad y puede disminuir en un ambiente con poca luz.
  • Ruido reducido: Esto se utiliza principalmente para la captura de una imagen con menos ruido, y también en un ambiente con poca luz.
  • Vídeo brillante: Esto se utiliza para la captura de una imagen brillante de toda la escena con el fin de hacer una zona oscura claramente visible. Esto puede reducir el contraste de la imagen.
  • Movimiento y Ruido reducido: Esto se usa cuando hay muchos objetos en movimiento y una imagen con menos ruido es necesaria en un entorno con poca luz.
  • Movimiento y Vídeo brillante: Se utiliza cuando hay muchos objetos en movimiento y cuando una zona oscura se deberían mostrar más brillante dentro de toda la imagen.
  • Vídeo intenso: Se utiliza para la visualización de toda la imagen brillante y vivo. Esto puede disminuir el contraste de la imagen y la gama de colores.
  • Usuario predeterminado: Se utiliza para la visualización de la imagen como establecida por el usuario.

Sensor

Establecer el número de fotogramas por segundo que el sensor CMOS de la cámara capturará (el valor máximo de [Básico]>[Perfil de vídeo]>[Velocidad de cuadro] de acuerdo con el número de fotogramas establecidos desde el sensor).

Modo Fotograma

Establezca una velocidad de fotograma. Una velocidad de fotogramas diferente, no se puede establecer para cada predefinido de imagen. La velocidad de fotogramas se aplica a todas las memorias de imagen por igual.

SSDR

SSDR aumenta el brillo de una zona oscura para revelar una imagen en un entorno donde hay una diferencia significativa entre una zona oscura y una zona brillante. El valor del nivel se ajusta automáticamente para cada predefinido de imagen proporcionada en la cámara y no se puede cambiar. Si no se utiliza la función o si desea cambiar el valor de nivel, utiliza el usuario predeterminado.

SSDR

Para utilizar SSDR, seleccione [Activar].

Nivel

Ajuste el nivel de rango dinámico. A medida que el nivel va más alto, una zona oscura se vuelve más brillante.

Rango-D

Seleccione el área de amplitud del rango dinámico.

Balance de blancos

Puede hacer correcciones para mostrar el color blanco y otros colores correctamente, normalmente, en cualquier entorno de iluminación. Esto se puede ajustar independientemente del ajuste preestablecido de imagen.

Modo

Seleccione el modo de balance de blancos de acuerdo con el entorno en el que se está utilizando la cámara.

  • Manual: Se puede ajustar la ganancia de rojo y azul manualmente. Si se muestra más de color rojo, baje la ganancia de rojo, y si se muestra más color azul, baje la ganancia de azul. Si se muestra un color más verde, aumente tanto los valores de rojo como azul.
  • ATW: Corrige el color de la cámara automáticamente.
  • Exterior: Los colores de la cámara se corrigen para ser optimizados para un entorno al aire libre.
  • Interior: La corrección se realiza para optimizar los colores para un ambiente interior.
  • AWC: La corrección de pantalla está hecha para optimizar los colores para la iluminación del entorno actual. Cuando dirija la cámara hacia el papel blanco y pulse el botón [Configuración], la ganancia de rojo y azul serán ajustada y aplicada a la imagen. Si el entorno cambia, esto se debe ajustar de nuevo.
  • MERCURIO: La corrección de pantalla está hecha para optimizar los colores para un entorno de iluminación con lámparas de mercurio.
  • SODIO: La corrección de pantalla está hecha para optimizar los colores para un entorno de iluminación con lámparas de sodio.

Ganancia de rojo

Ajustar la ganancia de rojo. Si la ganancia de rojo es alta, el color más rojo se muestra en la pantalla. Si se muestra demasiado rojo, baje la ganancia de rojo.

Ganancia de azul

Ajustar la ganancia de azul. Si la ganancia es alta azul, un color más azul se muestra en la pantalla. Si se muestra demasiado azul, baje la ganancia de azul.

Nota

  • En cualquiera de los casos siguientes, el balance de blancos podría no funcionar normalmente. En tal caso, ajuste el balance de blancos a través del modo [AWC]:
    • Si el ambiente circundante excede el rango de corrección de temperatura de color, como un cielo despejado o la puesta del sol
    • Si el entorno que lo rodea es oscuro
    • Si la cámara se enfrenta directamente hacia una luz fluorescente o si hay un cambio significativo de la iluminación.

Luz trasera

Una imagen capturada en contra de la luz puede ser corregida. Seleccione entre Off, BLC, HLC, o WDR de [Modo] y cambie la configuración según el modo de luz de fondo seleccionado.

BLC

BLC (Compensación de la luz) selecciona el modo y corrige un área específica de una imagen oscura capturada frente a la luz, para mostrar un objeto en un área seleccionada con mayor claridad. Ajuste el tamaño de la zona mediante el control de los botones arriba, abajo, izquierda y derecha.

Nivel de BLC

Se puede seleccionar un nivel bajo, medio o alto. Cuanto más alto sea el nivel, más brillante es el área de configuración.

Configuración de ajuste

  • Arriba: Ajuste la distancia de la superficie superior a la parte superior de la pantalla.
  • Abajo: Ajuste la distancia de la superficie inferior a la parte inferior de la pantalla.
  • Izquierda: Ajuste la distancia de la superficie izquierda a la parte izquierda de la pantalla.
  • Derecha: Ajuste la distancia de la superficie derecha a la parte derecha de la pantalla.

HLC

El modo HLC (Compensación de luz alta) bloquea iluminación potente, como una farola o los faros de un vehículo, para evitar que un objeto sea irreconocible.

Nivel HLC

Se puede seleccionar un nivel bajo, medio o alto. Cuanto mayor sea el nivel, mejor es la corrección.

Máscara

Seleccione si desea o no aplicar la máscara. Seleccione [Todo el día] si desea aplicar la máscara durante todo el día, o seleccione [Solo noches] si desea aplicar la máscara sólo por la noche.

Color de máscara

Ajuste el color de la máscara.

Tono de máscara

Configurar el tono de máscara.

Atenuación

El brillo de la imagen se ajusta automáticamente de acuerdo al entorno de alrededor.

Configuración de ajuste

  • Arriba: Ajuste la distancia de la superficie superior a la parte superior de la pantalla.
  • Abajo: Ajuste la distancia de la superficie inferior a la parte inferior de la pantalla.
  • Izquierda: Ajuste la distancia de la superficie izquierda a la parte izquierda de la pantalla.
  • Derecha: Ajuste la distancia de la superficie derecha a la parte derecha de la pantalla.

WDR

El modo WDR (Wide Dynamic Range) muestra tanto una zona oscura como una zona brillante claramente en un entorno de luz de fondo. Este método captura una vez con una obturación rápida para mostrar un área brillante con claridad y de nuevo con una obturación lenta para mostrar una zona oscura con claridad, utilizando los dos obturadores de la cámara, y luego combina únicamente las partes que se muestran claramente a partir de las dos imágenes, en una sola imagen. Si se utiliza el modo WDR, se puede producir ruido entre la zona clara y la zona oscura.

Nivel de WDR

Se puede seleccionar un nivel bajo, medio o alto.

Transición sin problemas

Si se selecciona [Activar], se analiza la imagen y el modo WDR se activa automáticamente cuando es necesaria la corrección de retroiluminación.

Apagado en luz baja

Si se selecciona [Activar], el modo WDR se apaga automáticamente en un ambiente con poca luz.

IR desactivado

Si se selecciona [Activar], el modo WDR se apaga automáticamente cuando el modo B/N está en funcionamiento.

Nota

  • Cuando se selecciona el modo WDR, el valor del obturador se restablecerá. La pantalla se pone brillante y luego se vuelve oscura.
  • Cuando la lente Iris P se utiliza de forma manual o se utiliza el obturador antiparpadeo, el rendimiento de modo WDR se ve limitado.
  • Cuando se utiliza el modo WDR, la velocidad de fotogramas se reduce a la mitad.

Exposición

Cambie la exposición de acuerdo con el entorno de la cámara. Si el fondo es más oscuro que el sujeto, el sujeto se mostrará normalmente sólo cuando se disminuya la exposición. De manera complementaria, si el fondo es más brillante que el sujeto, el sujeto se mostrará normalmente sólo cuando se aumenta la exposición. Hay un valor predeterminado establecido por separado para cada predefinido de imagen; no se puede cambiar. Sin embargo, el brillo se puede cambiar por cualquier preestablecido imagen.

Brillo

El brillo de la pantalla se puede ajustar. Cuanto mayor sea el número, más brillante será la pantalla.

Obturador mínimo

El obturador puede ajustar la exposición del sensor automáticamente de acuerdo con el entorno; el obturador electrónico opera en un rango de entre obturador mínimo y obturador máximo. 'Obturador mínimo' hace referencia al valor mínimo del rango posible de tiempo de exposición y establece un tiempo de exposición largo.

Obturador máximo

'Obturador máximo' hace referencia al valor máximo del rango posible de tiempo de exposición y establece un tiempo de exposición corto.

Prerir obturador

Designe el obturador preferido por el usuario para reducir la aparición de desenfoque de día en caso de que el obturador esté en baja velocidad. Ajuste el obturador preferido.

Anti parpadeo

Evita el parpadeo de la pantalla que se produce debido a la inconsistencia en la iluminación en el entorno de captura.

SSNR

Elimine el ruido de la imagen.

  • Act/Des: Establece si se utiliza o no la función SSNR. Cuando se selecciona [Act], el nivel de SSNR se puede ajustar.
  • Wise NR: Cuando un objeto en movimiento está en la imagen, el nivel de ruido reducido se ajusta automáticamente para mejorar la discriminación del objeto.

Nivel SSNR 2D

SSNR 2D reduce el ruido en la imagen con píxeles adyacentes en un fotograma. Establezca el nivel de SSNR 2D. El nivel se puede ajustar cuando se selecciona [Act] o [Wise NR] se selecciona para [SSNR].

Nivel SSNR 3D

SSNR 3D reduce el ruido en la imagen utilizando los píxeles de varios fotogramas. Establezca el nivel de SSNR 3D. El nivel se puede ajustar cuando se selecciona [Act] o [Wise NR] se selecciona para [SSNR].

Iris

Seleccione entre DC (automático), manual, e Iris P.

  • DC (Transición sin problemas): Ajusta el iris mediante la determinación de la cantidad de luz entrante.
  • Manual: Ajusta el iris manualmente.
  • Iris P: Ajusta el iris a través del motor de paso. Una gran profundidad de campo se puede asegurar de manera que una serie de objetos a varias distancias se puedan poner en foco al mismo tiempo.

Iris F-No

Se activa cuando el modo [Iris] es [Manual] y establece el número de iris f.

Lente

Seleccione entre DC (automático), manual, e Iris P disponibles.

  • DC (Transición sin problemas): Ajusta el iris mediante la determinación de la cantidad de luz entrante.
  • Manual: Ajusta el iris manualmente.
  • Iris P: Ajusta el iris a través del motor de paso. Una gran profundidad de campo se puede asegurar de manera que una serie de objetos a varias distancias se puedan poner en foco al mismo tiempo. Seleccione entre los tipos de iris disponibles.

Iris P

Se activa cuando el Iris P se selecciona desde [Lente]. Cuando se selecciona [Transición sin problemas], el iris se ajusta automáticamente en función del brillo. Cuando se selecciona [Manual], el usuario define la posición del iris directamente.

Posición de iris P

Se activa cuando [Manual] se ha seleccionado para [Iris P]. La posición de iris P se puede ajustar.

AGC

AGC (Control Automático de Ganancia) ajusta el brillo mediante el control de la sensibilidad de la ganancia de la imagen al capturar un objeto en la penumbra.

Día/Noche

Cambie la imagen a color o B/N según el ambiente de la cámara. Cuando el tiempo de permanencia se establece, la imagen se puede cambiar a color o B/N según la preferencia. Cuando se cambia a Día/Noche, un evento de detección de movimiento o de análisis de imágenes no podrá ser detectado.

Modo

Permite seleccionar el modo que cambia la cámara a color o b/n.

  • Color: Produce una imagen en color.
  • B/N: Produce una imagen en b/n.
  • Transición sin problemas: Cambie a modo de color durante el día y a b/n por la noche o en condiciones de poca luz. Si [Des] está ajustado para [AGC] de [Exposición], Día/Noche no se podrá ajustar de forma automática.
  • Externo: Cuando un terminal de entrada de alarma está relacionado con un equipo externo, color y b/n pueden ser controlados.
  • Programación: Día/Noche se cambia a la programación prevista de [Tiempo de activación(Color)].

Tiempo de permanencia

Cambia a color o b/n cuando [Transición sin problemas] se selecciona para Día/Noche y la condición de brillo se mantiene durante el tiempo de permanencia establecido.

Duración

Establece el intervalo de tiempo cuando se produce un cambio a color o b/n.

Entrada alarma

Establezca la imagen a color o b/n de acuerdo con el estado de apertura y cierre del sensor de alarma. Cuando [Externo] está ajustado para [Modo] de [Día/Noche], el [Evento]>[Entrada de alarma] se desactivará y no aparecerá la página de configuración.

Enfoque simple después de D\/N

Ajusta el enfoque de la imagen automáticamente tras un cambio de Día/Noche

Tiempo de activación (Color)

Establece la programación para el funcionamiento en modo de color. Cuando [Cada día] se selecciona y se ajusta la hora, el modo de color se activa en el momento pertinente todos los días, y el modo de b/n se activa en cualquier otro momento. Si desea que no se configure el modo de color para la misma hora todos los días, se puede anular la selección de [Cada día] y ajustar el tiempo de activación a lun, mar, mié, jue, vie, sáb o dom, respectivamente.

Especial

Puede ajustar la nitidez, contraste, color y nivel de filtro contra neblina de una imagen.

DIS

Cuando se detecta temblor en imagen, el DIS (Estabilización Digital de Imagen) lo corrige automáticamente y estabiliza la pantalla.

Nota

  • Si cambia la configuración de DIS, la pantalla de vídeo puede verse reducida o ampliada. Cambie la configuración de análisis y estadísticas para que pueda analizar el vídeo o acumular los datos estadísticos de acuerdo con el vídeo cambiado. Estos ajustes se pueden cambiar en el menú [Análisis] y en el menú [Estadísticas].

Nota

  • Si cambia la configuración de DIS, la pantalla de vídeo puede verse reducida o ampliada. Cambie la configuración de análisis en el menú [Análisis] para que el vídeo pueda ser analizado de acuerdo con el vídeo cambiado.

Nitidez

Ajuste la nitidez general de la imagen.

Nivel de nitidez

Ajuste la nitidez general de la imagen. Puede ajustar el nivel de nitidez mediante el establecimiento de [Act] para [Nitidez].

Gamma

Ajuste el contraste de la imagen. Esto significa que la distancia entre la parte más brillante y la parte más oscura de una imagen; un valor gamma más alto significa una pantalla más clara de la diferencia de brillo.

Contraste

Ajustar el contraste de sombra de la imagen.

Nivel de color

Ajuste la intensidad del color en la imagen.

Filtro contra neblina

Corrija la imagen cuando haga neblina o esté nublado. Cuando [Transición sin problemas] está establecido, se detecta el entorno que lo rodea y la imagen se corrige automáticamente. Cuando [Manual] está establecido, el usuario puede ver el entorno que lo rodea y fijar directamente el nivel de corrección. Configure [Manual] si desea ejecutar la función de filtro contra neblina si la cantidad de neblina cambia.

Nivel de filtro contra neblina

Se activa cuando [Manual] está configurado para Filtro contra neblina; se ajusta el nivel del filtro contra neblina. Un nivel más alto de filtro contra neblina muestra la imagen más clara. Si el nivel de neblina es bajo la y se establece un alto nivel de filtro contra neblina, la pantalla puede parecer más oscura.

LDC

LDC (Distorsión de lente de corrección) corrige la distorsión de imagen en el borde de una lente gran angular.

  • Des: La función de corrección de distorsión de la imagen no se utiliza.
  • Manual: Ajusta el nivel de corrección de la distorsión de forma manual.
  • Transición sin problemas: Ajusta la distorsión de la imagen automáticamente.

Nivel LDC

Ajuste el nivel de la función de LDC. Activado cuando [Manual] está configurado para [LDC].

OSD

Muestre el título de la cámara o la fecha y la hora, y establezca la ubicación, el tamaño, el color y la transparencia de las letras.

Título de cámara

Establece si se muestra o no el título de la cámara. Cuando [Activar] está seleccionado, los botones [Agregar]/[Eliminar] se activarán.

Al hacer clic en el botón [Agregar], un campo en el que un título de cámara se puede introducir será añadido a la lista. Introduzca el título de la cámara y establezca la ubicación moviendo las coordenadas X e Y. Hasta 5 títulos de la cámara se pueden agregar.

Para eliminar un título de cámara, seleccione el título que desee eliminar y haga clic en el botón [Eliminar]. Para una vista previa de cómo se muestra el título de la cámara en la pantalla después de entrar en él, introduzca el título de cámara y haga clic en el botón [Vista previa].

Fecha y hora

Para mostrar la fecha y la hora en la pantalla, seleccione [Activar]. Seleccione el tipo de visualización de la fecha y fije la posición para mostrar la fecha y la hora moviendo las coordenadas X e Y.

Tamaño

Ajuste el tamaño de la OSD.

Color

Ajuste el color de la OSD.

Transparencia

Establezca la transparencia de la OSD.

Superposición

Seleccione información como la posición PTZ o el nombre de preajuste para que se muestren en la pantalla de vídeo de la cámara.

Enfoque

Puede definir el método de funcionamiento de enfoque de la cámara, si usar o no el zoom digital, y reinicializar la cámara.

Modo

Defina el método de funcionamiento de enfoque de la cámara.

  • Manual: Ajuste el enfoque de la cámara manualmente.
  • Automático: El enfoque de la cámara se ajusta automáticamente.
  • EA de disparo: En caso de que haya un evento de usuario (Enfoque automático o Acción de zoom) el enfoque se ajustará automáticamente una vez.

Zoom digital

Para usar el zoom digital, seleccione [Activar].

Límite de zoom digital

Defina el rango de tamaño máximo de zoom digital disponible. Puede definir el rango de zoom digital definiendo [Uso] para [Zoom digital].

Modo de zoom

Defina el método para sincronizar el enfoque de la cámara cuando el zoom esté en funcionamiento.

  • Automático: Incluso si la magnificación del zoom ha cambiado, el zoom funciona mientras se mantenga el enfoque evaluando y modificando continuamente la condición del enfoque.
  • Manual: El zoom funciona de acuerdo con el trazo de enfoque definido. La imagen podría estar fuera de enfoque si la magnificación del zoom cambia.

Velocidad de zoom

Defina la velocidad de seguimiento de zoom.

Lente inicial

Puede reinicializar la lente manualmente una vez o definir el período de reinicialización para que la lente se reinicie automáticamente.

Para reinicializar la lente manualmente, seleccione [Manual] desde la opción [Lente inicial] y haga clic en el botón [Definir]. Para definir el período automático de reinicialización, seleccione el tiempo que desee entre 1 y 7 días. La lente se reinicializará en el período definido desde el tiempo definido.

Nota

  • Como la función de enfoque automático ajusta el enfoque en base a un asunto en el medio de la pantalla, los asuntos que estén en la periferia de la pantalla podrían quedarse fuera de enfoque.
  • La función de enfoque automático podría no funcionar de manera normal bajo las siguientes condiciones.
    • Si hay una luz brillante o parpadeante, como una señal de neón.
    • Si la iluminación del área incluida de monitoreo es baja
    • Si la iluminación del área incluida de monitoreo es excesivamente alta
    • Si la cámara funciona con un obturador lento
    • Si el asunto está oscuro
    • Si no hay diferencia de intensidad, como entre el cielo y una pared
    • Si la cámara está apuntando a un línea fina horizontal

Calentador

Acto único

Si las heladas siguen presentes y le gustaría descongelar haciendo funcionar el calentador, haga clic en una vez.

Ventilador

Acto único

Si desea operar el ventilador, haga clic una vez en el botón .

IR

Modo

Seleccione [Des] si desea no utilizar la función de IR. Select [Automático 1] para compensar la exposición automática por saturación del área central debido al LED IR o [Automático 2] para suprimir la saturación de toda la pantalla y compensar la exposición automática. Seleccione [Manual] si desea manejar el IR manualmente.

Nivel

Establezca el nivel de IR.

El nivel IR se activa cuando se selecciona [Manual] en [Modo].

Tiempo de activación

La cámara siempre funciona a una hora predeterminada específica. Configure y use un ajuste preestablecido de imagen deseado en el momento deseado. Si no se selecciona un predefinido de imagen específico, el predefinido de imagen seleccionado en la parte superior se activa por defecto. El color del tiempo de activación cambia junto con la definición de la imagen seleccionada.

Des

La cámara funciona a la hora seleccionada en el predefinido de imagen.

Sólo tiempo programado

Puede configurarlo para permitir a un usuario ejecutar un ajuste preestablecido de imagen deseado en un momento específico. Haga clic en el botón (Configuración), seleccione un preajuste de imagen, defina la hora de inicio y la hora de fin. Hasta 5 memorias se pueden ajustar dentro de un día. Ajuste el preajuste de imagen para cada hora de acuerdo con el uso de la cámara.
