Configuración del servidor de datos

Al registrar un servidor Retail Insight, los datos de la cámara (recuento de personas, gestión de colas y datos del mapa de calor) se transfieren al servidor registrado. En la página [Configuración del servidor de datos], es posible ingresar y registrar la información del servidor Retail Insight. Haga clic en el botón [Aplicar] a continuación después de ingresar la información necesaria.

Configuración del servidor de datos

Activar servidor de datos

Si desea transferir los datos de la cámara al servidor Retail Insight, seleccione [Activar servidor de datos].

Ingresar

Es posible ingresar información del servidor para registrar el servidor Retail Insight.

IP

Escriba la dirección IP del servidor Retail Insight.

Nota

  • Al lado del campo [IP], se muestra un mensaje que indica el estado de la conexión con el servidor.
    • Conectando: Aparece cuando se intenta conectar al servidor Retail Insight. Si se selecciona el botón [Aplicar] en la parte inferior de la página o [Activar servidor de datos], se intentará la conexión con el servidor.
    • Correcto: Aparece cuando hay una conexión al servidor Retail Insight.
    • Fallo: Aparece cuando no se puede conectar al servidor Retail Insight. En caso de que este mensaje persista, asegúrese de que la información del servidor Retail Insight de la página [Configuración del servidor de datos] sea correcta.

Puerto

Introduzca el valor de puerto del servidor Retail Insight.

El valor del puerto puede ser 80 o un valor entre 1024 y 65535.

ID

Ingrese el ID de la cuenta para iniciar sesión en el servidor de Retail Insight.

El ID puede contener hasta 8 letras, pero no acepta caracteres especiales: #%&+=\:<>"'

Contraseña

Escriba la contraseña de la cuenta para iniciar sesión en el servidor de Retail Insight.

La contraseña puede tener de 8 a 20 letras, pero no acepta caracteres especiales: #%&+=\:<>"’

Nombre de la tienda

Escriba el nombre de su tienda.

El servidor Retail Insight utiliza el nombre de la tienda para distinguir su cámara.
