검색

히트맵은 화면에서의 움직임 빈도를 색상으로 보여줍니다. 히트맵의 '검색'에서는 히트맵 데이터를 검색하고 결과를 확인할 수 있습니다.

알아두기

  • 히트맵의 검색 기능을 사용하려면, 먼저 [통계]>[히트맵]>[설정]에서 [히트맵 사용]을 선택하여 히트맵 기능을 활성화해야 합니다.
  • [통계]>[히트맵]>[설정]에서 [히트맵 사용]을 선택하지 않았다면, [설정] 버튼이 나타납니다. [설정] 버튼을 클릭하면 히트맵 설정 페이지로 이동합니다.

히트맵

화면에 현재 영상의 움직임 빈도에 따른 히트맵 정보가 나타납니다. 화면에 나타난 색상별 움직임의 빈도는 [인덱스]의 색상을 참고하십시오.

인덱스

인덱스 색상 막대는 움직임 빈도를 나타냅니다. 움직임 빈도가 낮을수록 파란색으로 표시되고 높을수록 빨간색으로 표시됩니다.

검색

[최근]을 선택하면 최근 하루, 일주, 한달 단위로 검색할 수 있습니다. [기간]을 선택하면 일간, 월간, 연간 단위로 기간을 지정하여 검색할 수 있습니다.

히트맵 검색 결과가 화면에 바로 나타납니다.

히트맵 검색하기 및 결과 보기
  1. [검색]의 [날짜]에서 검색할 날짜를 선택합니다.
  2. [검색] 버튼을 클릭합니다.
  3. [결과]에 검색 결과가 나타납니다.

다운로드

검색 실행 후 [다운로드] 버튼을 클릭하면 검색 결과를 이미지 파일로 다운로드할 수 있습니다. png 형식으로 다운로드할 수 있으며, 다운로드한 파일은 각 브라우저의 기본 저장 경로에 저장됩니다. 파일 이름은 영문자와 숫자를 포함하여 최대 64자까지 입력할 수 있습니다.

결과

히트맵 데이터를 검색한 경우, 결과가 나타납니다. 검색을 실행한 시점의 영상을 배경으로 검색한 기간의 히트맵 정보를 보여줍니다.
