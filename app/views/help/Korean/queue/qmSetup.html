설정

대기열 관리는 카메라 영상 분석을 통해 지정한 영역에 체류한 평균 인원수와 누적 체류 시간을 모니터링할 수 있는 기능입니다. 대기열 관리의 '설정' 페이지에서는 분석할 대기 영역을 설정하고 분석 정확도를 높이기 위해 세부 사항을 설정합니다. 설정을 완료하면 페이지 아래의 [적용] 버튼을 클릭하십시오.

설정

대기열 관리 사용

대기열 관리 기능을 사용하려면 [대기열 관리 사용]을 선택합니다.

영역

대기 인원이나 대기 시간을 분석할 대기열 영역을 설정합니다. 대기열 영역은 3개까지 설정할 수 있습니다.

대기열 관리 영역 설정하기

대기열 관리 영역을 설정하려면, [번호] 컬럼 앞의 체크박스를 선택합니다. 영상 화면에 대기열 관리 영역이 나타나면, 영역의 꼭지점을 마우스로 드래그하여 대기열 관리 영역의 크기와 모양을 조절합니다. [이름] 필드에 영역 이름을 입력합니다. 영역 이름은 영문자와 숫자만 15자까지 입력할 수 있습니다. 입력한 영역 이름은 [통계]>[대기열 관리]>[검색]에 나타납니다.

영역별 대기열 관리 분석 활성화 및 비활성화하기

대기열 관리 분석을 활성화하려면, 해당 [번호] 컬럼 앞에 있는 체크박스를 선택합니다.

대기열 관리 분석을 비활성화하려면, 해당 [번호] 컬럼 앞에 있는 체크박스를 선택 해제합니다. 영역별 대기열 관리 분석 설정은 그대로 남아 있으므로, 체크상자를 다시 선택하면 동일한 설정으로 대기열 분석 기능이 활성화됩니다.

대기열 이벤트

대기열 이벤트가 발생하기 위한 조건을 설정합니다.

[영역]영에서 설정한 대기열마다 각각 다르게 대기열 이벤트를 설정할 수 있습니다.

감지 레벨

감지 레벨은 대기열 영역에 체류 중인 인원에 따라 '높음'과 '중간'으로 분류됩니다. [최대]와 [높음] 인원수를 설정하면 '중간' 값은 자동으로 설정됩니다.

감지 레벨 '높음'은 '높음' 값 이상 '최대' 값 이하의 인원이 감지된 상태입니다. 감지 레벨 '중간'은 '중간' 값 이상 '높음' 값 미만의 인원이 감지된 상태입니다.

예를 들어[최대] 값을 20으로 [높음] 값을 18로 설정한 경우, '중간' 값은 9로 자동으로 설정됩니다. 해당 대기열 영역에 18명 이상 20이하의 인원이 대기 중이면 감지 레벨은 '높음'으로 분석됩니다. 9명 이상 18명 미만의 인원이 대기 중이면 감지 레벨은 '중간'으로 분석됩니다.

최소 관찰 시간

감지 레벨별로 대기열 이벤트를 발생하기 위한 최소 대기 시간을 설정할 수 있습니다. [감지 레벨]의 '높음'에 해당하는 인원이 [최소 관찰 시간]의 [높음]에 입력한 시간보다 더 많은 시간을 대기하면 대기열 이벤트가 발생합니다.  

즉 [감지 레벨]에서 '높음'을 '18'명으로 설정하고 [최소 관찰 시간]의 '높음'을 '90초'로 설정한 경우, 대기열 영역에 18명 이상의 인원이 90초 이상 동안 대기중이면 대기열 이벤트가 발생합니다.

캘리브레이션

대기열 관리 기능에서 대기 인원수 분석 정확도를 향상시키기 위해 카메라 높이나 카운트할 사람의 크기를 설정합니다. 카메라 높이나 사람의 크기 중 하나만 선택하여 적용할 수 있습니다.

카메라 높이

실제 카메라가 설치된 높이를 지정합니다. + 나 - 표시를 클릭하여 값을 변경하거나 카메라 높이를 수치로 입력할 수 있습니다. 입력할 수 있는 값은 최소 2.5m, 최대 4.5m입니다. 카메라 설치 높이를 설정하면, 설치 높이를 기반으로 위치별로 대기 인원으로 분석할 사람의 크기를 자동으로 추정합니다.

크기

대기 인원으로 분석할 사람의 크기를 직접 지정합니다. 영상 화면에 표시된 상자의 오른쪽 하단 모서리를 마우스로 드래그하여 크기를 조절할 수 있습니다.

리포트

사용

주기적으로 대기열 데이터를 이메일과 FTP로 수신하려면 [사용]을 선택합니다.

FTP와 이메일에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

스케줄

매일 동일한 시간에 대기열 데이터를 수신하려면 [일간]을 선택한 후 시간을 선택합니다.

매주 동일한 요일과 시간에 대기열 데이터를 수신하려면 [주간]을 선택한 후 요일과 시간을 선택합니다.

파일 이름

FTP 서버나 이메일로 대기열 데이터를 보낼 파일 이름을 입력하고 파일 형식을 선택합니다. 파일 이름은 영문자와 숫자를 포함하여 최대 64자까지 입력할 수 있습니다.

이벤트 동작

FTP

대기열 관리 이벤트가 발생했을 때 영상 이미지를 FTP 서버에 저장하려면 [사용]을 선택합니다. FTP에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

이메일

대기열 관리 이벤트가 발생했을 때 영상 이미지를 이메일로 전송하려면 [사용]을 선택합니다. 이메일에 대한 상세 설정은 [이벤트]>[FTP/이메일]에서 할 수 있습니다.

알람 출력

대기열 관리 이벤트가 발생했을 때 특정 알람 번호로 알람이 출력되도록 설정할 수 있습니다. 알람 출력 번호를 선택한 후, 알람을 출력하지 않으려면 [사용 안 함]을 선택하고 알람을 출력하려면 출력 시간을 선택합니다. [항상 동작]을 선택하여 사용자가 알람을 멈추기 전까지 계속 출력되도록 하거나 5, 10, 15초까지 출력하도록 설정할 수 있습니다. 알람 출력에 관한 상세 설정은 [이벤트]>[알람 출력]에서 할 수 있습니다.

프리셋 이동

자동 추적 이벤트가 발생했을 때 카메라 뷰 위치를 미리 설정해 둔 프리셋 번호로 이동하도록 설정할 수 있습니다. 카메라 프리셋은 [PTZ]>[디지털 PTZ]나 [외부 PTZ]의 [프리셋]에서 설정할 수 있습니다.

프리셋 이동

자동 추적 이벤트가 발생했을 때 카메라 위치를 미리 설정해 둔 위치로 이동하도록 설정할 수 있습니다. 카메라 위치 프리셋은 [PTZ]>[PTZ 설정]의 [프리셋] 탭에서 설정할 수 있습니다.

모든 데이터 삭제

대기열 분석 데이터를 모두 삭제하려면 [모든 데이터 삭제]를 클릭합니다.

알아두기

  • [비디오 및 오디오] > [비디오 설정] > [비디오 회전]에서 [상하 반전]이나 [좌우 반전] 설정을 변경한 경우, 대기열 관리에서 설정한 규칙이 모두 자동으로 초기화됩니다. 새 규칙에 맞춰 데이터를 다시 수집하려면 [모든 데이터 삭제]를 클릭하여 기존 데이터를 삭제하십시오.
