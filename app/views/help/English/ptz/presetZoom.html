Preset setup

You can set and delete a camera preset, select whether or not to save the current position as the last position of the camera, and set the saving interval.

Preset setup

Preset refers to the setting of the camera views. You can easily set a camera preset and move to the relevant view. 300 presets can be set.

Setting a preset

  1. Adjust the screen as follows to adjust the camera view.
    • Select image magnification by clicking the [+] button to zoom the screen in or clicking the [-] button to zoom the screen out from [Zoom].
    • Adjust the camera focus by clicking the button or button from [Focus].
    • To adjust the camera focus automatically for once, click the button from [Focus].
  2. Click the [Preset setup] button.
  3. Select a desired preset number from [Number] on the [Preset setup] dialog box.
  4. Enter the relevant preset name in [Name].
  5. Click the [OK] button.

Last Position

You can set the camera to move to the last position when turning the camera off and on. Select whether or not to save the last position of the camera and select the saving interval.

Last Position

To save the last position, select [Use].

Time

Set the saving interval to save the current position as the last position. The unit used to measure the interval is seconds. Enter the saving interval on the field and click [Set].

Preset setup

The set preset list will appear.

The number and the preset name entered on the [Preset setup] dialog box will appear on the [Number] and [Name] columns, respectively.

Moving to a preset

  1. Select a desired preset from the check box next to the [Number] column.
  2. When you click the [Move] button, the camera views will move to the set location.

Deleting a preset

  1. Select the preset you wish to delete from the check box next to the [Number] column.
  2. When you click the [Remove] button, the preset will be deleted.
