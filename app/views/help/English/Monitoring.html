Monitoring screen

You can check the monitoring screen to view what is being captured by the camera and control screen capture, alarm output and microphone and other features. When you click the button on the screen, the monitoring screen will appear.

You can move to the monitoring screen, playback screen or setup screen by clicking the buttons at the top.

  • (Monitoring): Check the monitoring screen to view what is being captured by the camera and control a variety of camera features.
  • (Playback): Search and playback a recorded image from an SD card or NAS.
  • (Setup): Change the web viewer settings.

Note

  • When you connect to a web viewer through Internet Explorer or Safari, a plugin is required before you can use the capture, record, pixel count or microphone functions. To install the plugin, click the button at the top right.

Monitoring icons

The icons at the bottom of the monitoring screen provide the following functions:

Icon Description of function

(PTZ)

 

You can control the Pan/Tilt/Zoom motions of the camera. PTZ operates only if the video profile applied to the current monitoring screen supports digital PTZ. To change to a digital PTZ control profile, select [DPTZ profile] from [Setup]>[Basic]>[Video profile]>[Video profile]>[Profile type].

 

(Video settings)

 

You can check or change the profile applied to the current monitoring screen. You can also change the display settings of the monitoring screen.

 

(Status)

 

Check the connection information for each profile and for connected users at the same time.

 

(Full screen)

 

View the monitoring screen in full screen. To return to the size of the previous web browser, click the button in full screen mode or press the [Esc] key on the keyboard.

 
Size option

The size changes to the next size each time you click it.

  • (Fit): Fit the camera image to the size of the web browser.
  • (Original Size): View the camera image at its actual resolution.
  • (Aspect Ratio): View the monitoring screen in the web browser by zooming in or out, while maintaining the aspect ratio of the camera image.

 

(Capture)

 

Capture and save the monitoring screen as a PNG image file. A captured image file is saved in the default recording path of each browser.

 

(Record)

You can record and save the monitoring screen to a PC. Click the Record button to start recording; click the button again to stop recording. Recorded files are saved in .avi format and can be saved in the default path of the browser or a file path can be set in the ‘Save as’ window.

 

(Pixel count)

 

You can check the number of image pixels in an area selected by the mouse on the monitoring screen. Click the Pixel count button and click and drag the desired area with the mouse. The selected area will be displayed and the number of pixels in the image will be displayed. When you click the Pixel count button again, the Pixel count function ends.

 

(Microphone)

 

Use the microphone function. To use the microphone function, the [Audio in] function should be activated for the relevant profile. To activate the video input function, select [Enable] from [Setup]>[Basic]>[Video profile]>[Audio in].

 

(Alarm output)

 

When you click a desired alarm output number, the alarm will be outputted as previously set. The alarm can be set from the [Setup]>[Event]>[Alarm output] page. The number of alarm outputs varies depending on the camera.

 

(Speaker)

 

Adjust the audio volume of the monitoring screen. Click the button to activate the audio and adjust the volume.

 

PTZ

You can control the pan, tilt, zoom in/zoom out motions of the camera or use the digital auto tracking function. When you click the button at the bottom of the main screen, the screen where Zoom and Pan & Tilt can be changed will appear at the bottom. To use PTZ, [DPTZ Profile] should be selected from [Setup]>[Basic]>[Video profile]>[Profile type]. DPTZ can be applied only to profiles that use H.264 and H.265 codecs.

Zooming in/out

When you click the [+] button in the zoom area, the screen will zoom in and when you click the [-] button, the screen will zoom out.

Controlling pan/tilt motions

Move the camera direction and adjust the focus by clicking the button in the Pan & Tilt area.

Moving a preset

When you select a desired preset from the preset list and click the [Playback] button, it will move to the desired preset location. The preset list can be added to or changed from the [Setup]>[PTZ]>[Preset] tab.

Operating a group motion

When you select a desired group from [Group] and click the [Playback] button, the preset group motion will operate. The group list can be added to or changed from the [Setup]>[PTZ]>[Group] tab.

Auto tracking

A moving object or person can be tracked automatically on the monitoring screen. When you click the (Auto tracking) button, the auto tracking function will be executed. When you click the (Auto tracking) button again, the tracking function will stop. If you operate the PTZ function while using the auto tracking function, the auto tracking function will stop.

Setup

Profile

The name and detailed information of the video profile applied to the current monitoring screen will be displayed. When you press the [Profile] drop-down button, a video profile list that can be used from the current web viewer will appear; when you select the desired video profile, it will be applied to the monitoring screen immediately. You can check the resolution, codec, frame rate and target bitrate of the selected video profile.

Display

Set the contrast, brightness, sharpness and color level of the monitoring screen; when you enter the setting, it will be applied to the monitoring screen immediately.

Status

Profile access

Check the connection status of all currently-set profiles or check the status of currently-connected users. Check the name of all set profiles, actual bitrate (kbps) and applied bitrate (kbps) for each profile, ATC for each profile (%) and the number of connected users for each profile from [Profile access].

Current users

You can check the applied profile for each user, bitrate (kbps), network connection status and IP address of all users currently connected to the camera.
