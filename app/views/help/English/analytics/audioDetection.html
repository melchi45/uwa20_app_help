Audio detection

An audio detection event can be created when audio above a set level is detected while the camera is capturing an image. When you complete the setting, click the [Apply] button at the bottom of the page. The setting on the audio detection event applies to [Audio detection] on the [Event]> and [Event setup] page equally.

Audio detection

Enable Audio detection

To apply the audio detection event, select [Enable Audio detection].

Level of detection

This sets the level standard for audio detection. An audio detection event is created when audio above the set level is detected.

When audio is detected, a graph will appear, and when an audio detection event is created, the graph color will change.

Note

  • Audio gain can be set from [Video & Audio]>[Audio setup]>[Gain].

Event Action Settings

FTP

To save an image on an FTP server when an audio detection event occurs, select [Enable]. Detailed settings for FTP can be done from [Event]>[FTP/E-mail].

E-mail

To send an image by e-mail when an audio detection event is created, select [Enable]. Detailed settings for e-mail can be done from [Event]>[FTP/E-mail].

Record

To save an image on an SD card or NAS when an audio detection event is created, select [Enable]. Detailed settings for storage can be done from [Event]>[Storage].

Alarm output

It is possible to set an alarm output number for a specific alarm when an audio detection event is created. After selecting an alarm output number, select [Off] if you wish no alarm, or select the output time if you wish to output an alarm. You can set a continuous alarm until turned off by the user by selecting [Always] or set the alarm for 5, 10 or 15 seconds. Detailed settings for alarm output can be done from [Event]>[Alarm output].

Goto preset

You can set the position of camera view to move to a preset number when an audio detection event is created. The camera preset can be set in [PTZ]>[Digital PTZ] or [Preset] of [External PTZ].

Goto preset

You can set the position of camera view to move to a preset position when an audio detection event is created. The camera preset can be set in [Preset setup] section of  [PTZ]>[Preset setup].

Goto preset

You can set the position of camera to move to a preset position when an audio detection event is created. The camera preset can be set in [Preset] tab of [PTZ]>[PTZ setup].

Event activation time

This sets the event action settings from [Event action settings] to operate always or only at a set time, when an audio detection event is created.

Always

The set event action settings are always executed when an audio detection event is created.

Only scheduled time

The set event action settings are executed when an audio detection event occurs during the set time. The date from Sunday to Saturday, based on the current camera time, is displayed on the time table shown when [Only scheduled time] is selected. You can set the time for event action settings by clicking or dragging on the time table with the mouse. The set event activation time is executed repeatedly on the relevant weekday and time.

You can change the time view unit by clicking the [1 min], [30 min] or [1 h] button. When you click the [Reset] button, all set event activation times will be deleted. To confirm or change the camera time, refer to [Basic]>[Date & Time].
