外部 PTZ

用户可通过与 RS-485 或 RS-422 终端相连的平移/倾斜接收器，对 PTZ 控制进行设置。您可以设置预设或为与摄像机相连的平移/倾斜接收器设置通讯模式。

预设

预设是指摄像机视图的设置。可以设置 128 个预设项。

设置预设

  1. Expand the camera image by clicking the button on the PTZ control screen, or adjust the camera views by clicking the button and dragging the mouse.
  2. Select a preset number from [Number].
  3. Enter the relevant preset name in [Name].
  4. Click the [Add] button.

Moving to a preset

  1. Select a preset number from [Number].
  2. When you click the [Go] button, the camera views will move to the set location.

Deleting a preset

  1. Select the preset number you wish to delete from [Number].
  2. When you click the [Remove] button, the preset will be deleted.

Serial port setup

You can set the connection value of an external pan/tilt receiver. When you complete the setting, click the [Apply] button at the bottom of the page.

Protocol

You can choose the communication protocol of receiver and camera.

Camera ID

The camera ID will be shown. You cannot change it to a fixed value.

Transfer rate

Set the connection speed of RS-485.

Data bit

Set the data bit.

Parity bit

Set the parity bit for checking data transmission handling errors.

Stop bit

Set the stop bit which shows the end of a character.

ONVIF focus movement

Select [Enable] if you want to set to adjust the camera focus via an ONVIF protocol.
