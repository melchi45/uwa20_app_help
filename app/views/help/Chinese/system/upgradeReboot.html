升级 / 重启

可以更新目前正在使用的产品软件、进行出厂默认设置、备份或恢复配置或重启。

升级

有新固体版本发行时，可对软件进行更新。

软件版本

显示正在使用的产品的软件版本。无法更改软件版本。

当前 ISP 版本

显示正在使用的产品的当前 ISP 版本。无法更改当前 ISP 版本。

软件升级

可以升级正在使用的产品软件。如需更新软件，请单击[...]按钮。选择一个更新文件，并单击[打开]按钮。单击[更新]按钮时，将开始进行更新。更新进度将以 % 显示。升级结束后，摄像机将重启，并自动断开连接。这时，需要重新连接至 web 查看器。

说明

  • 更新过程中，切勿终止程序。否则，会导致无法顺利更新程序。

出厂默认设置

将系统恢复至购买产品时的配置。单击[重置]按钮，然后单击确认窗口中的[确认]按钮，将执行出厂默认设置。

如需将系统配置（网络设置和已安装的开放式平台设置除外）重置为出厂默认设置，请选择[网络参数和开放式平台除外]。执行出厂默认设置时，将中断摄像机的连接。首次重新连接至 web 查看器时，需重新输入密码。　

配置备份与恢复

可以备份并保存当前的摄像机设置，或恢复所需的配置。可以针对所需的配置创建各种备份文件，或根据用途或使用产品时的环境恢复并使用所需配置。

备份

单击[备份]按钮时，将创建一个‘型号名称 Config.bin’的备份文件。

恢复

单击[恢复]按钮，将出现一个窗口，在该窗口中，可以选择需要恢复的备份文件。选择备份文件，并按[打开]按钮时，将根据对应的备份文件恢复配置。

说明

  • 使用 Safari 连接至 web 查看器时，不能使用配置备份与恢复功能。
  • 备份或恢复配置时，将自动终止与摄像机的连接，需要重新连接至 web 查看器。
  • 如果使用不同型号的导出备份文件，而不是当前使用的产品，那么产品可能出现故障。切勿使用不同型号的备份文件或手动更改配置。

重新启动

重新启动摄像机系统。单击[重新启动]按钮，出现确认窗口时，单击[确认]按钮。摄像机将重新启动，web 查看器窗口将关闭。这时，需要重新连接至 web 查看器。
