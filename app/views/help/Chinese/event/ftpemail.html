FTP / 电子邮件

摄像机捕捉图像过程中创建事件时，所捕捉到的图像会发送至 FTP 服务器或通过电子邮件进行发送。输入合适的 FTP 服务器或所需的电子邮件配置信息。

输入 FTP 服务器或电子邮件服务器信息，并单击页面底部的[应用]按钮时，将执行 FTP 服务器连接测试或电子邮件测试。如果输入的 FTP 服务器地址或电子邮件服务器地址不正确，将出现一条“失败”信息。

FTP 配置

服务器地址

输入 FTP 服务器的 IP 地址，以便将事件创建图像发送至该 FTP 服务器。

ID

输入 FTP 服务器登录账户的 ID。

密码

输入 FTP 服务器登录账户的密码。

上传目录

输入 FTP 服务器的目录，以便在该服务器上录制事件创建图像。

端口

输入 FTP 服务器的端口值。FTP 服务器的默认端口值为 21， 可根据 FTP 服务器设置进行更改。该端口值可在 1024 至 65535 之间进行更改。

被动方式

当由于防火墙或 FTP 服务器配置的原因，需要选择被动方式的连接时，请选择 [启用]。

电子邮件配置

服务器地址

输入电子邮件 SMTP 服务器地址，以便通过电子邮件发送事件创建图像。

验证

选择是否在每次发送电子邮件时进行 ID 和密码验证。

TLS 使用

设置是否使用 TLS。使用安全电子邮件服务器时，请选择[启用]。

ID

输入登录账户的 ID，以便连接至电子邮件 SMTP 服务器。

密码

输入登录账户的密码，以便连接至电子邮件 SMTP 服务器。

端口

输入电子邮件 SMTP 服务器的端口值。电子邮件服务器的默认端口值为 25， 使用 TLS 时，该端口值为 465。

收件人

输入收件人的电子邮件地址。

发件人

输入发件人的电子邮件地址。如果发件人的电子邮件地址不正确，预定的收件人将无法收到电子邮件。

主题

输入创建事件时待发送电子邮件的主题。

正文

输入创建事件时待发送电子邮件的正文。创建电子邮件时，已捕捉的图像将以电子邮件附件进行发送。
