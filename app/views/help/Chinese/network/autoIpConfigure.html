自动 IP 配置

可以自动设置摄像机 IP。可分配另一个 IP 地址，连接至同一本地网络中的另一台摄像机，或设置摄像机 IP，在 Windows 或 Mac 操作系统中检查联网摄像机。完成设置后，单击页面底部的[应用]按钮。

链路本地 IPv4 地址

可分配另一个 IP，从同一本地网络连接至摄像机。

自动配置

如需执行链路本地 IPv4 地址的自动配置，请选择[启用]。

  • IP 地址：显示已分配的 IP 地址。
  • 子网掩码：显示已分配 IP 地址的子网掩码。

UPnP 发现

可从支持 UPnP（通用即插即用） 协议的客户端和操作系统搜索摄像机。

UPnP 发现

如需执行 UPnP 发现，请选择[启用]。

  • 友好名称：显示摄像机名称。按 WISENET - 型号名称 - MAC 地址的顺序，显示友好名称。

Bonjour

可以从支持 Bonjour 协议的客户端和操作系统搜索摄像机。在默认支持 Bonjour 的 Mac 操作系统中，已连接的摄像机显示在 Safari web 浏览器的 Bonjour 书签中。

Bonjour

如需使用 Bonjour，请选择[启用]。

  • 友好名称：显示摄像机名称。按 WISENET - 型号名称 - MAC 地址的顺序，显示友好名称。

说明

  • 如果未显示书签，请勾选“首选项”菜单中的书签。
