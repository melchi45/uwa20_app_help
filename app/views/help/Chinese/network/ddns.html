DDNS

如果使用 DDNS（动态域名服务），可以将摄像机的 IP 地址更改为用户易记住的一般主机名。如果摄像机的 IP 地址为 198.160.0.100，则可以输入诸如 http://ddns.hanwha-security.com/camera1 的主机名，连接至摄像机，而不必输入这个 IP 地址。即使摄像机的 IP 地址更改后，用户可通过 DDNS 地址连接至摄像机，因此，非常方便。

DDNS 可为 Hanwha Techwin 的专用 Wisenet DDNS 或公共 DDNS。输入所需的 DDNS 信息，并单击页面底部的[应用]按钮。如果已连接至所选 DDNS，则会出现一条“成功”信息，如果未连接，则会出现一条“失败”信息。

DDNS

关

不使用 DDNS 时选择此项。

Wisenet DDNS

使用 Hanwha Techwin 提供的 DDNS 时选择此项。如需使用 Wisenet DDNS，请在 Wisenet DDNS 主页 (http://ddns.hanwha-security.com) 注册成为会员，并在[我的 DDNS]>[注册产品]完成产品注册。

  • 服务器：输入想要使用的 DDNS 服务器的名称。
  • 产品 ID：输入在 Wisenet DDNS 服务器注册的产品的 ID。如果使用支持 UPnP（通用即插即用）功能的路由器时选择[快速连接]，则与外部建立连接时，支持自动端口打开。

公共 DDNS

使用公共网站提供的 DDNS 时选择此项。使用前，需在相应网站进行注册，获取相关服务。

  • 服务器：选择打算使用的公共 DDNS 服务器。
  • 主机名：输入在 DDNS 服务器注册的主机名。
  • 用户名：输入 DDNS 服务器的用户名。
  • 密码：输入 DDNS 服务器的密码。

说明

  • 如果路由器不支持 UPnP 功能或在不使用[快速连接]的情况下使用 DDNS 服务器，请将路由器的端口转发设置为“手动”。有关路由器端口转发的设置方法，请参阅产品随附的说明书。
