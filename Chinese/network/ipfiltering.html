IP 过滤

可以制定一个 IP 地址列表，列出允许或拒绝连接的具体 IP。IPv4 和 IPv6 的 IP 地址分开管理。完成设置后，单击页面底部的[应用]按钮。

过滤类型

[拒绝注册的 IP]和[允许注册的 IP]过滤条件适合所有注册的 IP 地址。

  • 拒绝注册的 IP：拒绝某注册 IP 的访问。
  • 允许注册的 IP：允许某注册 IP 的访问。

说明

  • 注册一个授权访问的 IP时，也应注册当前连接至摄像机的 IP。特别是，但在[基本设置]>[IP 与端口]>[IPv6 设置]选择[启用]时，应添加当前连接的 IP 地址的所有 IPv4 和 IPv6 地址。
  • 当前连接的 IP 地址不能注册为[拒绝注册 IP 地址]。

IPv4

这是一个针对 IPv4 类型的地址列表；可以添加或移除 IP 地址。可输入多达 10 个 IP 地址。

添加 IPv4 地址

  1. 单击[添加]按钮。将创建一个字段，可在该字段中输入 IPv4 地址。
  2. 输入 IP 和前缀信息。将显示所输入信息的过滤范围。
  3. 仅当勾选[使用]复选框时，才能过滤相关范围。
  4. 在页面底部单击[应用]按钮，可保存列表中所添加的所有信息。

移除 IPv4 地址

  1. 选择希望移除的 IPv4 地址。
  2. 单击[移除]按钮。
  3. 单击删除确认窗口中的[确认]按钮。IPv4 地址将被移除。

说明

  • 无法提供多播所用的 224.0.0.0 至 239.255.255.254 地址。
