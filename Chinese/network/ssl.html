HTTPS

可以选择安全连接系统或安装公共证书。完成设置后，单击页面底部的[应用]按钮。

安全连接系统

根据与安全等级有关的服务环境选择安全连接系统。HTTPS（安全套接层之上的超文本传输协议）通过对超文本传输协议下方 SSL 子层中用户的页面请求加密和解密，进行数据交换。因此，这种方式比 HTTP 模式更安全。可以选择唯一证书或公共证书，在 HTTPS 模式下进行安全连接。

HTTP（不使用安全连接）

选择在不加密的情况下发送数据

HTTPS（使用唯一证书的安全连接模式）

使用摄像机提供的唯一证书建立安全连接。选择[HTTPS（使用唯一证书的安全连接模式）]并单击页面底部的[应用]按钮时，将在 HTTPS 模式下建立安全连接。

HTTPS（使用公共证书的安全连接模式）

使用公共证书建立安全连接模式。在安装公共证书后，可选择这一模式。选择[HTTPS（使用公共证书的安全连接模式）]并单击页面下方的[应用]按钮时，将在 HTTPS 模式下建立安全连接。

安装公共证书

在[安全连接系统]中选择[HTTPS(使用公共证书的安全连接模式)]时，应先安装公共证书。

  • 证书名称：输入待安装证书的名称。
  • 证书文件：选择安装证书的路径。
  • 密钥文件：选择安装密钥文件的路径。

须知

  • 在应用[HTTPS(使用公共证书的安全连接模式)]模式的状态下，不能安装或删除公共证书，必须先将其改为[HTTP(不使用安全连接)]或[HTTPS(使用唯一证书的安全连接模式)]模式后再安装或删除证书。

    • 安装公共证书

    1. 输入证书名称，安装在[证书名称]上。
    2. 单击[证书文件]上的[...]按钮，从打开的窗口选择证书文件，然后单击[打开]按钮。
    3. 单击密匙文件上的[...]按钮，从打开的窗口选择密钥文件，然后单击[打开]按钮。
    4. 单击[安装]按钮。证书安装已完成。

    删除公共证书

    安装公共证书[安装公共证书]下方的[删除]按钮。
