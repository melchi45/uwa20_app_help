QoS

QoS（服务质量）功能可设置网络过载时数据传输的优先级，并根据设定的优先级别确保数据传输的质量。可在 IPv4 或 IPv6 输入 QoS IP 地址。完成设置后，单击页面底部的[应用]按钮。

IPv4

可在 IPv4 添加或移除 QoS IP 地址。对于前缀而言，默认值为 32，对于 DSCP 而言，默认值为 63。

  • 前缀：此项可设置 IP 动态范围，在 IPv4 中，其值可设置为 1 至 32。
  • DSCP：QoS 优先级别为 DSCP（差分服务代码点）。DSCP 值可设置为 0 至 63；越接近 0，优先级越低。

说明

  • 无法提供多播所用的 224.0.0.0 至 239.255.255.254 地址。

添加 IPv4 地址

  1. 单击[添加]按钮。将创建一个字段，可在该字段中输入 IPv4 地址。
  2. 输入 IP、前缀和 DSCP 信息。
  3. 只有在选中[使用]复选框后，QoS 才能应用于相关的 IPv4 地址。
  4. 在页面底部单击[应用]按钮，可保存列表中所添加的所有信息。

移除 IPv4 地址

  1. 选择希望移除的 IPv4 地址。
  2. 单击[移除]按钮。
  3. 单击删除确认窗口中的[确认]按钮。IPv4 地址将被移除。

IPv6
