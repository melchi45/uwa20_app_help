IP 与端口

输入 IP 地址和端口。可以在[IP 地址]选项栏中设置 IPv4 和 IPv6。可以在[端口]选项卡中为每个协议设置端口。完成设置后，单击页面底部的[应用]按钮。点击[应用]按钮后，需要重新连接至 web 查看器。

IPv4 设置

可以查看或更改 IP 类型、MAC 地址、IP 地址、子网掩码、网关以及基于 IPv4 网络通信所用的 DNS 信息。

IP 类型

选择 IP 连接类型。如果使用的是固定 IP 地址，将其设置为[手动]并输入相关信息。如果使用的是动态 IP 地址，将其设置为[DHCP]并只能输入 DNS 地址。如果使用的是[PPPoE]，输入 DNS 地址、ID 和密码。

  • 手动：直接输入和设置 IP 地址、子网掩码、网关、DNS1 和 DNS2。
  • DHCP: 设置 DNS1 和 DNS2。
  • PPPoE: 设置 DNS1、DNS2、ID 和密码。

MAC 地址

显示的是摄像机的 MAC 地址。

IP 地址

显示的是当前 IPv4 地址。通过将[IP 类型]设置为[手动]，可以更改 IP 地址。

子网掩码

显示的是当前 IP 地址的子网掩码。通过将[IP 类型]设置为[手动]，可以更改子网掩码。

网关

显示的是当前 IP 地址的网关。通过将[IP 类型]设置为[手动]，可以更改网关。

DNS 设置 (DHCP)

当[IP 类型]设置为 [DHCP] 时显示此内容。如选择启用，将自动分配 DNS 地址。

DNS1/DNS2

显示的是 DNS（域名服务）服务器的地址。

主机名

指用于检索 ONVIF GetHostname 指令中所用主机名的名称。首字符必须为字母，且只能输入字母字符。最多可输入 32 个字符。摄像机名称默认输入，并不一定需要填写，因此无需设置。

ID

当[IP 类型]设置为[PPPoE]时，会显示此项。输入正确的 ID 进行连接。

密码

当[IP 类型]设置为[PPPoE]时，会显示此项。输入正确的密码进行连接。

IPv6 设置

IPv6 是下一代互联网地址系统，在 IPv4 基础上扩展了数据处理速度、并行数据处理能力和互联网地址系统。如需使用 IPv6，请选择[启用]。可以设置 IP 类型、IP 地址、前缀和网关。从 IP 安装器中选择摄像机型号时，可以选择 IPv4 或 IPv6 地址，并通过输入合适的地址直接连接至 web 查看器。

IP 类型

选择 IP 连接类型。默认值为[默认]。如未检测到 DHCP，该值会自动更改为之前的设置。

  • DHCP: 显示的是通过 DHCP 分配的 IPv6 地址。
  • 手动：用户可以选择输入一个 IPv6 地址。
  • 默认：显示的是当前 IPv6 地址。

IP 地址

输入 IPv6 地址。

前缀

该值设定了 IP 的范围。如果[IP 类型]为[默认]，则[前缀]值为 64。如果 IP 类型为[手动]，则可以更改 [前缀]值。

网关

当[IP 类型]设置为[手动]时，会显示网关。用户可以直接输入网关地址。

端口

端口是用来发送和接受数据的位置，点击[端口]选项卡，设置相应的项目，然后单击页面底部的[应用]按钮。建议使用 HTTPS 和 RTSP 加强图像安全性。

HTTP

HTTP 端口使用 web 查看器连接至摄像机。默认值为 80 (TCP)。在 Safari、Google Chrome 等浏览器中进行连接时，因为浏览器的安全规定，HTTP 端口无法设置为 65535。当 HTTP 端口改变时，网页浏览器窗口将关闭。在 IP 地址后面输入新的 HTTP 端口，重新连接。当 HTTP 端口为80时，可以省略端口号。（比如：摄像机 IP 地址为192.168.1.100，如果 HTTP 端口为 8080 的话-> http://192.168.1.100:8080）

HTTPS

HTTPS 比 HTTP 的安全性高。当 SSL 设置为 HTTPS 模式，且默认值为 443 (TCP)时，可以使用 HTTPS。可用的 HTTPS 范围为 1024 至 65535。在 Safari、Google Chrome 等浏览器中进行连接时，因为浏览器的安全规定，HTTP 端口无法设置为 65535。

RTSP

此端口用于在 RTSP（实时串流协议）下发送图像；默认值为 554。

使用超时提示

如需使用超时提示，请选择[启用]。当连接设置为 RTSP，且一段时间内没有反应时，请重置端口连接。
