用户

对连接至摄像机的用户帐户进行管理。可以更改管理员密码，进行访客设置、认证设置和当前用户设置。完成设置后，单击页面底部的[应用]按钮。

更改管理员密码

可以更改管理员密码。为了增加安全性，创建的密码应由随机英文大小写字母、数字和特殊符号组成。

当前密码

输入当前密码。为了防止他人更改密码，只有输入之前的密码后才能更改管理员密码。

新密码

输入新密码。

确认新密码

这是一个确认流程，防止输入的新密码有误。再次输入新密码。

说明

  • 如果从多地同时使用同一个 ID 进行连接，或打开多个互联网浏览器更换密码时，互联网浏览器可能会发生故障。建议只通过同一个地址或一个互联网浏览器更改密码。

访客设置

选择[允许访客访问]后，访客可以连接至 web 查看器屏幕。使用访客账号连接时，只能在 web 查看器上看到实时画面。访客 ID 和密码为‘guest/guest’，且无法更改。

认证设置

选择[启用不需要认证的 RTSP 连接]后，可以使用 RTSP（实时串流协议）连接至摄像机图像，而不需要登录认证。

当前用户

可以设置与管理员不同的用户账号连接信息，并设置包括音频输入、音频输出、警报输出和配置文件的使用许可。一名注册用户登陆时，只会启用为该用户设置的功能。默认设置的当前用户账号为 10 个；可以增加或删除账号。当前 IP 最多可允许使用 10 个用户账号。

名称

输入 ID

密码

输入密码。密码设置规则与管理员密码设置规则相同。

音频输入

为输入的音频设置访问许可。选择音频输入时，连接到相应账号的用户不仅可以观看屏幕图像，还可以听到声音，如果没有选择音频输入，则用户只能观看屏幕上的图像。

音频输出

设置音频输出许可。选择音频输出，连接到相应账号的用户可以通过麦克风发送音频。

警报输出

设置警报输出许可。设置好警报后，连接到相应账号的用户可以输出警报。

PTZ

在[实时]模式中设置控制PTZ的权限。

配置文件

设置为[默认]时，用户只能在默认模式下观看图像；设置为[全部]时，用户可以观看各种配置文件中的图像。

输入当前用户

  1. 选择想要使用的当前用户账号的单选按钮。当前用户账号的状态变为可以输入。
  2. 选择[使用]栏中的复选框。
  3. 在[名称]和[密码]栏中输入 ID 和密码。
  4. 在[音频输入]、[音频输出]和[警报输出]栏中分别选择可以允许的功能，然后在[配置文件]栏中选择允许的配置文件种类。
  5. 当前用户账号输入完成时，点击页面底部的[应用]按钮。
  6. 弹出确认窗口时，单击[确认]按钮。

说明

  • 如果当前的用户账号小于 10 个，可以单击[添加]按钮，添加当前用户账号。

修改当前用户

  1. 选择想要修改的当前用户账号的单选按钮。
  2. 更改功能设置，然后单击页面底部的[应用]按钮。
  3. 弹出确认窗口时，单击[确认]按钮。用户的信息将被更改。

删除当前用户

  1. 选择想要移除的当前用户账号的单选按钮，然后单击[删除]按钮。
  2. 单击页面底部的[应用]按钮。
  3. 弹出确认窗口时，单击[确认]按钮。所选中的当前用户的信息将被删除。
