Preset setup

You can set and delete a camera preset, select whether or not to save the current position as the last position of the camera, and set the saving interval.

Preset setup

Preset refers to the setting of the camera views. You can easily set a camera preset and move to the relevant view. 300 presets can be set.

Setting a preset

  1. Adjust the screen as follows to adjust the camera view.
    • Select image magnification by adjusting the zoom bar, or by clicking the icon to zoom the screen in or clicking the icon to zoom the screen out from [Zoom].
    • Adjust the camera focus by clicking the icon or icon from [Focus].
    • To adjust the camera focus automatically for once, click the icon from [Focus].
  2. Click the [Set preset] button.
  3. Select a desired preset number from [Number] on the [Set preset] dialog box.
  4. Enter the relevant preset name in [Name].
  5. Click the [OK] button.

The added preset will appear on the list of [Preset setup]. You can set the details for the relevant preset on the preset list.

Navigating to the home position

  • Click the [Go home] button.

Setting the home position

  1. Adjust the camera position you wish to set as the home as follows.
    • Select image magnification by adjusting the zoom bar, or by clicking the icon to zoom the screen in or the icon to zoom the screen out from [Zoom].
    • Adjust the camera focus by clicking the icon or icon from [Focus].
    • To adjust the camera focus automatically for once, click the icon from [Focus].
  2. Click the [Set home] button.

Note

  • Camera is moved to the Home position if the last position is not set when you turn the camera off and on.

Preset setup

The set preset list is displayed.

The number and the preset name entered on the [Preset setup] dialog box will appear on the [Number] and [Name] columns, respectively.

Moving to a preset

  1. Select a desired preset from the check box next to the [No.] column.
  2. When you click the [Go] button, the camera views will move to the set location.

Deleting a preset

  1. Select the preset you wish to delete from the check box next to the [No.] column.
  2. When you click the [Remove] button, the preset will be deleted.

Last position

You can set the camera to move to the last position when turning the camera off and on. Select whether or not to save the last position of the camera and select the saving interval.

Last Position

To save the last position, select [Enable].

Time

Set the saving interval to save the current position as the last position. The unit used to measure the interval is seconds. Enter the saving interval on the field.

Auto run

An action specified in advance is automatically executed if the user does not operate the camera for a certain period of time.  

Mode

  • Off: The auto run function is not used.
  • Home: The camera moves to the home position.

Time

Select a time that the camera is not operated to execute the camera automatically.

This option is activated when [Mode] is set to [Home].
