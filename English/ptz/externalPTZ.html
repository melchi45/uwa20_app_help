External PTZ

The user can set to control the PTZ through the pan/tilt receiver which is connected to the RS-485 or RS-422 terminal. You can set a preset or the communication mode of the pan/tilt receiver connected to the camera.

Adjusting the PTZ

You can adjust the pan, tilt, and zoom actions of the camera.

Adjusting the PTZ

  • When you drag the icon with the mouse, the camera moves to the pan and tilt.
  • Select an image magnification by clicking the icon to zoom the screen in or clicking the icon to zoom the screen out from the zoom bar.
  • Move the focus near by clicking the icon or move the focus far by clicking the icon.
  • To adjust the camera focus automatically for one shot, click the icon.

Preset

Preset means the setting of the camera views. 128 presets can be set.

Setting a preset

  1. Expand the camera image by clicking the icon on the PTZ control screen, or adjust the camera views by clicking the icon and dragging the mouse.
  2. Select a preset number from [Number].
  3. Enter the relevant preset name in [Name].
  4. Click the [Add] button.

Moving to a preset

  1. Select a preset number from [Number].
  2. When you click the [Go] button, the camera views will move to the set location.

Deleting a preset

  1. Select the preset number you wish to delete from [Number].
  2. When you click the [Remove] button, the preset will be deleted.

Serial port setup

You can set the connection value of an external pan/tilt receiver. When you complete the setting, click the [Apply] button at the bottom of the page.

Protocol

You can choose the communication protocol of receiver and camera.

Camera ID

The camera ID will be shown. You cannot change it to a fixed value.

Transfer rate

Set the connection speed of RS-485.

Data bit

Set the data bit.

Parity bit

Set the parity bit for checking data transmission handling errors.

Stop bit

Set the stop bit which shows the end of a character.

ONVIF focus movement

Select [Enable] if you want to set to adjust the camera focus via an ONVIF protocol.

Note

  • All these operations above mentioned will be enabled only if the network camera is properly connected to the pan/tilt-compliant receiver. And the serial port for the pan/tilt operations should be properly specified.
