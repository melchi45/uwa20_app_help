You can add a channel fixed as a specific type among a variety of Fisheye view types.

Click the [Apply] button below when the setup is complete.

Video channel

Adding a video channel allows you to select and view a specific view type on the live screen. There are a variety of Fisheye view types. If you add a certain view type in [Video channel], you can select and view only the profile of the view type you selected in the [Live] page.

Add video channel

  1. Click the [Add] button.
  2. Select a view type you want from the dropdown menu in the [Fisheye view type] column.

If you click the [Delete All] button, all the video channel view types previously set will be deleted.

Note

  • Up to 2 video channels can be set.
  • You cannot set both [Single panorama] and [Double panorama] at the same time for channel view type.   
  • Even if you add a Fisheye view type in [Video channel], if there is no profile that supports the view type in [Video profile], it won't play on the [Live] page.
