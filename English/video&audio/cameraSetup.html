Camera setup

In the environment where a camera is installed, you can change the camera setup to record an optimal quality video. By default, image presets suitable for various environments are provided, and users can set up their cameras on their own. On the camera preview screen, you can check how the camera image will look. Once you have completed the setup, click the [Apply] button at the bottom of the page. If timeout occurs (240 sec) without clicking the [Apply] button after changing the settings, then the settings will be reverted to the previous settings.

Select channel

After selecting a camera channel, you can set the details of each channel. The channel number last selected on the live or play page is automatically selected.

The fixed camera channel is displayed as a number and the PTZ channel is displayed as ‘PTZ’.

If you select another channel without applying the changes in the setting, an alarm message will appear.

Adjusting the PTZ

You can adjust the pan, tilt, and zoom actions of the camera.

Only after you select a [PTZ] channel in [Select channel] channel, the PTZ jog panel will appear.

Adjusting the PTZ

  • When you drag the icon with the mouse, the camera moves to the pan and tilt.
  • By using the zoom bar, you can adjust the video zooming ratio. Or, click the icon to magnify the screen or click the icon to reduce the screen.
  • Move the focus near by clicking the icon or move the focus far by clicking the icon.
  • To adjust the camera focus automatically for one shot, click the icon.
  • Place the screen center toward the north, and click the (Set North) button to change the location to the north.

Type

If you want to apply the same camera setting to every preset, select [Global]. If you want to apply a different setting to each PTZ preset, select [Preset]. On the Preset Setting page, only if the preset is enabled, the [Preset] option will be activated. If you select [Preset] and set the camera setting for each preset, then you may not set some of the items in the detailed camera settings.

Only after you select a [PTZ] channel in [Select channel], you can set the 'Hallway View' function.

Sensor mode

Set how many frames per second the CMOS sensor of the camera will capture (the maximum value of [Basic]>[Video profile]>[Frame rate] according to the number of frames set for the sensor).

Sensor mode

Set a desired frame rate. A different frame rate can be set to each image preset and the frame rate is applied in the same way to every image preset.

Image preset mode

Various image presets are provided for different purposes. Select an image preset appropriate for your camera environment.

  • User defined preset 1: Used for displaying an image as set by user.
  • User defined preset 2: Used for displaying an image as set by user.
  • Outdoor daytime: Used to display a clear and smooth video taken outdoor in daytime.
  • Outdoor nighttime: Reduces the level of noises for video taken outdoor at night with low brightness, and lighten up dark areas.
  • Indoor backlight: Displays video that is visible both indoors and outdoors with indoor backlight.
  • Indoor with bright lighting: Provides clear picture quality and reduces flickering in an illuminated indoor environment.
  • Car license plate: Used to improve the ability to identify the license plate of a car in daytime and at night.
  • Vivid video: Used to improve color and clarity.

Note

  • After selecting an image preset mode, change the detailed settings of camera image, such as white balance and night/day modes, and click [Apply]. The changed value will be saved as the image preset value. If you want to revert it to the initial value, click the [Reset] button.

SSDR

The SSDR function is used to make everything look regular by only increasing the brightness of dark parts where both dark and bright parts are significantly different from each other. A level value is automatically set to each image preset provided by the camera and they cannot be changed. If you do not want to use the function or change the level, then you need to use a user preset.

SSDR

To use SSDR, select [Enable].

Level

Adjust the dynamic range. The higher the level is, the brighter the dark part gets.

D-Range

Select the amplitude area of the dynamic range.

White balance

You can make corrections to show the color white correctly and other colors normally, in any lighting environment. The setting is irrelevant to the image preset.

Mode

Select the white balance mode according to the camera environment.

  • Manual: Manually adjust the red and blue gains. If there is too much red, then reduce the red gain; if there is too much blue, then reduce the blue gain; if there is too much green, then increase the blue and red values at the same time.
  • ATW: Corrects the camera color automatically.
  • Outdoor: The colors of the camera are corrected to be optimized for an outdoor environment.
  • Indoor: The colors of the camera are corrected to be optimized for an indoor environment.
  • AWC: Adjust the screen to optimize the camera color to the current lighting environment. Place a white paper in front of the camera and press the [Setting] button to adjust the red/blue gains and apply the results to the video. The white balance value continues to be applied to the currently displayed video and you need to adjust it again when the environment changes.
  • MERCURY: Screen correction is made to adapt the colors to the mercury lighting environment.
  • SODIUM: Screen correction is made to adapt the colors to the sodium lighting environment.

Red gain

Adjust the red gain. If the red gain is high, then there is too much red color on the screen. In that case, reduce red gain.

Blue gain

Adjust the blue gain. If the blue gain is high, then there is too much blue color on the screen. In that case, reduce blue gain.

Note

  • In any of the following cases, the white balance may not work properly. If so, then use the [AWC] mode to adjust the white balance.
    • If the surrounding environment exceeds the color temperature correction range, such as a clear sky or sunset
    • If the surrounding environment is dark
    • If the camera faces directly toward a fluorescent light or there is a significant change in illumination.

Backlight

You can correct an image exposed to backlight. Select either Off, BLC, HLC, or WDR under [Mode], and change the setting according to the selected backlight mode.

BLC

The BLC (Back Light Compensation) mode selects and corrects a specific area on a dark image captured due to backlight, to show an object in the selected area more clearly. Use the up/down/left/right button to adjust the area size.

BLC level

You can select either low, middle, and high. The higher the level, the brighter the area.

Area setting
  • Up: Set how far the top surface is away from the top of the screen.
  • Down: Set how far the bottom surface is away from the bottom of the screen.
  • Left: Set how far the left surface is away from the left side of the screen.
  • Right: Set how far the right surface is away from the left side of the screen.

Note

  • The green box that displays the menu to apply weights to the brightness setting of BLC will appear for about 15 seconds before disappearing.
  • To prevent malfunctions while setting the BLC, the vertical interval of the BLC is limited to max 60 / min 40 and the horizontal interval is limited to max 60 / min 30.

HLC

The HLC (High Light Compensation) mode blocks strong lighting such as a streetlight or the headlights of a vehicle, to prevent an object from being unrecognized. It can prevent failures to detect a neighboring object such as the car license plate due to saturation of the strong lights.

HLC level

Set the level of exposures and adjust the highlight areas.

You can select either low, middle and high. The higher the level, the higher the correction and the bigger the difference is for highlight-blocked area.

Mask

Selects whether or not to apply the mask.

  • Off: Does not create blocked area.
  • On: Always blocks areas of brightness above a certain level.
  • All day: Blocks areas except during very bright or dark time.
  • Night only: Blocks areas exists until it gets really dark.
Mask color

Sets the color of the blocked area.

Mask tone

Set the tone of the blocked area.

Dimming

Detects the strongly illuminated area on the screen and adjusts brightness to reduce the size of the saturated areas.

Area setting

Set the blocked area’s size and location.

  • Up: Set how far the top surface is away from the top of the screen.
  • Down: Set how far the bottom surface is away from the bottom of the screen.
  • Left: Set how far the left surface is away from the left side of the screen.
  • Right: Set how far the right surface is away from the left side of the screen.

Note

  • At night, it only operates when there is a highlight shone to a certain area size or bigger in a dark environment.
  • At night, if it is overall too bright or dark, then HLC will not work. In daytime, if it is too dark, HLC will not work.
  • The green box that displays the menu to apply weights to the brightness setting of initial HLC will appear for about 15 seconds and then disappear.
  • To prevent malfunctions while setting the HLC, the vertical interval of the HLC are is limited to max 60 / min 40 and the horizontal interval is limited to max 60 / min 30.

WDR

WDR (Wide Dynamic Range) mode is used to improve the visibility of both dark and bright areas in the backlight. Using the dual shutters of camera, you can take two shots of photos—one with a short shutter to make the bright part more visible and other with a long shutter to make the dark part more visible. When in WDR mode, there can be noises between the dark and bright areas.


WDR mode off                            WDR mode on

WDR level

Adjusts the intensity of backlight correction.

Auto

If [Enable] is selected, the image is analyzed and WDR mode is activated automatically when backlight correction is necessary.

Turn off in low light

If [Enable] is selected, the WDR mode is turned off automatically in a low-light environment.

Turn off in B/W

If [Enable] is selected, the WDR mode is turned off automatically when the B/W mode operates.

Note

  • The shutter value resets if the WDR mode is selected. Thus, the screen gets bright and then dark.
  • When the P-Iris lens is used manually or the anti flicker shutter is used, the performance of WDR mode becomes limited.
  • When WDR mode is used, the frame rate is reduced by half.
  • It is recommended to use the WDR mode when a camera is installed indoor and there is very strong backlight.
  • If the WDR mode is used, noise may occur between the bright area and the dark area.
  • If the WDR mode is used, noise may occur in the motion detection area.
  • Please turn off the WDR mode because, depending on the lighting condition in WDR mode, the following problems could occur:
    • Unnatural color changes, unnatural symptom on the screen.
    • When there is noise on the bright part of the screen.
  •  The WDR performance may vary depending on the size of bright areas on the screen, so adjust the installation angle properly for the best WDR performance.
  • If you set the WDR level too high, the screen may exhibit unnatural symptoms.
  •  For the best WDR performance, it is recommended to set the exposure iris to [Auto].

Exposure

You can adjust the level of exposure to meet the camera recording environment. If the background is darker than the object, you need to reduce the level of exposure to display the object properly. On the contrary, if the background is brighter than the object, then you need to increase the level of exposure. Each image preset is set to default which cannot be changed. But, the brightness for any image preset can be adjusted.

Brightness

Adjusts screen brightness. The greater the number is the brighter the screen gets.

Minimum shutter

The sensor exposure can be automatically set by the shutter depending on the environment. The electric shutter operates within the min/max shutter range. The minimum shutter is the minimum duration of shutter&apos;s low speed exposure and it is used to set the limit of long exposure.

If the selected minimum shutter value is smaller than the fps value of [Sensor] mode, the frame rate can be reduced in the dark.

Maximum shutter

Maximum shutter means the maximum value of the possible range for exposure time and sets a short exposure time.

Prefer shutter

Set the proper exposure time that has a high priority in operation within the range of exposure time.

Anti flicker

Prevents screen flickering that occurs due to inconsistency in the lighting in the capturing environment.

Control speed

If you want to speed up the screen exposure when there is a sudden change in intensity (for example, when entering or exiting a tunnel), then select [High]. If the screen brightness is changes too often as it reacts very sensitively to the intensity change, then it is recommended to set it to [Normal].

SSNR

Removes noise from the image.

  • On/Off: SSNR function can be enabled or disabled. If you select [Enable], then the SSNR level can be adjusted.
  • Wise NR: When a moving object is in the image, the noise reduction level is adjusted automatically to improve the accuracy of object identification.

SSNR 2D Level

SSNR 2D uses adjacent pixels in a single frame to reduce the level of noises in an image. Set the SSNR 2D level. The setting can be used by setting [SSNR] to [Enable] or [Wise NR].

The higher the level, the less the noises, but then the video might become fuzzy.

SSNR 3D Level

SSNR 3D uses pixels in multi frames to reduce the level of noises in an image. Set the SSNR 3D level. Enable this by setting [SSNR] to [Enable] or [Wise NR].

The higher the level, the less the noises, but then the video might lag.

Iris

Select either (Auto), manual, or P-Iris.

  • DC(Auto): Automatically adjusts the iris depending on the optical intensity of the surroundings.
  • Manual: Manually adjusts the iris&apos;s F value.
  • P-iris: Snap motor is used to adjust the shutter. A large depth of field can be secured so that a number of objects at various distances can be brought into focus at the same time.

Iris F-No

Activated when [Iris] mode is set to [Manual], and it sets the iris f-number.

Lens

Select either DC (Auto), manual, or any available P-Iris.

  • DC (Auto): Adjusts iris by determining the amount of incoming light.
  • Manual: Manually adjusts iris.
  • P-iris: Snap motor is used to adjust the shutter. A large depth of field can be secured so that a number of objects at various distances can be brought into focus at the same time. There are many different types of iris you can select from.

P-iris

It is enabled if you select the P-iris under [Lens]. If you select [Auto], then the iris is automatically adjusted based on the brightness. If you select [Manual], then you can set the iris location on your own.

P-iris position

It is enabled if [P-iris] is set to [Manual]. The P-iris location can be adjusted as well.

AGC

AGC (Auto Gain Control) adjusts the brightness by controlling the sensitivity of image gain when capturing an object in dim light.

Note

  • The screen exposure can be saturated depending on the max/min range of shutter settings.
  • It may take a while until the effect is applied or canceled after setting the Wise NR mode (SSNR 2D and SSNR 3D).
  • The level of noise might go up despite the improved video detection ability after setting the Wise NR mode (SSNR 2D and SSNR 3D).
  • In the bright environment with less noises, the effect of Wise NR (SSNR 2D and SSNR 3D) can be small.
  • Since the motion detection function is used internally when the Wise NR (SSNR 2D and SSNR 3D) is enabled to detect motion in video, it is detected according to the operational condition of motion detection event. For the detailed motion detection settings, go to [Analysis] > [Motion Detection].

Day/Night

You can switch a camera&apos;s image to color or black/white depending on the environment. In addition, by deciding the switching time, it can be automatically switched to black/white or color at the specific time. When switching night and day modes, you cannot detect a motion detection event or a video analysis event.

Mode

Determines the change of the camera mode between color and B/W.

  • Color: Displays an image in color.
  • Black and white: Displays an image in B/W all the times.
  • Auto: It switches to color mode in daytime and to black and white at night or time with low brightness. If [Off] is set for [AGC] of [Exposure], Day/Night cannot be set automatically.
  • External input: When an alarm input terminal is linked with external equipment, color and B/W of the video can be controlled.
  • Schedule: Day/night is changed to the schedule set from [Activation time (Color)].

Dwell time

Changes to color or B/W when [Auto] is selected for Day/Night and the brightness condition is maintained for the set dwell time.

Duration

Sets the time interval when switching to color or B/W occurs.

Alarm input

Sets the video to color or B/W depending on opening or closing of the alarm sensor. When [External] is set for [Mode] of [Day/Night], the [Event]>[Alarm input] function will be deactivated and the setup page will not appear.

Simple focus after Day/Night

Adjusts the focus of the image automatically after a Day/Night switch.

Activation time (Color)

Schedules the time of color mode operation. If you select [Everyday] and set the time, then it operates in color mode during the specified time period everyday, and it operates in black and white mode during the rest of the day. If you do not want it to run at the specified time period everyday, then deselect [Everyday] and set the operation time for color mode for each day (Mon, Tue, Wed, Thr, Fri, Sat, and Sun).

Note

  • When switching between night and day modes, the motion detection event will not take place. In other words, even if you enabled the motion detection event, it will not detect when switching between night and day modes occurs. You can enable or disable the motion detection event in [Analytics] > [Motion detection].  

Special

Adjusts the sharpness, contrast, color level, and defog of images.

DIS

DIS (Digital Image Stabilization) is used to reliably display the screen by automatically correcting images when the camera is shaking due to external impacts, such as wind.

Note

  • If you change the DIS setting, the video will zoom in or out. To help you analyze the changed video and accumulate the statistics, change the analytics and statistics settings. The settings can be changed in the [Analysis] and [Statistics] menus.

Note

  • If you change the DIS settings, the video screen will zoom in or out. To help you analyze the changed video, change the analysis settings in the [Analysis] menu.

Sharpness

Adjusts the overall sharpness of an image.

Sharpness level

Sets the overall sharpness of video. You can set the sharpness level by setting [Sharpness] to [On].

The higher the edge level is, the stronger and clearer the video edge is.

Gamma

Adjusts the video contrast. This refers to the brightness gap between the brightest part and the darkest parts of an image; a higher gamma value means a clearer display of the difference in brightness.

Contrast

Adjusts the shade contrast of the image.

Color level

Adjusts the intensity of color in the image.

Defog

Corrects the image when there is a for or the weather is bad. Set it to [Auto] to automatically correct an image by detecting the surrounding environment. If you set it to [Manual] then you can observe the surrounding environment and set the correction quantity on your own.

The initial value is set to [Disable].

Defog level

If the Defog function is set to [Manual], it is enabled to adjust the Defog level. If the Defog level is high, the image is displayed more clearly. If the fog level is low but the Defog level is high, then the screen can get darker.

Note

  • If you set the Defog mode to [Auto], as the amount of fog decreases, the level of defog function is also lowered. If you want to maintain the same effect of the function despite a decrease in the fog, then set the Defog mode to [Manual].
  • If the fog level is low and a manual high defog level is high, the screen may look darker.

LDC

LDC (Lens Distortion Correction) corrects image distortion at the edge of a wide angle lens.

  • Disable: Do not use the image distortion correction function.
  • Manual: Adjusts the distortion correction level manually.
  • Auto: Adjusts the image distortion automatically.

LDC Level

Adjusts the level of distortion correction. Set [LDC] to [Manual] to enable it.

OSD

Displays the camera title or the date & time, and sets the location, size, color, and transparency of the characters.

Camera title

Sets whether to display a camera title or not. If you select [Enable], then the [Add]/[Delete] buttons will be activated.

Click the [Add] button to add a field where you can enter the camera name for the list. After entering the camera title, move the X and Y coordinates to set the location. You can add up to 5 camera titles.

To delete a camera title, select a camera name to delete and click the [Delete] button. After entering the camera title, if you want to see how it looks, then enter the camera name and click the [Preview] button.

Date & Time

To display the date & time on the screen, select [Enable]. Select the date display type and set the position to display the date & time by moving the X and Y coordinates.

Weekday overlay

Adjusts the size of OSD.

Size

Adjusts the size of OSD.

Color

Sets the color of OSD.

OSD text transparency

Sets the transparency of OSD.

Overlay

The user can display the camera&apos;s PTZ position, or preset names on the screen.

  • PTZ position: Sets whether to display the PTZ location on the screen or not.
  • Preset name: Sets whether to display the preset name on the screen or not.
  • Camera ID: Sets whether to display the camera number on the screen or not.
  • Azimuth: The function is used to display the azimuth of the area that you are looking at on the screen currently. When the function is enabled, the 8 directions of E(East)/W(West)/S(South)/N(North)/NE(Northeast)/NW(Northwest)/SE(Southeast)/SW(Southwest) are displayed on the right bottom of the screen.

Note

  • In case of the item with a location that can be adjusted(camera title, date & time), if it overlaps with other fixed OSD items, the screen may fail to display it properly.
  • The camera ID in &apos;Overlay’ is displayed at the left top of screen while the &apos;PTZ position&apos; is displayed at the left bottom of screen, but the locations cannot be changed.
  • Only Korean, English, numbers and symbols are allowed.

Color Palette

You can choose the color palette you want, so that you can visually analyze the thermal camera images.

Color Palette

Selects the desired color palette in the option items. The selected option is applied in the same way to the [Live] page.

The color representation scheme for each color palette option is as follows.

  • White hot: The temperature change from high to low is expressed in the color change from white to black.
  • Black hot: The temperature change from high to low is expressed in the color change from black to white.
  • Rainbow: The temperature change from high to low is expressed in the color change from red, orange, yellow, green, blue, indigo, to violet.
  • Wisenet: The temperature change from high to low is expressed in the color change from orange to black.
  • Sepia: The temperature change from high to low is expressed in the color change from yellow to black.
  • Red: The temperature change from high to low is expressed in the color change from black to red.
  • Iron: The temperature change from high to low is expressed in the color change from yellow to violet.

Focus

Sets the camera focus operation method, whether or not to use digital zoom, and reset the camera.

Mode

Sets the camera focus operation method.

  • Manual: Adjusts the camera focus manually.
  • Auto: Adjusts the camera focus automatically.
  • One shot AF: In case of a user event (Auto Focus or Zoom Action), the focus will be automatically adjusted once.

Digital zoom

To use digital zoom, select [Enable].

Digital zoom limit

Sets the maximum range of digital zoom available. You can set the range of digital zoom by setting [Digital zoom] to [Enable].

Zoom mode

Sets the method to synchronize the camera focus when the zoom is operating.

  • Auto: Even if the zoom magnification is changed, the zoom operates while maintaining the focus by continuously appraising and modifying the condition in focus.
  • Manual: Zooming moves along the focal track that is set. When the zoom magnification is changed, it may go out of focus.

Zoom speed

Sets the zoom speed when the Zoom mode function is enabled.

Lens reset

You can reset the lens manually once or set the reset period to reset the lens automatically.

If you want to manually reset the lens, then go to the [Init. lens] option and select [Manual] and click the [Set] button. If you want to set the auto reset period, then select the desired time from day 1 to 7. The lens reset starts at the set time and it is reset according to the set period.

Note

  • Since the automatic focusing function adjusts the focus based on a subject in the middle of the screen, subjects on the periphery of the screen may be out of focus.
  • The automatic focusing function may not operate normally under the following conditions.
    • If there is a bright light or flashing light, such as a neon sign
    • If the illumination in the monitoring area is low
    • If the illumination in the monitoring area is excessively high
    • If the camera operates with a slow shutter
    • If the subject is dark
    • If there is no intensity difference, such as between the sky and a wall
    • If the camera is aimed at a thin horizontal line
    • If there are objects both far and near within the monitoring area,

Heater

One time act

If frost is present and you would like to defrost by operating the heater, click the button once.

Note

  • In low temperature regions with frosts, the heater is always on. In high temperature regions that may affect the product, it will not operate even if a user turn on the setting.

Fan

One time act

When the internal humidity is formed due to a sudden temperature change in the outside, the fan will run to remove it.

If you would like to operate the fan, click the button once.

Note

  • In low and high temperature regions, the fan is always on and it cannot be turned off by a user changing the settings.

IR

Mode

In the B/W mode when the IR LED is turned on, the saturation at the center of screen is prevented, so it is possible to identify a nearby object.

  • Disable: Disables the IR mode.
  • Auto 1: Depending on the brightness of object at the center of screen, the IR brightness is automatically adjusted.
  • Auto 2: Depending on the brightness of objects at the center of screen and near it, the IR brightness is automatically adjusted.
  • Manual: Manually adjusts the IR brightness.

Level

Set the IR level.

The IR level is enabled when [Mode] is set to [Manual].

Activation time

A camera always operates with a specific image preset. You can set the desired image preset at the desired time. If you do not select a specific image preset, then the image preset selected at the top will operate by default. The color will change according to the image preset selected above during the operation time.

Off

The camera operates at the time selected in the image preset.

Only scheduled time

You can set the settings to run a desired preset at the scheduled time. Select an image preset and set the Start and End times. Set an image preset for each hour appropriate to the camera environment.

The date from Sunday to Saturday, based on the current camera time, is displayed on the time table shown when [Only scheduled time] is selected. You can set the time for action settings by clicking or dragging on the time table with the mouse. The set activation time is executed repeatedly on the relevant weekday and time.

You can change the time view unit by clicking the [1 min], [30 min] or [1 h] button. When you click the [Reset] button, all set activation times will be deleted. To confirm or change the camera time, refer to [Basic]>[Date & Time].
