Video setup

You can set a privacy area on the camera image or mirror the image screen up, down, left or right. You can also set an analog image output or change the video output type. When you complete the setting, click the [Apply] button at the bottom of the page.

Select channel

After selecting your camera channel, you can set the details of Video for each channel.

Privacy area

You can set a part of the camera image as a hidden privacy area. You can set a privacy area on the screen simply by selecting [Enable Privacy area] and clicking the [Apply] button at the bottom of the page.

If you prefer to not have a privacy area, deselect [Enable Privacy area] and click the [Apply] button at the bottom of the page. You can check the camera image with no hidden privacy area. Set privacy areas will not be deleted; you can check such areas in the privacy area list.

Pattern

Select the mosaic pattern which you prefer to apply to the privacy area. If [Solid] is set, the color set from the privacy area window will apply. When you select a pattern, it applies to all privacy areas equally.

Setting a privacy area (for fixed channel)

  1. Select [Enable Privacy area] and click the [Apply] button at the bottom of the page.
  2. Click the 4 edges of the camera image screen with the mouse. The privacy area window will appear.
  3. Enter the name of the privacy area in [Name], select the color to cover the image in [Color] and click the [OK] button.
  4. The new privacy area will be added to the privacy area list. The privacy area is displayed in the set color on the camera image screen.

Setting a privacy area (for PTZ channel)

  1. Select [Enable Privacy area] and click the [Apply] button at the bottom of the page.
  2. Drag the mouse to set the privacy area on the camera image screen. The privacy area window will appear.
  3. Adjust the privacy area by dragging the icon with the mouse if desired.
  4. Enter the name of the privacy area in [Name], select the color to cover the image in [Color].
  5. Select the option for [Zoom threshold] and click the [OK] button. The privacy area will appear only when you zoom in more than you set in the [Set zoom threshold] window.
  6. If 'Zoom threshold' is set to 'On' in step 5, [Set zoom threshold] window will appear. Ajust the zoom to the desired magnification and click the [OK] button. The privacy area will appear only when you zoom in more than you set in the [Set zoom threshold] window.
  7. The new privacy area name will be added to the privacy area list. Privacy area that apply 'zoom threshold function' is shown with [ZOOM] following the name.The privacy area is displayed in the set color on the camera image screen.

Note

  • Only English letters, numbers, dash (-) and period (.) can be entered for the name of a privacy area.

Deleting a privacy area

  1. Select the privacy area which you wish to delete from the privacy area list. The selected area will be displayed on the camera image screen.
  2. Click the [Delete] button. The selected area will be deleted.

Adjusting the PTZ

You can adjust the pan, tile, and zoom actions of the camera. When you click [PTZ] in the Select channel menu, the PTZ jog panel displays for camera PTZ adjustment.

  • When you drag the icon with the mouse, the camera moves to the pan and tilt.
  • Select an image magnification by adjusting the zoom slide bar, or by clicking the icon to zoom the screen in or clicking the icon to zoom the screen out.
  • Move the focus near by clicking the icon or move the focus far by clicking the icon.
  • To adjust the camera focus automatically for one shot, click the icon.
  • After pointing the center of the camera view to the north and clicking the (Set North) button, the corresponding orientation will be saved as north.

Video rotation

If the screen is displayed upside down after installing the camera, you can correct this through the flip mode and mirror mode functions.

The hallway view feature provides a long vertical view of the screen which allows the user to operate the surveillance area efficiently.

Note

  • If you change the video rotation setting, the video screen will be flipped vertically or horizontally or changed. Change the analytics and statistics settings so that you can analyze the video or accumulate the statistical data according to the changed video. These settings can be changed in the [Analytics] menu and [Statistics] menu.

Note

  • If you change the video rotation setting, the video screen will be flipped vertically or horizontally or changed. Change the analytics setting in the [Analytics] menu so that the video can be analyzed according to the changed video.

Flip

You can flip the camera image upside down. To flip the camera image, select [Enable].

Mirror

You can mirror the camera image. To mirror the camera image, select [Enable].

Flip/Mirror

Select [Use] to flip the camera output horizontally or vertically.

Digital flip

When changing the camera position using the PTZ function, the camera output may be flipped upside down, or from one side to the other. At this time, click [Use] if you want the video screen to be automatically reversed and displayed normally.

Hallway view

You can rotate the camera image and enlarge the top and bottom monitoring areas in an environment such as a narrow hallway or an alley where it is difficult to monitor with a normal camera. You can rotate the image 90 degrees clockwise and 270 degrees clockwise; when you set to 0 degrees, the image returns to its original state.

Note

  • In order to set the hallway view, you need to set the installed camera to 0 degrees, or rotate to 90 degrees clockwise or 270 degrees clockwise, and select the same angle in [Hallway view].
  • When you set [Hallway view], the web browser will close automatically. You will need to reconnect to the web viewer.

Camera mounting mode

Selects the camera mounting mode. Selects [Wall] when the camera is installed on the wall, or [Ceiling] when it is installed on the ceiling.

Note

  • When you change the camera mounting mode, all the profile settings set in [Video profile] are reset and the camera reboots. You also need to change the video analytics and statistics settings to fit the screen that changes. These settings can be changed in the [Analytics] and [Statistics] menus.

Analog video

Video output

You can set whether or not to output analog video. Analog video output is useful since you can set it while checking the lens location or focus when installing the camera.

Video output type

Select the analog video output type. You can select either [NTSC] or [PAL] as the video output type.

  • NTSC: NTSC (National Television System Committee) is the standard for color television in the U.S. It operates at 30 frames per second with 525 horizontal scanning lines. The number of frames per second is large so that the screen can display naturally. This type is mainly used in the U.S., Japan and Canada.
  • PAL: PAL (Phase Alternation Line) originated in Germany, and while it is similar to NTSC, there is a difference in the method it uses to handle color signals. This type is used mainly in Europe. Although this type has a smaller number of frames per second in comparison to NTSC, it has a larger number of horizontal scanning lines (625) and less color deformation; it does not require advanced specifications for broadcasting equipment.
