PTRZ setup

PTRZ is a function that allows a camera without PTZ function to pan, tilt, move, and rotate as well as zooming in/out on the camera. This is especially useful for cameras that are mounted on the ceiling/wall or in the corner since it allows the directional movement of the camera position to offer an optimal view.

In 'PTRZ setup' page, you can use the PTRZ function to adjust the camera position and set up zoom & focus.

Note

  • It is recommended to use the PTRZ function to set the desired field of view or align the video screen horizontally when installing the camera.

Select channel

After you select a camera channel, you can specify the details of PTRZ (camera pan & tilt location, rotating, and zooming) for each channel.

Once you select a channel, only the video of the selected channel will be displayed on the screen, and you can specify the details of each individual PTRZ for each channel.

If you select the Multi view icon, you can see the videos of every channel all at once, and apply the same values to all cameras. You can also apply the same values to only one or more cameras of your choice.

Also, in the Multi view mode, a message that describes the status of each camera is displayed at the bottom of the preview screen of each channel. One of the following messages will be displayed:

  • Initializing: This message is displayed when the camera initialization is in progress after [Initialize PTRZ] is clicked. Until initialization is complete, no other action can be made.
  • Initialization failed: This message is displayed due to several reasons such as camera stepping out of phase during the process of camera setup initialization. If initialization is incomplete, PTR control might not work properly. In that case, change the camera position and reinitialize.
  • Moving to preset: This message is displayed when camera is currently moving to a preset position you already set up upon clicking [360], [200], or [User preset]. Until the movement is complete, no other action can be made.
  • Preset failed: This message is displayed when camera fails to move to a preset position due to problems such as out-of-phase issue. Change the camera position by adjusting the PTR of the camera, and retry the Goto preset operation.
  • Currently moving: This message is displayed when any one of the PTR (Pan / tilt / rotation) operations is in action.
  • Collided: This message is displayed when a camera is unable to move as it comes into contact with another camera that uses a different channel.
  • Standby: This message is displayed when PTR is in idle mode and cameras are keeping a certain distance from one another.

To initialize the PTRZ setup, select [Initialize PTRZ]. [Initialize PTRZ] is enabled only in the Multi view mode.

Lens status

The information of lens location per channel is displayed. Each camera is displayed as a Channel number icon. After selecting a channel, change the PTR information to display the changed location of the selected camera.

Two preset options, [360] and [200], are provided so that camera can easily move to the set PTRZ.

If [360] is selected, as shown in the image below, the four cameras are panned and positioned to make the total angular field of view to be 360 degrees. Meanwhile, tilt value is set to 45 degrees, rotation to 0 degree, and zoom is located at the end of the wide.

If [200] is selected, as shown in the image below, the three cameras (Channel no. 1, 2, and 4 cameras) are panned and positioned to make the total angular field of view to be 200 degrees. Meanwhile, tilt value is set to 45 degrees, rotation to 0 degree, and zoom is located at the end of the wide.

Users can customize the PTRZ preset position. To move to the user-customized position, select [User preset].

Setting up PTRZ preset

  1. Select a channel to set up preset in [Lens status]. To select more than one channels, select the Multi view icon in [Select channel].
  2. Set up the camera PTRZ information as below.
    • Use the icon in [PTR setup] to adjust the camera panning and tilting in order to adjust the camera view, or use the or button to rotate the camera.
    • Set Zoom & Focus values you want in [Zoom & Focus setup].
  3. Click [Set preset].

If you click [User preset], the camera moves to the last PTRZ preset location you set.

PTRZ setup

To adjust the camera view, click the button using the mouse and drag it up, down, left, or right. The camera position moves by panning and tilting.

You can also click the button to rotate the camera clockwise.The camera will continue to rotate clockwise until the button is released. Click the button to rotate the camera counterclockwise.Continue to rotate it counterclockwise until the button is released.

Click the triangular icons on the screen movement pad’s top, bottom, left, and right sides to fine adjust the camera’s pan and tilt. If you press and hold the triangular icon, it moves the pan and tilt at a constant speed.

Zoom & focus setup

It is possible to adjust and set the zoom and focus of the camera.

To reset the focus, click the [Initialize focus] button. The focus is adjusted to the default position.

Focus

Click the Focus button on the screen and adjust the focus manually. The value displayed on the button refers to the focus moving interval. ‘1’ means moving the focus by a factor of 1, while ‘10’ means moving the focus by a factor of 10; ‘100’ means moving the focus by a factor of 100. ‘-‘moves the focus area nearer and ‘+’ moves the focus area farther away.

Zoom

Manually adjusts the zoom magnification. The value displayed on the button refers to the zoom moving interval. ‘1’ means moving the zoom by a factor of 1, ‘10’ means moving the zoom by a factor of 10, and ‘100’ means moving the zoom by a factor of 100. ‘-‘means zoom out (wider) and ‘+’ means zoom in (closer).

Simple Focus

The focus is adjusted automatically, based on the location designated by the user. When you set the focus location by clicking and dragging on the screen with the mouse and then click the [Simple Focus] button, N (Near) and<->F (Far) are displayed on the screen and the focus is controlled automatically, based on the location designated by the user.

Note

  • In any of the following cases, the simple focus function may not operate properly:
    • If there is a sudden change (movement or disappearance) of object during focus adjustment
    • If there is a sudden change in the brightness during focus adjustment
    • If the image contrast is low
    • If there is a source of strong light nearby
  • In this case, if it is out of focus after simple focus operation, then use the Zoom button (-100, -10, -1, 1, 10, or 100) to manually adjust the focus.
  • If it is hard to perform a simple focus operation in the environment, it is recommended to manually adjust the focus.
