Playback screen

You can import and playback a recorded image from an SD card or NAS. When you click the button on the screen, the recorded screen will appear. A time bar will appear at the bottom of the playback screen and an image recorded according to the set schedule or by an event is displayed on the time bar. You can search for a recorded image by event type or date, and capture or save it to a PC.

You can move to the live screen, playback screen or setup screen by clicking the buttons at the top.

  • (Live): Check the live image being captured by the camera and control various camera functions.
  • (Playback): Search and playback a recorded image from an SD card or NAS.
  • (Setup): Change the camera settings.

Note

  • If there is no connection to an SD card or NAS, the playback function cannot be used.
  • Connecting to the web viewer via Chrome, the record playback screen feature can be securely used.

Playback a recorded image through event search

You can search for a recorded image by event type.

  • Event: When you click the [All] button, the event types will appear. Selecting an event and clicking the [Apply] button, only recorded images for that event will appear on the time bar.
  • Overlapped section: When the camera time is adjusted, an overlapped time will be created. To search for recorded images during the overlapped time, select an overlapped section.

Playback a recorded image through time search

Search for a recorded image by selecting a date and time on the calendar. When you click the date on top of the time bar, the start time and end time will appear. Select the desired date on the calendar and set the start time and the end time. When you select [All time], the start and end times will be set automatically from 00:00:00 to 23:59:59. When you complete the setting and click the [Apply] button, only recorded images at the relevant time will appear on the time bar.

When you click the [Today] button on the time bar, only images recorded today will be searched.

Backing up a searched image

To backup a recorded image, select a recorded image and click the [Backup] button. You can set the start time and the end time of backup below. When you complete the setting, click the [Apply] button. When you select a path to save and click the [Save] button after the save window appears, the image will be backed up on the relevant path.
